## 1. 評価の目的
<!-- section_id: sec_1 -->

株式会社サンプル(以下「当社」)の人事評価制度は、社員一人ひとりの貢献と成長を公正に把握し、経営戦略の実現と個人のキャリア形成を両立させることを目的とする。本制度が果たす役割は、(1)処遇への反映、(2)育成への活用、(3)適正配置の判断資料という「3軸」に整理される。

### 1.1 処遇への反映
評価結果は、毎年4月の昇給(基本給改定)、6月・12月の賞与算定、ならびに昇格・昇進判断に直接連動する。具体的には、年度総合評価ランクに応じた賞与係数(後述)・昇給テーブル・等級昇格判定の基礎データとして利用される。給与制度規程および賞与規程と整合的に運用し、評価結果以外の恣意的な処遇決定を排除する。

### 1.2 育成への活用
評価は「査定」のためだけでなく、強み・弱みの自己認知を深め、能力開発を計画的に進めるためのツールとして機能する。期末評価面談では、評価点数だけでなく、SBI(Situation-Behavior-Impact)に基づく具体的な行動事実をフィードバックし、次期の能力開発計画(IDP:Individual Development Plan)を策定する。受講推奨研修、ストレッチアサイン、社内公募制度の利用などを上司と本人で合意する。

### 1.3 適正配置の判断資料
評価データはタレントマネジメントシステム「カオナビ」および「HRBrain」に蓄積され、人事部・経営層が異動・抜擢・後継者計画(サクセッションプラン)を検討する際の基礎情報となる。評価ランク・コンピテンシー・1on1の蓄積・360度評価結果を多面的に参照し、属人的判断を回避する。

### 1.4 評価の二軸構成
評価は「成果評価(MBO/業績)」と「行動評価(コンピテンシー)」の二軸で実施する。等級別のウェイト配分は下表のとおり。

| 等級区分 | 成果評価ウェイト | 行動評価ウェイト |
|---|---|---|
| 管理職(M1〜M3) | 70% | 30% |
| 専門職(P3〜P5) | 60% | 40% |
| 専門職(P1〜P2) | 50% | 50% |
| 一般職(G3〜G5) | 50% | 50% |
| 一般職(G1〜G2) | 40% | 60% |

新入社員(G1)および入社2年目までの若手は、成果より行動・成長プロセスを重視する設計とし、育成段階に応じた評価バランスを担保している。

## 2. 評価サイクル
<!-- section_id: sec_2 -->

評価サイクルは「年度評価(4月〜翌3月)」を基本単位とし、その中に「半期目標管理」「月次1on1」を組み込んだ三層構造で運用する。年間スケジュールは以下のとおり。

### 2.1 年間評価カレンダー

| 時期 | 主要イベント | 主担当 | 使用システム |
|---|---|---|---|
| 4月第1〜2週 | 全社方針・部門方針カスケード | 経営企画・部門長 | 社内ポータル |
| 4月第3〜4週 | 目標設定面談、MBO/OKR登録 | 本人・一次評価者 | HRBrain |
| 5月〜8月 | 月次1on1(隔週推奨) | 一次評価者 | カオナビ1on1 |
| 9月第1〜2週 | 中間レビュー(進捗・難易度修正) | 本人・一次評価者 | HRBrain |
| 10月〜2月 | 月次1on1継続、四半期レビュー | 一次評価者 | カオナビ1on1 |
| 3月第1週 | 自己評価入力締切 | 本人 | HRBrain |
| 3月第2週 | 一次評価入力 | 一次評価者 | HRBrain |
| 3月第3週 | 二次評価・部門内調整会議 | 部門長 | HRBrain |
| 3月第4週 | 全社キャリブレーション会議 | 人事部・役員 | HRBrain |
| 4月第1週 | 評価開示・フィードバック面談 | 一次評価者 | HRBrain |
| 4月第2週 | 不服申立受付期間(10営業日) | 人事部 | 専用フォーム |

### 2.2 半期目標管理
年度目標は4月に設定するが、9月の中間レビュー時に、事業環境変化(市場・組織・顧客動向)を踏まえて目標の追加・削除・難易度修正を行うことができる。修正は一次評価者の承認を必須とし、修正履歴はHRBrain上に保存する。期中の戦略変更により当初目標が陳腐化したまま放置されることを防ぐ仕組みとして位置づける。

### 2.3 月次1on1の位置づけ
月次1on1は「評価面談」ではなく、目標進捗と障害除去、キャリア・コンディションを話す場である。詳細は別途「1on1ミーティング実施ガイド」(doc_050)に定める。1on1の記録は評価期間中、一次評価者が随時参照可能であり、期末評価の根拠資料となる。

## 3. 評価者
<!-- section_id: sec_3 -->

評価の客観性と公正性を担保するため、当社は「多段階評価」と「キャリブレーション会議」を組み合わせた評価者体制をとる。

### 3.1 評価者の役割
| 区分 | 役割 | 担当者の例 |
|---|---|---|
| 自己評価者 | 本人 | 全社員 |
| 一次評価者 | 直属長(課長相当) | M1〜M2 |
| 二次評価者 | 部門長(部長相当) | M3 |
| 最終評価者 | 本部長または担当役員 | 執行役員 |
| 調整事務局 | 人事部評価グループ | HRBP |

一次評価者は、日常業務での観察、1on1の蓄積、成果物・KPI実績、ピア・フィードバックなどに基づき素点を算出する。二次評価者は、複数の一次評価者間での評価基準のばらつきを是正し、部門内での相対バランスを取る。

### 3.2 評価会議(キャリブレーション)
3月第4週に開催される全社キャリブレーション会議では、人事部・各本部長・担当役員が出席し、以下を実施する。
1. 評価分布の確認(後述の目安分布との乖離点検)
2. ハイパフォーマー・ローパフォーマーのレビュー
3. 部門間での評価基準の擦り合わせ
4. 抜擢・要注意人材の共有
5. 最終評価ランクの確定

調整会議の議事は人事部が記録し、評価変更の根拠を残す。一次評価者の素点と最終評価が乖離した場合、一次評価者にはフィードバック面談前に必ず変更理由を伝達する。

### 3.3 評価者研修
全評価者は年1回(2月)、「評価者研修」(オンライン3時間+集合演習3時間)の受講を必須とする。研修ではハロー効果・寛大化傾向・中心化傾向などの評価エラー、SBIによるフィードバック技法、ハラスメントに該当しない指摘の作法を扱う。未受講者は当該期の評価権限を停止し、二次評価者へ評価権が移譲される。

## 4. 評価結果
<!-- section_id: sec_4 -->

### 4.1 5段階評価と分布目安
最終評価はS・A・B・C・Dの5段階とし、全社目安分布は以下のとおり「絶対評価をベースに相対評価で調整」する。

| ランク | 定義 | 分布目安 |
|---|---|---|
| S | 卓越:期待を大幅に上回る成果と模範行動 | 10% |
| A | 優秀:期待を上回る成果 | 20% |
| B | 良好:期待どおりの成果 | 40% |
| C | 改善要:期待を一部下回る | 20% |
| D | 不十分:期待を大きく下回る | 10% |

部門事情(成熟度・繁忙度・組織変更)により±5%の調整を認めるが、S+Aで40%、C+Dで20%を上限・下限とする。新入社員・育休復帰直後・長期休業者については、特例運用(B固定または評価対象外)を人事部判断で適用する。

### 4.2 処遇反映
**(1) 賞与係数表(年度業績連動の前)**

| ランク | 賞与係数 |
|---|---|
| S | 1.50 |
| A | 1.20 |
| B | 1.00 |
| C | 0.80 |
| D | 0.60 |

**(2) 昇給テーブル(基本給に対する昇給率の目安)**

| ランク | G1〜G3 | G4〜G5 | P1〜P3 | P4〜P5 | M1〜M3 |
|---|---|---|---|---|---|
| S | 5.0% | 4.5% | 4.0% | 3.5% | 3.0% |
| A | 3.5% | 3.0% | 2.5% | 2.0% | 1.8% |
| B | 2.0% | 1.8% | 1.5% | 1.2% | 1.0% |
| C | 0.5% | 0.5% | 0% | 0% | 0% |
| D | 0% | 0% | -1.0% | -1.5% | -2.0% |

(注)P4以上のD評価では降給を伴う場合がある。詳細は給与規程に従う。

### 4.3 フィードバック面談
4月第1週に60分以上のフィードバック面談を実施する。一次評価者は、(1)最終ランクとその根拠、(2)強み3点・改善点2点、(3)次期に期待する成長領域、(4)研修・アサインの推奨を、SBI形式で具体的に伝達する。本人サイン(電子)をもって受領確認とする。

### 4.4 本人開示範囲
本人に開示する情報は以下のとおり。
1. 自身の総合評価ランク
2. 成果評価・行動評価の素点(100点満点換算)
3. 各評価項目のコメント
4. 賞与係数・昇給率(給与明細にて反映)

非開示情報は、(a)他社員の評価、(b)キャリブレーション会議での個別発言、(c)分布調整の詳細とする。

### 4.5 不服申立
評価開示後10営業日以内に、人事部評価グループ宛て「評価不服申立書」を提出することで再審査を請求できる。再審査は、人事担当役員・所属本部長・人事部長で構成する「評価審査委員会」が30営業日以内に審議し、結論を書面で通知する。再審査結果は最終確定とする。不服申立を理由とした不利益取扱いは一切行わない。
