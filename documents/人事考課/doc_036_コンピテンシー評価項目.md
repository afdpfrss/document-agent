## 1. 全社共通コンピテンシー
<!-- section_id: sec_1 -->

株式会社サンプルの全社員に共通して期待される行動特性として、以下の5項目を定める。すべての等級(G1〜M3)で評価対象とし、HRBrainの行動評価モジュールに登録される。

### 1.1 5つの全社共通コンピテンシー
1. **誠実性(Integrity)**:倫理観・コンプライアンス意識、約束遵守
2. **顧客志向(Customer Orientation)**:社内外の顧客の期待を理解し応える姿勢
3. **チームワーク(Teamwork)**:相互尊重、協働、知見共有
4. **自律性(Self-Direction)**:主体的な業務遂行、自己管理
5. **成長意欲(Growth Mindset)**:学習・挑戦への姿勢

### 1.2 5段階レベル定義表(共通フォーマット)

| レベル | 名称 | 定義 |
|---|---|---|
| Lv.1 | 基礎 | 指示・育成の下で発揮できる |
| Lv.2 | 標準 | 独力で安定的に発揮できる |
| Lv.3 | 応用 | 状況に応じて柔軟に発揮し、周囲に影響を与える |
| Lv.4 | 模範 | チーム・部門の規範となり、他者を導く |
| Lv.5 | 卓越 | 全社の文化形成に貢献し、社外でも認知される |

### 1.3 項目別レベル定義(抜粋)

**(1) 誠実性**
- Lv.1:社内ルールと業界倫理を理解し、自身の業務で逸脱しない
- Lv.2:疑義のある事案を自ら上司・コンプライアンス窓口に相談する
- Lv.3:同僚の不正の兆候に気付き、適切に進言・是正する
- Lv.4:倫理的判断が問われる場面でチームを正しい方向に導く
- Lv.5:全社のコンプライアンス文化醸成に主導的に貢献する

**(2) 顧客志向**
- Lv.1:顧客要望を正確に把握し、応える
- Lv.2:顧客の潜在ニーズを言語化し提案する
- Lv.3:顧客の事業課題を構造的に理解し、長期視点の解を示す
- Lv.4:顧客の経営層と対等に対話し、信頼関係を構築する
- Lv.5:業界全体の顧客課題を捉え、新たな価値創造を主導する

**(3) チームワーク**
- Lv.1:自分の役割を遂行しチームに貢献する
- Lv.2:同僚を支援し、知見・情報を積極共有する
- Lv.3:多様な意見を引き出し、合意形成を促進する
- Lv.4:部門を超えた協働をリードする
- Lv.5:全社の協働文化を変革する取組みを牽引する

**(4) 自律性**
- Lv.1:与えられた業務を計画的に進める
- Lv.2:課題を自ら発見し、解決策を提案する
- Lv.3:不確実な状況下でも判断し、行動する
- Lv.4:組織課題に対し主体的に提言・実行する
- Lv.5:全社視点で経営課題に向き合う

**(5) 成長意欲**
- Lv.1:必要研修を計画的に受講する
- Lv.2:業務外の自己研鑽(資格取得・社外勉強会等)を行う
- Lv.3:学びを業務適用し、周囲にも展開する
- Lv.4:組織の学習文化を醸成する
- Lv.5:業界・社会の学習を牽引する立場となる

### 1.4 等級別期待レベル

| 等級 | 期待レベル |
|---|---|
| G1〜G2 | Lv.1〜Lv.2 |
| G3〜G5 | Lv.2〜Lv.3 |
| P1〜P3 | Lv.3 |
| P4〜P5・M1〜M3 | Lv.4以上 |

## 2. 管理職コンピテンシー
<!-- section_id: sec_2 -->

管理職(M1〜M3)は、全社共通コンピテンシーに加え、以下5項目の管理職コンピテンシーを評価する。行動評価における管理職コンピテンシーのウェイトは50%(全社共通50%)。

### 2.1 5つの管理職コンピテンシー
1. **戦略思考(Strategic Thinking)**:中長期視点で組織の方向性を構想する
2. **人材育成(People Development)**:部下のキャリア・能力開発を計画的に支援する
3. **意思決定(Decision Making)**:不確実性下で迅速・的確に意思決定する
4. **変革リーダーシップ(Change Leadership)**:組織変革を構想・推進する
5. **コンプライアンスマネジメント(Compliance Management)**:組織のコンプライアンス・リスクを管理する

### 2.2 レベル定義(抜粋)

**(1) 戦略思考**
- Lv.3:所管組織の3年計画を構想し実行に落とせる
- Lv.4:事業領域の戦略を構想し、競合・市場視点を組み込める
- Lv.5:全社レベルの戦略形成に貢献し、外部環境変化を先読みする

**(2) 人材育成**
- Lv.3:部下個別のキャリア計画を策定し、ストレッチアサインを提供
- Lv.4:後継者候補を計画的に育成し、組織のサクセッションを担保
- Lv.5:全社の人材育成体系に影響を与える取組みを主導

**(3) 意思決定**
- Lv.3:必要なデータ・関係者意見を踏まえ期限内に判断する
- Lv.4:情報が不完全な状況下でも、リスクを評価し意思決定する
- Lv.5:全社経営に影響する重大判断に責任を持って関与する

**(4) 変革リーダーシップ**
- Lv.3:組織内の改善活動を主導し、文化変化を起こす
- Lv.4:部門横断のBPR・組織変革を構想・実行する
- Lv.5:全社規模のトランスフォーメーションを牽引する

**(5) コンプライアンスマネジメント**
- Lv.3:法令・社内規程を踏まえた業務運営を徹底する
- Lv.4:組織のリスク管理体制を構築・運用する
- Lv.5:全社リスクマネジメントに主導的に関与する

### 2.3 管理職昇格との連動
M1昇格時にはLv.3、M2昇格時にはLv.4、M3昇格時にはLv.4以上(うちLv.5項目1つ以上)が要件。

## 3. 専門職コンピテンシー
<!-- section_id: sec_3 -->

専門職(P1〜P5)は、全社共通コンピテンシーに加え、以下5項目の専門職コンピテンシーを評価する。専門職コンピテンシーのウェイトは50%(全社共通50%)。

### 3.1 5つの専門職コンピテンシー
1. **専門性(Expertise)**:担当領域における知識・スキルの深さ
2. **課題設定(Problem Framing)**:不確実な事象から本質的課題を見出す力
3. **解決力(Problem Solving)**:複雑な課題に対する解決策の構築・実行力
4. **知見展開(Knowledge Dissemination)**:獲得した知見を組織知化する力
5. **後進育成(Mentoring)**:若手・後輩への技術指導・キャリア助言

### 3.2 レベル定義(抜粋)

**(1) 専門性**
- Lv.3:担当領域でチーム内のトップ専門家
- Lv.4:全社の専門領域リーダーとして認知される
- Lv.5:社外講演・寄稿・学会発表を通じて社外でも認知される

**(2) 課題設定**
- Lv.3:現場の問題から本質課題を構造化できる
- Lv.4:組織が認識していない潜在課題を発見し提起する
- Lv.5:業界全体に関わる課題を提起し議論を主導する

**(3) 解決力**
- Lv.3:複雑な課題に対し独自の解決策を立案・実行する
- Lv.4:前例のない課題に対し、新たな方法論で取り組む
- Lv.5:解決事例が業界のベンチマークとなる

**(4) 知見展開**
- Lv.3:勉強会・ドキュメント等で知見を組織共有する
- Lv.4:体系化された知見が部門の標準となる
- Lv.5:全社・社外に通用する知見体系を構築する

**(5) 後進育成**
- Lv.3:OJTを通じて若手を独力遂行レベルまで育成
- Lv.4:後継者を育成し、自身が担当領域から離れても組織が機能する状態を作る
- Lv.5:全社の専門人材育成体系を構築する

### 3.3 専門職等級別ウェイト
| 等級 | 全社共通 | 専門職項目 |
|---|---|---|
| P1〜P2 | 60% | 40% |
| P3 | 50% | 50% |
| P4〜P5 | 40% | 60% |

P4以上では「専門性」「知見展開」のウェイトを特に高く設定し、専門家としての社内外への影響力を重視する。

## 4. 評価方法
<!-- section_id: sec_4 -->

### 4.1 評価の基本姿勢
コンピテンシー評価は「印象」ではなく「事実」に基づく。当社では行動事実の収集と評価に以下の手法を組み合わせる。

### 4.2 BEI(Behavioral Event Interview)
期末評価面談において、評価者は本人に対し「直近半年で最も成果を挙げた具体的場面」「困難に直面した場面」を尋ね、Situation(状況)・Task(課題)・Action(行動)・Result(結果)の流れで詳細に聴取する。これにより、本人の実際の行動特性を再現性高く把握する。

### 4.3 SBI(Situation-Behavior-Impact)記述
評価者がコンピテンシー評価コメントを記入する際は、必ずSBI形式で記述する。
- 例:「(S)新製品リリース直前のクリティカル不具合発生時、(B)担当外の領域にも自発的に踏み込みステークホルダー会議を主導した、(I)結果としてリリース遅延を最小限の3日に抑え、顧客信頼を維持した」

SBI形式により、本人が「何を評価されたか」を具体的に理解でき、納得感が向上する。

### 4.4 360度評価
全社員(G3以上)を対象に、年1回(11月)、360度評価を実施する。
- **評価者構成**:上司1名、同僚3〜5名、部下0〜5名(該当者のみ)、必要に応じ社外関係者
- **評価項目**:全社共通コンピテンシー5項目+等級別項目から5項目程度
- **形式**:5段階+自由記述コメント
- **匿名性**:本人には集計結果のみ開示。個別評価者の特定不可
- **活用**:期末評価面談での参考資料、自己認識ギャップの可視化、能力開発計画への反映

360度評価の結果のみで処遇判断は行わない(参考情報)。但し、上司評価と360度評価の大幅な乖離はキャリブレーション会議で議論される。

### 4.5 評価頻度とシステム運用

| 評価種類 | 頻度 | 使用システム |
|---|---|---|
| 自己評価(コンピテンシー) | 半期1回 | HRBrain |
| 上司評価(コンピテンシー) | 半期1回 | HRBrain |
| 360度評価 | 年1回(11月) | カオナビ360 |
| 1on1記録 | 隔週 | カオナビ1on1 |

### 4.6 評価エラーへの注意喚起
評価者研修にて、以下の典型的評価エラーを学習する。
1. ハロー効果:一つの突出した点で全体評価が引きずられる
2. 寛大化傾向:全員に高評価をつける
3. 中心化傾向:全員にB評価をつける
4. 近接効果:期末直前の事象に評価が偏る
5. 対比効果:特定の他社員と比較してしまう

評価者は半期分の行動事実を継続的に記録し(カオナビ1on1の活用)、期末評価時に近接効果を回避する。
