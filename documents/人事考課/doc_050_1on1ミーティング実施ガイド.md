## 1. 1on1の目的
<!-- section_id: sec_1 -->

株式会社サンプルにおける1on1ミーティング(以下「1on1」)は、上司が部下のために対話の場を提供し、部下の成長・関係構築・業務支援を継続的に行うための制度である。当社では2022年から全社導入し、2026年現在、隔週30分・月1回必須を基準として運用している。

### 1.1 1on1の3つの目的
1. **部下育成**:中長期キャリア・スキル開発・行動変容の支援
2. **関係構築**:心理的安全性の醸成、信頼関係の継続的構築
3. **業務支援**:障害除去、優先度整理、判断の支援

### 1.2 評価面談との明確な違い
1on1は「評価面談」とは目的・スタンスが異なる。両者を混同すると部下の本音は出てこなくなり、1on1の効果が失われる。

| 項目 | 1on1 | 評価面談 |
|---|---|---|
| 目的 | 部下の成長支援・障害除去 | 評価結果の伝達と合意 |
| 主導権 | 部下 | 上司 |
| 話す割合 | 部下7:上司3 | 上司6:部下4 |
| 頻度 | 隔週(月2回) | 年2回 |
| 時間 | 30分 | 60〜90分 |
| 記録 | 共有メモ | 評価記録(HRBrain) |
| 雰囲気 | カジュアル・対話 | フォーマル・合意形成 |

評価結果や処遇に関する具体的議論は1on1では行わず、別途評価面談を設ける。

### 1.3 心理的安全性の重要性
1on1は「部下が安心して本音を話せる場」であり、上司は以下のスタンスを徹底する。
- 部下の発言を遮らない、否定から入らない
- 沈黙を恐れず、待つ
- 「アドバイス」より「質問」を優先する
- ネガティブな話題でも受容する
- 1on1で聞いた内容を、本人の許可なく他者に共有しない(評価会議への参照は事前に伝える)

### 1.4 上司にとっての価値
1on1は部下のためだけでなく、上司にとっても、部下理解の深化、組織課題の早期把握、マネジメント力の向上に直結する。1on1の質はキャリブレーション会議や人事評価における上司への評価指標の一つとして参照される。

## 2. 頻度と時間
<!-- section_id: sec_2 -->

### 2.1 推奨頻度
- **隔週30分**:全社推奨基準
- **月1回**:多忙時の最低ライン(必須)
- **新人・育成対象者**:週1回30分(入社後3ヶ月および異動直後3ヶ月)
- **リモート勤務者**:月2回以上を厳守(対面機会が少ないため)

### 2.2 時間配分の目安(30分の場合)
| 時間 | 内容 |
|---|---|
| 0-5分 | 近況・体調確認(アイスブレイク) |
| 5-20分 | 部下からのテーマ(業務・キャリア・対人等) |
| 20-25分 | 上司からの質問・コーチング |
| 25-30分 | 次回までのアクション、次回日程 |

### 2.3 緊急中止・変更条件
以下の場合のみ1on1の延期・中止を認める。
1. 顧客対応(クレーム・障害対応等)で緊急性が高い場合
2. 当事者いずれかの体調不良
3. 重大な経営イベント(役員報告・取締役会等)との重複

延期する場合は、原則として同週内に再設定する。連続2回スキップは禁止し、人事部からアラートが配信される。

### 2.4 1on1の場所・形式
- 対面実施:会議室(個室)で実施。フリーアドレス席や打ち合わせブースは推奨しない
- リモート実施:カメラオン推奨。背景は雑然としていないこと
- ハイブリッドチーム:対面とリモートを公平に組合せ、特定メンバーが不利にならないよう配慮

### 2.5 上司が話す割合
1on1の主役は部下であり、上司の発言量は全体の30%以下を目安とする。コーチングスタイルを基本とし、「Tell(指示)」より「Ask(質問)」を優先する。

## 3. 話題例
<!-- section_id: sec_3 -->

1on1のテーマは部下が選ぶが、ネタが尽きないよう以下のテーマローテーション表を用意している。月次で異なるテーマに触れることで、部下を多面的に理解する。

### 3.1 5つのテーマ領域

| テーマ | 内容例 | 推奨頻度 |
|---|---|---|
| 業務 | 進捗、課題、優先度、リソース | 毎回 |
| キャリア | 中期希望、スキル開発、異動希望 | 月1回 |
| 対人関係 | チーム内コミュニケーション、他部門連携 | 月1回 |
| 心身状態 | 体調、ストレス、ワークライフバランス | 月1回 |
| 上司への要望 | マネジメント改善、サポート要請 | 四半期1回 |

### 3.2 業務テーマの質問例
- 今、最も時間を使っているのは何ですか
- うまく進んでいることは何ですか
- 障害になっていることはありますか
- 私(上司)に取り除いてほしい障害はありますか
- 来週までに何を達成したいですか

### 3.3 キャリアテーマの質問例
- 3年後にどのような仕事をしていたいですか
- 今の業務で身につけたいスキルは何ですか
- 興味のある領域・プロジェクトはありますか
- 社内公募・異動を考えていますか
- ロールモデルとなる人物はいますか

### 3.4 対人関係テーマの質問例
- チーム内で連携がうまくいっている人・難しい人は
- 他部門との協働で困っていることは
- 会議の進め方で改善したいことは
- 後輩・新人との関わり方で悩みは

### 3.5 心身状態テーマの質問例
- 睡眠・食事は取れていますか
- 残業時間に無理はありませんか
- 業務以外でリフレッシュできていますか
- 仕事のプレッシャーをどの程度感じていますか
- 産業医・カウンセラーに相談したいことはありますか

### 3.6 上司への要望テーマの質問例
- 私のマネジメントで改善してほしい点は
- もっと欲しいサポートは
- 情報共有の頻度・粒度は適切ですか
- 私からのフィードバックは役立っていますか

### 3.7 NGテーマ
以下は1on1の場で扱わない、または慎重に扱う。
- 他者の人事評価・処遇詳細
- 上司の上司への不満(必要時は別ルートで扱う)
- 政治・宗教・個人信条
- 個人のプライバシー(家族構成等を執拗に尋ねる行為)

## 4. 記録と活用
<!-- section_id: sec_4 -->

### 4.1 記録ツールと運用
当社では1on1の記録に「カオナビ1on1」モジュールおよび共有Notionページを併用する。部下が自身のNotionページにアジェンダを事前記入し、面談中に双方が追記する形式とする。

### 4.2 記録項目
| 項目 | 記入者 | 内容 |
|---|---|---|
| 日時 | 自動 | 実施日時 |
| アジェンダ | 部下 | 話したいテーマ |
| 議論内容 | 双方 | 話された内容の要点 |
| 気付き | 双方 | 新たに見えてきたこと |
| アクション | 双方 | 次回までに行うこと(担当・期限) |
| 上司メモ | 上司 | 評価・育成上の所見(部下非公開可) |

上司メモ欄は、評価期間中の参考資料として利用する場合がある。本人非公開で記録する場合でも、本人にその旨を事前に伝える。

### 4.3 評価への参照可否
1on1の記録は、期末評価の根拠資料として参照可能であるが、以下のルールを守る。
1. 本人が知っている事実のみを評価根拠とする(隠れた情報での評価不可)
2. 個人的な悩み・心身状態に関する情報は評価に使用しない
3. キャリブレーション会議で個別1on1内容を読み上げない
4. 産業医・カウンセラー相談関連は人事評価と完全に分離

### 4.4 上長による棚卸
人事部は四半期に1回、各管理職の1on1実施状況を棚卸する。指標は以下。
- 実施率(計画件数に対する実施件数)
- 平均実施時間
- 記録の質(アジェンダの多様性)
- 部下からの1on1満足度(年2回のサーベイ)

実施率80%未満・満足度顕著に低い管理職には、人事部から個別フォロー(コーチング、研修受講)を行う。

### 4.5 1on1の質を高めるTips
1. 上司は事前に「前回のアクション」を確認してから臨む
2. ノートPC・スマホをしまい、対話に集中する
3. 沈黙は思考の時間。3秒以上待つ習慣をつける
4. 「なぜ」より「どのように」「何が」で質問する
5. 部下の感情(嬉しい・悔しい・不安)に名前を付けて受容する
6. アドバイスは部下が求めたときのみ行う
7. 1on1の最後に「今日話せてよかったこと」を相互に確認する

### 4.6 1on1サーベイ
半期に1回(5月・11月)、部下に対して1on1満足度サーベイを実施。質問項目は以下。
- 1on1は自分の成長に役立っているか(5段階)
- 上司は自分の話を傾聴してくれているか(5段階)
- 1on1で安心して話せているか(5段階)
- 1on1の頻度は適切か(多すぎ・適切・少なすぎ)
- 自由記述(改善要望)

サーベイ結果は上司本人と人事部にのみ集計値で開示される。
