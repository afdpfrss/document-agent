## 1. 等級体系
<!-- section_id: sec_1 -->

株式会社サンプルの等級体系は、職務遂行能力と役割期待に基づく「役割等級制度」を採用し、一般職(G:General)・専門職(P:Professional)・管理職(M:Management)の3コース・計13等級で構成される。コース間の異動(複線型キャリア)が可能であり、専門職と管理職を行き来できる。

### 1.1 等級定義と年収レンジ
| 区分 | 等級 | 役割定義 | 年収レンジ(目安) |
|---|---|---|---|
| 一般職 | G1 | 新入社員、基礎業務の習得 | 350〜420万円 |
| 一般職 | G2 | 定型業務を独力遂行 | 400〜480万円 |
| 一般職 | G3 | 担当業務に責任を持ち遂行 | 450〜550万円 |
| 一般職 | G4 | 後輩指導・小チームの実務リーダー | 500〜620万円 |
| 一般職 | G5 | 業務改善を推進、部内エキスパート | 550〜700万円 |
| 専門職 | P1 | 専門領域での独力遂行 | 550〜680万円 |
| 専門職 | P2 | 高度専門業務、若手指導 | 650〜800万円 |
| 専門職 | P3 | 部門横断課題の専門的解決 | 750〜950万円 |
| 専門職 | P4 | 全社専門領域のリード | 900〜1,200万円 |
| 専門職 | P5 | 業界に通じる高度専門家 | 1,100〜1,600万円 |
| 管理職 | M1 | 課長(チーム10名規模) | 850〜1,100万円 |
| 管理職 | M2 | 部長(部門50名規模) | 1,100〜1,500万円 |
| 管理職 | M3 | 本部長(事業領域全体) | 1,400〜2,200万円 |

### 1.2 コース転換
- G5⇔P1、P3⇔M1、P4⇔M2、P5⇔M3 は同格として相互転換可能。
- 転換には所属本部長の推薦と人事委員会承認が必要。
- 専門職から管理職への転換は、マネジメント適性検査(外部アセスメント)を必須とする。
- 管理職から専門職への転換は、本人の希望またはマネジメント不適合時に検討される(降格扱いとはしない)。

### 1.3 等級と職務記述書
各等級にはJob Description(職務記述書)が紐づき、期待される役割・成果・行動レベルが明文化されている。職務記述書はカオナビ上で全社公開されており、本人がキャリア計画を立てる際の参照資料となる。

## 2. 昇格要件
<!-- section_id: sec_2 -->

### 2.1 共通要件
すべての等級昇格に共通する要件は以下のとおり。
1. 現等級での評価実績(後述の等級別基準を満たす)
2. 必須研修の修了
3. 一次評価者と部門長の推薦
4. コンプライアンス違反による懲戒処分歴がない(直近3年)
5. 必要勤続年数の充足

### 2.2 等級別必要勤続年数と評価点数

| 昇格 | 最短在籍年数 | 必要評価実績 | 必須研修 |
|---|---|---|---|
| G1→G2 | 1年 | B以上 | 新入社員研修・ビジネス基礎 |
| G2→G3 | 2年 | 直近2期B以上 | ロジカルシンキング |
| G3→G4 | 2年 | 直近2期にA1回以上 | リーダーシップ基礎 |
| G4→G5 | 2年 | 直近2期にA1回以上 | OJTトレーナー研修 |
| G5→P1/M1 | 2年 | 直近2期A以上 | 専門コース別研修 |
| P1→P2 | 2年 | 直近2期B以上 | 専門深化研修 |
| P2→P3 | 3年 | 直近3期にA2回以上 | プロジェクトマネジメント |
| P3→P4 | 3年 | 直近3期A以上 + 役員推薦 | 戦略思考研修 |
| P4→P5 | 3年 | 直近3期A以上 + 役員推薦 | 外部アセスメント |
| M1→M2 | 3年 | 直近3期A以上 + 役員推薦 | マネジメント上級 |
| M2→M3 | 3年 | 直近3期にS1回・A2回以上 + 役員推薦 | 経営塾(社外派遣) |

### 2.3 推薦プロセス
昇格推薦は毎年1月に開始する。
1. 1月10日:一次評価者が候補者リストを作成、部門長に提出
2. 1月20日:部門長レビュー、本部長へエスカレーション
3. 1月末:本部長推薦書を人事部に提出
4. 2月:人事委員会で候補者の事前審議
5. 2月下旬〜3月:昇格試験(該当者)
6. 3月末:昇格判定会議(役員会)
7. 4月1日:昇格発令

### 2.4 飛び級制度
特に顕著な成果を挙げた者については、「飛び級昇格」(2等級同時昇格)を例外的に認める。直近2期連続Sランクで、本部長と人事担当役員の連名推薦が条件。年間枠は全社で5名以内とし、人事委員会の特別審議を要する。

## 3. 昇格試験
<!-- section_id: sec_3 -->

### 3.1 試験構成(対象等級別)

| 昇格 | 試験内容 | 配点 | 合格基準 |
|---|---|---|---|
| G5→P1/M1 | 論文 + 面接 | 論文40・面接60 | 合計70点以上 |
| P3→P4 | 論文 + プレゼン + 面接 | 30・40・30 | 合計75点以上 |
| P4→P5 | 外部アセスメント + 論文 + 面接 | 40・30・30 | 合計75点以上 |
| M1→M2 | プレゼン + ケース面接 + 役員面接 | 30・40・30 | 合計75点以上 |
| M2→M3 | 外部アセスメント + 戦略提案 + 役員面接 | 30・40・30 | 合計80点以上 |

### 3.2 論文試験
- 字数:4,000字程度
- テーマ:自身の業務経験から導いた専門的提言、または当社の経営課題に対する解決案
- 評価観点:論理性、独自性、実現性、文章力
- 試験官:人事部および外部審査員(社外取締役、人事コンサルタント等)2名以上

### 3.3 プレゼンテーション
- 時間:発表15分+質疑15分
- テーマ:候補者が自ら設定した経営・事業課題に対する提案
- 評価観点:課題設定の的確性、解決策の妥当性、表現力、質疑応答力
- 審査員:本部長3名+人事担当役員

### 3.4 ケース面接(管理職昇格)
- 形式:架空のマネジメント課題ケース(部下のパフォーマンス不振、組織変更、コンプライアンス事案など)に対する対応方針を口頭で説明
- 時間:準備15分+面接30分
- 評価観点:意思決定の論理性、人間理解、リスク認識、コンプライアンス感度

### 3.5 外部アセスメント
P4以上・M2以上の昇格では、外部機関(ヒューマンキャピタル社等)によるアセスメントセンター方式の評価を導入。インバスケット、グループ討議、個別面接で構成され、報告書は人事部経由で候補者本人にもフィードバックされる。

### 3.6 不合格時の取扱い
試験不合格者は翌年再受験可能。連続2回不合格の場合は1年の受験猶予期間を設け、その間に上司・人事部が能力開発計画を策定する。不合格は処遇に直接影響しないが、昇格は当該年度延期となる。

## 4. 降格
<!-- section_id: sec_4 -->

### 4.1 降格事由
以下のいずれかに該当する場合、降格を検討する。
1. **業績による降格**:2期連続D評価、または3期連続C評価
2. **能力不足による降格**:現等級の職務記述書に定める役割を継続的に遂行できないと判断された場合
3. **懲戒による降格**:就業規則に定める懲戒処分のうち「降格」処分が確定した場合
4. **管理職不適合による格付け変更**:マネジメント職務において部下管理上の重大問題が継続する場合(M→P転換扱い、給与影響は調整)

### 4.2 対象等級と影響範囲
| 降格区分 | 対象等級 | 給与影響 | 役職影響 |
|---|---|---|---|
| 1等級降格 | G2以上 | 基本給5〜10%減 | 役職解任の場合あり |
| 2等級以上降格 | P3以上・M1以上 | 基本給15〜20%減 | 原則役職解任 |
| コース変更(M→P) | M1〜M3 | 役職手当のみ減 | 役職解任、専門職への配置転換 |

### 4.3 改善期間(PIP)
業績・能力不足による降格検討に先立ち、原則として6ヶ月の改善期間(Performance Improvement Plan)を設ける。
1. 改善目標を上司・人事・本人で合意
2. 月次レビューを実施し、進捗を記録
3. 期間終了時に改善判定会議を開催
4. 改善が認められない場合、降格判定会議へ付議

PIPは「降格を前提とした追い込み」ではなく、本人の能力発揮を支援する制度として運用する。PIP期間中のハラスメントは一切認めない。

### 4.4 降格通知と異議申立
降格決定は、人事担当役員・所属本部長・人事部長で構成する「処遇審議委員会」が決定し、本人に書面で通知する。通知から10営業日以内に異議申立が可能で、申立があった場合は外部社労士を含む再審議を行う。

### 4.5 降格後の再昇格
降格後1年間は原則として再昇格不可。1年経過後、現等級でA評価以上を直近2期連続で取得すれば、通常の昇格プロセスでの再昇格が可能となる。
