## 1. ストレスチェック
<!-- section_id: sec_1 -->

株式会社サンプル(以下「当社」)は、労働安全衛生法第66条の10および同法施行規則第52条の9以下の規定に基づき、年1回(原則として6月1日〜6月30日の1か月間)、常時使用する全従業員(役員・正社員・契約社員・派遣社員のうち当社で1年以上継続勤務する者)を対象としてストレスチェックを実施しています。実施者は当社の選任産業医(統括産業医:佐藤健一医師)および外部委託先である一般財団法人日本ストレスチェック協会、実施事務従事者は人事部健康管理グループ所属の保健師2名が務めます。

### 1-1. 実施スキームと使用尺度

設問は厚生労働省推奨の「職業性ストレス簡易調査票(57項目版)」を採用し、加えて当社オリジナルの「組織エンゲージメント設問(15項目)」を組み合わせた計72項目のWebアンケート形式で実施します。回答所要時間は概ね10〜15分。社内ポータル「Sample Workplace」のメンタルヘルスケアタブからログインし、業務時間内に受検することを推奨しています(直属上長は受検時間を業務時間として承認する義務があります)。

### 1-2. 結果のフィードバック

受検後2週間以内に、結果通知書(ストレス反応プロファイル、仕事のストレス要因、周囲のサポート状況、総合判定)を本人のみに電子配信します。会社・上長・人事部は個人の結果にアクセスできません(本人同意がある場合を除く)。判定区分は以下の通りです。

| 判定区分 | 該当目安 | 推奨アクション |
| --- | --- | --- |
| A:低ストレス | 上位40% | セルフケア継続 |
| B:中程度 | 中位40% | セルフケア強化・社外EAP活用推奨 |
| C:やや高ストレス | 上位15% | 保健師面談を任意で案内 |
| D:高ストレス者 | 上位5% | 産業医面談を強く推奨(申出は本人) |

### 1-3. 高ストレス者面談

判定Dの従業員には、結果通知に併せて産業医面談申込URLを送付します。申出があった場合、原則1か月以内に産業医面談(30〜45分)を実施し、必要に応じて主治医受診勧奨、業務軽減、就業場所変更等の就業上の措置を講じます。面談を受けたこと自体を理由とする不利益取扱は労安衛法第66条の10第3項により禁止されており、当社就業規則第48条にも明記しています。

### 1-4. 集団分析と職場改善

個人結果を10名以上の単位(課・グループ)で匿名集計し、「健康いきいき職場度」「仕事の量的負担」「仕事のコントロール度」「上司・同僚の支援」の4軸で部門スコアを算出します。結果は経営会議および各事業部長に共有され、スコアが全社平均を1標準偏差以上下回った部署については、部長と人事部健康管理グループが共同で職場改善ワークショップ(年2回)を開催することを義務付けています。

## 2. 相談窓口
<!-- section_id: sec_2 -->

当社は「相談先がない」という状況を作らないことを基本方針とし、社内・社外・家族向けの多層的な相談窓口を整備しています。すべての相談は秘密厳守で、人事評価・配属・賞与等に一切影響しません。

### 2-1. 社内窓口

- **産業医面談**:統括産業医が月2回(第2・第4水曜日)虎ノ門本社4階ヘルスケアルームに来社。事業所(大阪・福岡)へは隔月でリモート面談を実施。予約は社内ポータル「健康相談予約システム」より24時間可能。
- **保健師相談**:常勤保健師2名(田中・鈴木)が平日9:00〜18:00、本社ヘルスケアルームで対応。心身の不調、睡眠、ダイエット、禁煙、女性特有の健康問題など幅広く相談可。
- **ハラスメント相談窓口**:人事部コンプライアンスグループおよび外部弁護士(西村あさひ法律事務所)で構成。専用フォーム「Speak Up Sample」より匿名相談可。

### 2-2. 社外EAP「ピースマインド」

メンタル不調の相談に特化した社外EAP(Employee Assistance Program)として株式会社ピースマインドと契約しています。利用実績・利用有無は会社に一切報告されません。

| サービス | 内容 | 利用可能回数 |
| --- | --- | --- |
| 電話・チャット相談 | 24時間365日・多言語対応(日英中韓越) | 無制限 |
| 対面・オンラインカウンセリング | 臨床心理士・公認心理師による50分セッション | 年間8回まで会社負担 |
| キャリア相談 | キャリアコンサルタントとのセッション | 年間3回まで会社負担 |
| 法律・税務相談 | 弁護士・税理士への一次相談 | 年間2回まで会社負担 |

### 2-3. 家族からの相談

従業員本人だけでなく、同居家族(配偶者・パートナー・子・親)もピースマインドのサービスを利用できます。「最近本人の様子がおかしい」「眠れていないようだ」といった家族からの相談も受け付けており、本人を介さず保健師に直接連絡することも可能です(連絡先:080-XXXX-XXXX 健康管理ホットライン)。

## 3. 休職と復職支援
<!-- section_id: sec_3 -->

メンタル不調による就労困難が生じた場合、当社は治療と仕事の両立を支援することを基本方針とします。主治医・産業医・人事部・上長・本人の5者連携によって、無理のない復帰を実現します。

### 3-1. 休職開始の手続

主治医による「就業困難」「○か月の自宅療養を要する」等の診断書を取得した場合、人事部に提出します。診断書受領後5営業日以内に休職判定会議(産業医・人事部長・本人所属部長)を開催し、休職発令日・休職期間を決定します。

### 3-2. 休職期間と通算ルール

| 勤続年数 | 休職可能期間(累計) |
| --- | --- |
| 1年未満 | 3か月 |
| 1年以上3年未満 | 6か月 |
| 3年以上10年未満 | 12か月 |
| 10年以上 | 最長18か月 |

同一傷病(精神疾患は同一分類として扱う)による休職は、復職後6か月以内に再休職した場合、前回休職期間と通算します。これにより安易な短期復職を防ぎ、本人の十分な回復期間を確保します。

### 3-3. 復職判定とリハビリ出社

主治医の「復職可」診断書取得後、以下のステップで復職判定を実施します。

1. **書類提出**:主治医診断書・復職願・生活記録表(直近2週間の起床/就寝/活動内容)
2. **産業医面談**:就業可否の医学的判断
3. **リハビリ出社(慣らし出勤)**:給与は発生しない試験期間として2〜4週間、本社/最寄り拠点に9:30〜15:30で出社。図書室での読書・軽作業を実施。
4. **復職判定会議**:産業医・人事部・所属部長で開催。週20時間の短時間勤務から開始するか、いきなり通常勤務とするかを決定。
5. **段階的勤務復帰プラン**:時短(4時間)→6時間→8時間時差→通常勤務、と概ね3か月かけて段階移行。

### 3-4. 復職後フォローと再休職時の取扱

復職後12か月間は産業医面談を月1回義務付け、加えて上長による1on1を週1回実施します。残業は復職後6か月間原則禁止、出張・夜勤・海外渡航は産業医許可制とします。前述のとおり、復職後6か月以内の再休職は期間通算となるため、本人の状態に応じて在宅勤務や業務量調整など柔軟な配慮を講じます。

## 4. ラインケア(管理職向け)
<!-- section_id: sec_4 -->

ラインケアとは、管理監督者が部下のメンタルヘルス不調を早期に発見し、適切な対応につなげる取組をいいます。セルフケア(本人による自己管理)と区別される概念で、厚生労働省「労働者の心の健康の保持増進のための指針」における4つのケア(セルフケア・ラインケア・事業場内産業保健スタッフによるケア・事業場外資源によるケア)の第二の柱です。

### 4-1. 兆候発見の4視点(SOAR)

部下の変化に気づくため、以下の4視点で日常的に観察します。

| 視点 | 観察ポイント | 具体例 |
| --- | --- | --- |
| **S** Schedule(勤務) | 遅刻・早退・欠勤・残業の急増/急減 | 月曜日の欠勤が3週連続 |
| **O** Output(業務) | ミス増加、納期遅延、報告の遅れ | 普段ミスしない人の凡ミス連発 |
| **A** Appearance(様態) | 表情・服装・体重・身だしなみの変化 | 笑顔消失、無精ひげ、急な痩せ |
| **R** Relationship(対人) | 雑談減少、会議発言減、孤立 | 昼食を一人で取るようになった |

これらが**2週間以上継続**している場合、要注意サインです。

### 4-2. 声かけの例(OK/NG)

| 状況 | NG例 | OK例 |
| --- | --- | --- |
| 元気がない部下に | 「最近やる気ないよね」 | 「最近よく眠れてる?顔色が気になって声をかけたんだけど」 |
| ミスが続く部下に | 「気合いが足りないんじゃないか」 | 「いつもの○○さんらしくないミスだったから心配で。何か困っていることがあれば聞かせて」 |
| 休みがちな部下に | 「みんな大変なんだから頑張れ」 | 「体調はどう?無理せず、必要なら産業医面談も手配できるよ」 |

「頑張れ」「気の持ちよう」「みんな大変」という励まし・比較は禁句です。傾聴(さえぎらず、評価せず聞く)と共感(「それは大変だね」)を基本とします。

### 4-3. 産業医・人事部への連携

部下から「眠れない」「死にたい」等の発言があった場合、または2週間以上のサイン継続を確認した場合は、速やかに人事部健康管理グループ(内線:5151)または産業医に連絡します。緊急時(自殺をほのめかす発言、極度の興奮・混乱)は、その場を離れず本人を一人にせず、すぐに119番または保健師ホットライン(080-XXXX-XXXX)に連絡してください。

### 4-4. セルフケアとの違いと管理職自身のケア

セルフケアは本人による予防(運動・睡眠・休養・趣味)、ラインケアは上司による早期発見・職場環境改善が役割です。管理職は部下のケアに注力するあまり自身のケアが疎かになりがちなため、年1回の管理職向けメンタルヘルス研修(eラーニング90分)受講を必須化しています。また、管理職自身もEAP・産業医面談を活用してください。
