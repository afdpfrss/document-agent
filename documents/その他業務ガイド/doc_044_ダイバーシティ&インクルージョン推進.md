## 1. D&I方針
<!-- section_id: sec_1 -->

### 1-1. 経営トップメッセージ

> 「私たちは、多様な背景・価値観・才能を持つ仲間が、互いを尊重し、安心して挑戦できる職場をつくることを、経営の最優先課題のひとつとして位置付けます。ダイバーシティは『あるべき姿』ではなく、未来の顧客と社員のための『競争力の源泉』です。一人ひとりの違いが活きる組織を、現場のリーダーシップとともに築いていきます。」
> ― 株式会社サンプル 代表取締役社長 山田 太郎(2026年4月「Sample D&I Declaration 2030」より)

### 1-2. D&Iの基本方針

当社は、すべての役職員が性別、年齢、国籍、人種、宗教、信条、障がいの有無、性的指向、性自認、家庭環境、雇用形態等の属性によって不当に区別・排除されることなく、各々の個性と能力を最大限に発揮できる職場の実現を目指します。本方針は、人権方針、行動規範、人事諸規程、サプライヤー行動規範と一体運用されます。

### 1-3. KPI(2030年目標)

中期経営計画「CSV2027」および「Sample D&I Declaration 2030」において、以下の定量目標を設定しています。実績は統合報告書および当社ウェブサイトのサステナビリティページで毎年開示します。

| 指標 | 2025年度実績 | 2030年度目標 |
| --- | --- | --- |
| 女性管理職比率(課長級以上・連結) | 18.2% | 30.0% |
| 女性役員比率(取締役・執行役員) | 22.0% | 35.0% |
| 男性育児休業取得率 | 62.0% | 80.0% |
| 男性育児休業平均取得日数 | 28日 | 60日 |
| 障がい者雇用率(単体) | 2.55% | 2.7% |
| 外国籍社員比率(連結) | 3.2% | 5.0% |
| 中途採用比率(管理職) | 31.0% | 40.0% |
| エンゲージメントスコア(D&I設問) | 72pt | 80pt |

### 1-4. 推進体制

- **D&I推進委員会**:社長を委員長、CHRO(人事担当執行役員)を副委員長とし、各事業本部長、労働組合代表、社外有識者1名で構成。年4回開催。
- **D&I推進室**:人事部内に設置(室長以下8名)。各事業部に「D&Iプロモーター」(計42名)を配置し、現場活動と連動。
- **取締役会への報告**:年2回、KPI進捗と重要課題を報告し、経営戦略と一体で議論。
- **外部評価**:「えるぼし(3つ星)」「プラチナくるみん」「PRIDE指標(ゴールド)」「DX認定」「健康経営優良法人(ホワイト500)」を継続認定。

### 1-5. インクルージョンの取組

属性の多様性(Diversity)を確保するだけでなく、多様な人材が組織に「居場所」と「発言権」を感じられるインクルージョン(Inclusion)の醸成を重視します。全管理職に対し「アンコンシャス・バイアス研修」(年1回・eラーニング90分+ワークショップ4時間)、全社員向け「インクルーシブ・リーダーシップ基礎」(年1回・60分)を必修化しています。

## 2. 女性活躍推進
<!-- section_id: sec_2 -->

### 2-1. 女性活躍推進法に基づく行動計画

「女性の職業生活における活躍の推進に関する法律」(女性活躍推進法)に基づき、第3期一般事業主行動計画(2024年4月〜2027年3月)を策定・届出しています。主要計画事項は次の通りです。

1. 採用における女性比率の向上(総合職新卒採用に占める女性比率45%以上)
2. 女性管理職パイプライン強化(課長候補となる主任クラス女性比率35%以上)
3. 男女間賃金格差の縮小(全労働者の男女賃金差異85%以上、正社員90%以上)
4. 育児・介護と仕事の両立支援拡充

### 2-2. キャリア形成支援:メンター制度

主任〜課長クラスの女性社員を対象に、当社役員・部長クラスをメンターとして配置する「Sample Women's Leadership Program(SWLP)」を運営しています。1年間・月1回・60分のメンタリングセッション、四半期に1度の合同セッション、社外交流イベント参加を組み合わせ、年間定員60名(うち上長推薦30名・公募30名)で運営しています。修了者の管理職昇格率は非参加者の約1.8倍です(2024年度実績)。

### 2-3. 両立支援制度

| 制度 | 内容 | 法定との比較 |
| --- | --- | --- |
| 産前産後休業 | 産前8週(法定6週)・産後8週 | 上回る |
| 育児休業 | 子が満3歳に達するまで(分割取得可) | 法定1歳超 |
| 短時間勤務 | 小学校3年生終了まで | 法定3歳まで |
| 子の看護休暇 | 年10日(子1人当たり、有給) | 法定5日無給 |
| ベビーシッター利用補助 | 1時間2,000円補助・月40時間まで | 任意 |
| 不妊治療休暇 | 年5日(有給) + 通院短時間勤務 | 任意 |
| 配偶者出産休暇 | 5日(有給) | 任意 |

### 2-4. 復職プログラム「Welcome Back Sample」

育児休業からの復職予定者を対象に、復職前3か月〜復職後6か月の伴走プログラムを実施しています。

- 復職前面談(2回):上長・人事・本人で復職時期、業務分担、勤務時間を擦り合わせ
- 復職前eラーニング:法改正、新システム、社内動向のキャッチアップ
- 復職初日オリエンテーション:配属先での業務インプット、メンター紹介
- 復職後フォロー1on1:月1回×6か月、人事担当者とのキャリア相談
- 「ママパパ・コミュニティ」:育休経験者の社内コミュニティ(参加者580名)

## 3. 障がい者雇用
<!-- section_id: sec_3 -->

### 3-1. 法定雇用率と当社実績

障害者雇用促進法の改正により、民間企業の法定雇用率は2024年4月から2.5%、2026年7月から2.7%へ段階引き上げとなりました。当社単体の障がい者雇用率は2025年度末時点で2.55%(法定達成)、2030年度には2.7%以上を維持目標としています。特例子会社「サンプル・ハートフル株式会社」(2012年設立、神奈川県川崎市、従業員128名)とのグループ算定により、安定的に法定達成を継続しています。

### 3-2. 特例子会社の事業内容

サンプル・ハートフル株式会社では、社内印刷・名刺・郵便仕分け、ノベルティ製造、社内カフェテリア運営、植物管理、データ入力、清掃、ITヘルプデスク等を運営し、グループ各社の業務をハートフルに移管することで安定雇用を実現しています。職場には精神保健福祉士、ジョブコーチを常駐させ、定着支援を行っています。

### 3-3. 合理的配慮の提供

採用選考時から在職中まで、当事者との対話を通じて以下のような合理的配慮を提供します。

| 障がい類型 | 配慮例 |
| --- | --- |
| 視覚障がい | スクリーンリーダー対応端末、画面拡大ソフト、点字対応、ガイドヘルパー帯同出張 |
| 聴覚障がい | UDトーク・要約筆記・手話通訳手配、Web会議の自動字幕化、緊急時の光・振動アラート |
| 肢体不自由 | バリアフリー執務環境、車いす用デスク、通勤時間配慮、駐車場確保 |
| 内部障がい | 通院休暇、休憩室確保、リモートワーク優先 |
| 精神・発達障がい | 業務指示の文書化・構造化、静音席、ノイズキャンセリングヘッドホン貸与、ジョブコーチ |

配慮の申出は人事部 D&I推進室(内線5161)、社内ポータル「合理的配慮申出フォーム」、所属長のいずれからも可能です。

### 3-4. 職場体験と採用パイプライン

特別支援学校・就労移行支援事業所・大学のキャリアセンターと連携し、年間60名規模の職場体験実習(2週間〜3か月)を受け入れています。実習を経た採用実績は年間8〜12名で、当社・特例子会社双方の重要な採用パイプラインとなっています。

## 4. LGBTQ+への対応
<!-- section_id: sec_4 -->

### 4-1. 基本姿勢

当社は、性的指向(SO:Sexual Orientation)・性自認(GI:Gender Identity)・性表現(GE:Gender Expression)・身体的性(SC:Sex Characteristics)、いわゆるSOGIESCに関わらず、すべての役職員が自分らしく働ける環境の実現を目指します。「カミングアウトを強要しない」「アウティング(本人同意のない暴露)は重大なハラスメントとして懲戒対象」を大前提とします。

### 4-2. 同性パートナー社内手続平等化

2022年4月より、同性パートナー(自治体パートナーシップ証明書または当社所定の宣誓書を提出した者)を、人事制度上、法律婚配偶者と同等に取り扱います。具体的には次の制度すべてで平等適用しています。

- 結婚祝金・結婚休暇(5日)
- 配偶者帯同転勤・帯同赴任制度
- 慶弔休暇・忌引休暇・慶弔見舞金
- 家族手当・配偶者手当
- 健康診断の家族健診割引
- 社宅・住宅補助の世帯適用
- 育児休業(パートナーの子の養育を含む)
- 介護休業(パートナー・パートナーの親の介護を含む)
- 福利厚生クラブ・保養所の家族利用

法律婚配偶者と取扱に差が残るのは、税制上・社会保険上の制約(配偶者控除、健康保険被扶養者認定等)による事項のみです。

### 4-3. 性別記載・トイレ・更衣室への配慮

- **社員番号・社内システム**:旧姓使用・通称使用を全システムで可とし、性別欄は「無回答」「自認の性」を選択可。
- **トイレ**:本社・主要事業所にオールジェンダートイレ(誰でも利用可)を各フロア1か所以上設置。
- **更衣室**:個室更衣ブースを整備。性別による利用区分は本人自認に基づく。
- **健康診断**:健診車・健診クリニックへの同行希望、項目選択(子宮頸がん検診・前立腺検査等)を本人選択制とし、強制しない。
- **服装規定**:制服のある事業所では男女選択制、ネクタイ・スカートは任意。
- **採用エントリーシート**:性別欄を任意・自由記述に変更(2023年新卒採用より)。

### 4-4. All Ally(オールアライ)研修

性的マイノリティ当事者を支援する「アライ(Ally)」を全社員から募り、自発的なネットワーク「Sample Ally Network」(2026年4月時点で全社員の38%が登録)を運営しています。

- 全社員必修:LGBTQ+基礎研修(eラーニング45分)
- 管理職必修:アライシップ・マネジメント研修(集合研修3時間)
- 新任管理職:アンコンシャス・バイアス×SOGIE 4時間ワークショップ
- 年次イベント:6月のPride月間に「Sample Pride Week」を開催(社内講演、レインボーカラーロゴ、社外パレード参加支援)
- アライ・ステッカー配布:希望者は自席・社員証ホルダーにアライ表示を装着可

### 4-5. 相談窓口

SOGIESCに関する相談は、属性問わず以下の窓口で受け付けます。すべて秘密厳守、アウティング厳禁、相談したことを理由とする不利益取扱は禁止です。

- 人事部 D&I推進室 SOGI担当:内線5163 / sogi@sample.co.jp(担当者2名・うち1名は当事者)
- ハラスメント相談窓口「Speak Up Sample」:社内ポータル(匿名可)
- 社外専門窓口:認定NPO法人ReBit「にじいろ電話相談」(当社契約により無料)
- 緊急時:健康管理ホットライン 080-XXXX-XXXX(24時間)
