## 1. 副業の方針
<!-- section_id: sec_1 -->

株式会社サンプル(以下「当社」)は、2020年4月の就業規則改定により、社員の副業・兼業を原則として容認する方針へ転換しました。多様なキャリア観・働き方を尊重し、社員が社外で得る知見・人脈・スキルを当社の競争力にも還流させていく狙いです。本ガイドは、副業・兼業を希望する社員と所属長が遵守すべき手続・基準・健康管理ルールを定めるものです。

### 1-1. 副業を認める意義

当社では副業を「個人のキャリア自律」と「企業の競争力強化」の両立を実現する重要な人事施策と位置付けています。具体的には次の3つの効果を期待しています。

1. **スキル・知見の獲得**:本業だけでは得られない業種・職種・規模の異なる経験を通じ、社員の視野・専門性を拡張する。
2. **イノベーション創出**:社外ネットワークから得た情報・課題感を本業に持ち帰り、新規事業・業務改善のヒントとする。
3. **多様な働き方の実現**:育児・介護・治療と仕事の両立、地域貢献、起業準備など、社員のライフステージに応じた選択肢を提供する。

### 1-2. 4つの基本原則

副業・兼業は次の4原則を満たす範囲で認められます。

| 原則 | 内容 |
| --- | --- |
| **本業優先** | 当社における職務遂行・成果が最優先。副業による遅刻・欠勤・パフォーマンス低下は認めない。 |
| **健康配慮** | 過重労働による健康被害を防ぐため、総労働時間・睡眠時間の自己管理を徹底する。 |
| **機密保持** | 当社の営業秘密・顧客情報・未公表情報を副業先で利用・開示しない。退職後も同様。 |
| **競業回避** | 当社と競合する事業・顧客に対する業務は原則禁止(詳細は3章)。 |

### 1-3. 対象者と対象外

| 区分 | 取扱 |
| --- | --- |
| 正社員(試用期間終了後・勤続1年以上) | 申請により可 |
| 正社員(試用期間中・勤続1年未満) | 原則不可(本業習熟を優先) |
| 契約社員・嘱託社員 | 申請により可 |
| 役員 | 取締役会の個別決議が必要(会社法第356条) |
| 派遣社員 | 派遣元会社の規程による |

なお、不動産賃貸(相続物件等)、株式・投資信託の運用、執筆・講演等の単発の謝金収入(年間50万円以下)は副業申請の対象外とします。

## 2. 申請と承認
<!-- section_id: sec_2 -->

副業を開始する前に、必ず「副業・兼業承認申請書」を提出し、所属長および人事部の承認を得る必要があります。事後申請・無申請での副業は就業規則違反として懲戒対象となり得ます。

### 2-1. 申請フロー

```
[本人] 申請書記入(社内ポータル「副業申請システム」)
   ↓ (即時)
[所属長] 業務影響・競業性の一次審査(目安5営業日)
   ↓ (承認)
[人事部労務グループ] 法令・規程適合性の二次審査(目安5営業日)
   ↓ (承認)
[本人] 承認通知受領後、副業開始可
```

申請から承認まで通常2週間程度を要します。海外企業との契約、競業性の判断が難しい案件は法務部レビューが追加され、最大1か月程度かかる場合があります。

### 2-2. 申請書記載事項

申請書には以下の項目を漏れなく記載してください。

- 副業先の名称・所在地・業種・規模
- 契約形態(雇用/業務委託/個人事業/法人代表 等)
- 業務内容(具体的に。「コンサルティング」等の抽象表現は不可)
- 就業時間帯・想定月間労働時間
- 報酬体系(時給/月額/成果報酬)と概算月収
- 契約期間(開始日・終了予定日)
- 競業性の自己評価(当社事業との関連有無)
- 機密保持についての自己宣誓

### 2-3. 承認の有効期間と更新

承認は1年間有効です。継続する場合は毎年4月に「副業継続申請書」を提出し、更新審査を受けてください。副業先・業務内容・労働時間に重大な変更があった場合は、年度途中でも変更申請が必要です。

### 2-4. 副業終了時の届出

副業を終了した場合(契約満了、自主終了、副業先都合いずれも)、終了後10営業日以内に「副業終了届」を提出してください。終了後も機密保持義務は退職時と同等に継続します。

## 3. 禁止事項
<!-- section_id: sec_3 -->

以下に該当する副業・兼業は申請を承認しません。承認後に該当事実が判明した場合は、副業中止命令および懲戒処分の対象となります。

### 3-1. 競合他社・競合事業での就労

当社の事業セグメント(プロダクト事業、プロフェッショナルサービス事業、グローバル事業、新規事業)と直接競合する企業・サービスでの就労は禁止します。具体的には、競合と認定される企業リスト(社内ポータル「コンペティター・リスト」、四半期更新)に記載された企業での副業は不可です。判断に迷う場合は、申請前に人事部または法務部に事前相談してください。

### 3-2. 機密情報を利用する業務

当社の顧客情報・営業秘密・技術ノウハウ・人事情報・未公表業績等を直接的・間接的に利用する業務は禁止します。たとえば、当社顧客に対するコンサルティング、当社開発技術を活用した類似製品開発、当社社員のヘッドハント仲介などが該当します。

### 3-3. 勤務時間中の副業

当社の所定労働時間中(コアタイム、リモート勤務時間を含む)に副業業務を行うことは厳禁です。有給休暇取得日に副業を行うこと自体は禁止しませんが、有休の本来目的(心身回復・私事)に照らして節度ある利用を求めます。

### 3-4. ブランド・信用毀損のおそれがある業務

当社の社会的信用・ブランドイメージを損なうおそれのある業務(風俗営業、ギャンブル関連、マルチ商法、特定の政治・宗教活動への営利的従事、反社会的勢力との関わりがある業務等)は禁止します。

### 3-5. 深夜・長時間労働を伴う業務

健康確保の観点から、以下のいずれかに該当する副業は不承認とします。

- 深夜帯(22:00〜翌5:00)を含む常態的な労働
- 副業のみで月45時間を超える労働時間
- 本業と副業の合計が法定上限(月100時間未満、複数月平均80時間以内)を超える見込み

### 3-6. NG例とOK例

| 例 | 判定 | 理由 |
| --- | --- | --- |
| 競合SIerの土日プロジェクト参画 | NG | 競業 |
| 当社顧客A社への個人コンサル | NG | 機密利用・競業 |
| 自身のYouTubeチャンネルでの料理動画配信 | OK | 競業・機密ともに無関係 |
| 地域コミュニティのIT講師(月4回・夜2時間) | OK | 健康影響軽微、社会貢献性高い |
| ベンチャー企業の技術顧問(週8時間) | 要審査 | 競業性・労働時間を個別判断 |
| 飲食店の深夜アルバイト(週5日23時〜) | NG | 深夜長時間・健康配慮 |

## 4. 労働時間通算と健康管理
<!-- section_id: sec_4 -->

労働基準法第38条第1項により、複数の事業場で労働する場合の労働時間は通算されます。当社の労働時間管理および健康確保措置は次のとおりです。

### 4-1. 自己申告制による労働時間管理

副業先での労働時間は本人による自己申告制を採用します。月初(毎月5日まで)に前月分の副業労働時間を「副業労働時間報告システム」に入力してください。雇用契約の場合は実労働時間、業務委託の場合は実作業時間を申告します。

### 4-2. 月次総労働時間モニタリング

人事部労務グループは、本業の労働時間と副業の申告時間を合算し、以下のしきい値で個別フォローを実施します。

| 月間総労働時間 | 対応 |
| --- | --- |
| 200時間未満 | 通常モニタリング |
| 200〜220時間 | 所属長から本人へ注意喚起 |
| 220〜240時間 | 産業医面談の案内(任意) |
| 240時間超 | 産業医面談必須・副業時間短縮指示 |
| 連続2か月で80時間超の時間外労働 | 副業一時停止命令 |

### 4-3. 健康診断・産業医面談

副業の有無にかかわらず、本業での定期健康診断(年1回)・特定業務従事者健診(年2回)の受診は必須です。副業に従事する社員は、健診結果について産業医のコメントを必ず確認してください。長時間労働者面談(月80時間超)の対象となった場合、面談を拒否することはできません。

### 4-4. 社会保険・労災・税務の取扱

副業形態によって取扱が異なるため、必ず以下の整理を理解してください。

| 項目 | 雇用契約による副業 | 業務委託/個人事業による副業 |
| --- | --- | --- |
| 健康保険・厚生年金 | 主たる事業所(通常は本業)で加入。副業先で要件該当時は二以上事業所勤務届を提出。 | 本業のみで加入。 |
| 雇用保険 | 主たる賃金を受ける事業所のみ。 | 該当なし。 |
| 労災保険 | 双方の事業場で適用(2020年9月以降、両事業場の賃金合算で給付基礎日額算定)。 | 本業のみ。副業中の事故は労災対象外。 |
| 確定申告 | 副業先給与20万円超で確定申告必要。 | 所得20万円超で確定申告必要。 |
| 住民税 | 「自分で納付(普通徴収)」選択により本業会社への通知を避けることが可能。 | 同左。 |

確定申告および税務上の取扱について不明点がある場合、社外EAP「ピースマインド」の税理士相談(年2回まで会社負担)を活用してください。

### 4-5. トラブル発生時の連絡

副業先でのトラブル(契約解除、報酬未払、ハラスメント、事故等)が本業に影響しうる場合、速やかに人事部労務グループ(内線5152)に相談してください。副業先での問題は原則本人の責任で解決すべきですが、当社の信用に関わる事案については会社として対応を検討します。
