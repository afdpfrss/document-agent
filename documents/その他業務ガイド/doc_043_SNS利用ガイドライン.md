## 1. 基本原則
<!-- section_id: sec_1 -->

SNS(ソーシャルネットワーキングサービス)は、X(旧Twitter)、Facebook、Instagram、LinkedIn、TikTok、YouTube、note、ブログ、口コミサイト、社内外のSlack/Teams等、不特定または特定多数に情報を発信・共有しうる媒体すべてを指します。本ガイドラインは、株式会社サンプル(以下「当社」)の役員・社員・契約社員・業務委託者・派遣社員・インターン等(以下総称して「役職員」)が、業務上・私生活上を問わずSNSを利用する際の基本姿勢を定めるものです。

### 1-1. 個人発信は個人責任、ただし社員としての影響を自覚する

SNSにおける個人としての発信は、表現の自由として尊重されます。しかし、ひとたび「サンプル社員」と特定された場合、その投稿は当社の評判・株価・採用ブランドに直接影響します。「自分は一個人として発信している」という主張は、SNS上ではほぼ通用しません。投稿前に「これが新聞一面に載っても、家族・顧客・上司に見られても問題ないか」を自問してください(SNSテン・セカンド・ルール)。

### 1-2. 所属明示時の追加配慮

プロフィール欄に「株式会社サンプル勤務」「SAMPLE Corporation」等の所属明示をする場合、または投稿の文脈から明らかに当社所属と推認される場合、以下の追加配慮を求めます。

- 政治・宗教・社会的に対立のある話題について断定的・攻撃的な発信を避ける
- 業務に関する発信は「個人の見解であり、所属組織を代表しない」旨を明記する(ディスクレーマー)
- 当社の競合・顧客・取引先に対する批判的言及を避ける
- アイコン・ヘッダー画像に当社ロゴ・商標を許可なく使用しない

### 1-3. 著作権・肖像権・パブリシティ権

他者が制作した文章・写真・動画・楽曲・イラスト等を投稿に利用する場合、著作権法に基づく適法な利用(引用要件の充足、ライセンス付与、パブリックドメイン等)が必要です。生成AIによる画像・テキストであっても、学習元データの権利関係や利用規約に注意してください。

人物が写った写真・動画の投稿には、原則として被写体本人の同意が必要です。同僚・顧客・イベント参加者を撮影・投稿する場合は事前許可を取得し、子どもが映る場合は保護者の同意を得てください。

### 1-4. 業務時間中の私的SNS利用

業務時間中の私的なSNS閲覧・投稿は、職務専念義務(就業規則第22条)に違反します。休憩時間中の私的利用は禁止しませんが、社用端末・社内Wi-Fiでの利用ログは情報システム部によって記録されている点に留意してください。

## 2. 禁止事項
<!-- section_id: sec_2 -->

以下の行為は、媒体・公開範囲(全公開/フォロワー限定/鍵アカ/DM)を問わず一切禁止します。違反は就業規則第57条以下に基づく懲戒対象となり、損害が発生した場合は民事上の賠償請求対象ともなり得ます。

### 2-1. 社外秘情報・未公表業績

| カテゴリ | NG例 |
| --- | --- |
| 財務情報 | 「来週の決算、過去最高益らしい!」「今期は減配の噂」 |
| M&A・事業提携 | 「うちが○○社を買収するって本当?」 |
| 人事情報 | 「○○部長が来月クビらしい」「次期役員候補は△△さん」 |
| 製品ロードマップ | 「次の新製品、来年Q2にこんな機能で出ます」 |
| 採用情報(未公表) | 「来春の内定者数、過去最多になった」 |

これらは金融商品取引法上のインサイダー情報や、公正取引委員会の調査対象となり得る情報を含みます。「自分のフォロワーだけだから」「鍵アカだから」は理由になりません。スクリーンショットで拡散され、結果として当社株価に影響したケースが多数報告されています。

### 2-2. 顧客情報・取引情報

顧客名・契約金額・打合せ内容を特定可能な形で発信することは、守秘義務違反に直結します。

- NG:「今日は○○銀行さんで提案。担当の佐藤部長が乗り気だった」
- NG:オフィス会議室から見える顧客資料・ホワイトボードを背景に自撮り
- OK:「金融業界のお客様向け提案。手応えあり!」(個社特定不可)

### 2-3. 他者への誹謗中傷・差別表現

特定の個人・団体・属性(性別、年齢、国籍、人種、宗教、性的指向、障がい、出身等)に対する誹謗中傷、差別、ハラスメント発言は厳禁です。「ネタ」「冗談」「リツイートのみ」も同様に対象となります。

- NG:「○○社の営業マジで使えない」「△△出身の人は仕事できない」
- NG:差別的なミーム・コラ画像のリポスト
- NG:競合企業の製品を貶める比較投稿(景品表示法・不正競争防止法に抵触する場合あり)

### 2-4. 著作物の無断利用

- NG:漫画・アニメ・映画のスクショに自分のコメントを添えた投稿(引用要件未充足)
- NG:報道記事の全文転載
- NG:他人が撮影した写真を出典明示なくアイコンに使用
- OK:出典・著作者を明記し、自分の論評が主、引用が従となる正当な引用

### 2-5. 社内資料・社内施設の撮影投稿

執務エリア、会議室、データセンター、研究開発室の写真・動画の社外公開は禁止です。社内ポスター・配布資料・PC画面・付箋・ホワイトボード・名札等が写り込んだ画像も同様です。広報部の事前承認を得た採用広報写真等を除きます。

## 3. 炎上時の対応
<!-- section_id: sec_3 -->

「炎上」とは、SNS上の投稿に対し短時間で批判・否定的反応が殺到し、拡散していく状態を指します。当社では、役職員個人または公式アカウントが炎上の当事者となった場合、初動の速さが被害規模を左右すると認識し、以下のフローで対応します。

### 3-1. 発見後30分以内に広報部報告

自身の投稿・他者からの言及が炎上の兆候(1時間で否定的リプライ・引用50件以上、または影響力のあるアカウントからの批判的言及)を示した場合、発見後**30分以内**に広報部クライシス対応窓口(internal: 5401 / crisis@sample.co.jp / 24時間対応Slackチャンネル #crisis-sns)に第一報を入れてください。家族・友人・同僚が当事者で、本人が連絡できない状況のときは、気付いた人が代理で連絡します。

報告フォーマット(第一報の最低限):①誰の投稿か、②URL、③投稿時刻、④概略内容、⑤現在の拡散規模(リポスト数・コメント数)、⑥本人の現状(連絡可否)。

### 3-2. 拡散防止と証拠保全

第一報後、当事者は次の行動を取ります。

1. **追加投稿の停止**:反論・釈明・補足を含め、広報部の指示があるまで一切投稿しない
2. **証拠保全**:該当投稿、批判リプライ、影響力のあるアカウントの引用について、削除前にスクリーンショット(URL・日時が分かる形)とブラウザ保存を実施
3. **アカウント保護**:鍵化はしない(隠蔽と見られる)。ただしDMの開放は閉じる
4. **私的連絡の停止**:友人・家族からの問い合わせには「会社と相談中」とのみ回答

### 3-3. 削除・訂正・謝罪の判断フロー

広報部・法務部・人事部・所属事業部で構成する「クライシス対応会議」が、原則発生後3時間以内に判断を下します。

```
[投稿内容の事実確認]
   ↓
[事実誤認あり?] ─Yes→ [訂正投稿・お詫び投稿] → [当該投稿削除]
   ↓No
[当社の社会的責任を問われる内容?] ─Yes→ [公式謝罪・対応方針発表]
   ↓No                                          ↓
[投稿維持・静観]                          [必要に応じ記者会見・取材対応]
```

判断軸:①事実誤認の有無、②法令違反の有無、③第三者の人権・名誉侵害の有無、④当社のブランド・株価への影響度、⑤当事者の心理的負荷。

### 3-4. 個人判断での反論・再投稿の禁止

「自分は悪くない」「言いたいことがある」と感じても、広報部の判断を経ずに反論投稿・引用反論を行うことは厳禁です。多くの炎上事案で、当事者の不用意な反論が二次炎上の起点になっています。事後の振り返り(ポストモーテム)は人事部・広報部とともに必ず実施し、再発防止を図ります。

## 4. 公式アカウント運用
<!-- section_id: sec_4 -->

当社は、コーポレートブランド・採用・IR・各事業の情報発信のため、SNS公式アカウントを運用しています。アカウント乱立による品質低下とブランド毀損を防ぐため、以下の運用統制を行います。

### 4-1. 公式アカウントの定義と承認

「公式アカウント」とは、当社名・事業名・商品名を冠した、または当社を代表して情報発信する全てのSNSアカウントを指します。新規開設には、広報部長の承認(申請から1か月程度)が必要です。承認なく開設されたアカウントは「非公式」として運用停止指示の対象となります。

### 4-2. 運用責任者と投稿レビュー

各公式アカウントには「運用責任者(管理職)」「運用担当者(2名以上)」「広報部担当者」を必ず置きます。投稿前のレビュー体制は次の通りです。

| 投稿カテゴリ | レビュー体制 | 投稿可能時間 |
| --- | --- | --- |
| 通常投稿(製品情報・コラム) | 担当者起案 → 運用責任者承認 | 平日9:00〜18:00推奨 |
| キャンペーン・プロモーション | 上記 + 広報部 + 法務部レビュー | 同上 |
| IR関連(業績・適時開示連動) | 上記 + IR部 + 経営企画部承認 | 適時開示後1時間以内 |
| クライシス時の情報発信 | 経営会議・広報部長承認 | 随時 |

### 4-3. FAQ対応・コメント対応SLA

公式アカウントへのリプライ・DM・コメントには、以下のSLA(Service Level Agreement)で対応します。

- お褒め・お礼:24時間以内に「いいね」または短文返信
- 質問:48時間以内に一次回答(回答困難時はその旨を返信)
- クレーム・トラブル報告:2時間以内に確認の返信、24時間以内に対応方針回答
- 緊急性のある安全情報・障害報告:即時(15分以内)に広報部・所管部署へ連携

### 4-4. KPIと効果測定

公式アカウントごとに以下のKPIを四半期で測定し、運用責任者がレビュー会議で報告します。

| KPI | 測定指標 | 目標(コーポレート公式の例) |
| --- | --- | --- |
| リーチ | インプレッション/月 | 100万以上 |
| エンゲージメント | エンゲージメント率 | 2.0%以上 |
| ファン獲得 | フォロワー純増数/月 | 1,000以上 |
| 採用貢献 | 当社採用サイト遷移数 | 月3,000以上 |
| ブランド | センチメント(ポジ比率) | 70%以上 |

### 4-5. 危機管理時の運用停止判断

社会的に重大な事件・事故・災害が発生した場合、当社の通常広告・プロモーション投稿が「不謹慎」と受け取られるおそれがあります。広報部長は次の事象発生時、全公式アカウントの一時運用停止を判断します。

- 震度6弱以上の地震、大規模災害発生
- 当社事業に重大インシデント発生(個人情報漏えい、製品事故、コンプライアンス違反 等)
- 著名人の急逝、社会全体が追悼ムードにある状況

予約投稿が稼働している場合、運用担当者は速やかに停止対応を行い、再開判断は広報部長の指示に従ってください。
