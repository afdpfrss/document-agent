## 1. 方針
<!-- section_id: sec_1 -->

### 第1条(経営者によるコミットメント)
株式会社サンプル(以下「当社」という。)は、お客様、お取引先様、株主、従業員その他の利害関係者からお預かりした情報資産、並びに当社が事業活動の遂行において取り扱う一切の情報資産を、あらゆる脅威から適切に保護し、その機密性(Confidentiality)、完全性(Integrity)及び可用性(Availability)を維持することが、社会的責任の遂行及び事業の継続的発展に不可欠であると認識する。

代表取締役社長は、JIS Q 27001:2023(ISO/IEC 27001:2022)に準拠した情報セキュリティマネジメントシステム(以下「ISMS」という。)を構築、導入、運用、監視、レビュー、維持及び継続的改善することをここに宣言し、必要となる経営資源(人員、予算、設備、教育機会)を継続的に投入することを約束する。

### 第2条(基本理念)
当社の情報セキュリティに関する基本理念は以下のとおりとする。
1. 情報資産は当社の事業継続に不可欠な経営資源であり、これを適切に管理することは経営の最重要課題である。
2. 情報セキュリティは情報システム部門のみの責任ではなく、当社に所属する全ての役員及び従業員、並びに業務委託先を含む全関係者の責任である。
3. 情報セキュリティは「予防」「検知」「対応」「復旧」のサイクルにより継続的に向上させるものとする。
4. リスクベースアプローチを採用し、特定された情報セキュリティリスクに対し、回避・低減・移転・受容のいずれかの方針を経営判断として明確に選択する。
5. 「Security by Design」「Privacy by Design」の理念に基づき、新規事業・新規システム導入の企画段階からセキュリティ要件を組み込む。

### 第3条(遵守すべき法令等)
当社は、情報セキュリティに関連する以下の法令、規範、契約上の義務を遵守する。
- 個人情報の保護に関する法律(令和3年改正、令和5年改正反映)
- サイバーセキュリティ基本法
- 不正アクセス行為の禁止等に関する法律
- 不正競争防止法(営業秘密管理)
- マイナンバー法(行政手続における特定の個人を識別するための番号の利用等に関する法律)
- 電子帳簿保存法、e-文書法
- EU一般データ保護規則(GDPR)、英国GDPR、カリフォルニア州消費者プライバシー法(CCPA/CPRA)
- 業界ガイドライン(経済産業省「サイバーセキュリティ経営ガイドライン Ver3.0」、IPA「情報セキュリティ10大脅威」対応指針等)
- お客様及びお取引先様との間で締結する秘密保持契約(NDA)、データ処理契約(DPA)

## 2. 適用範囲
<!-- section_id: sec_2 -->

### 第4条(物理的適用範囲)
本方針及び本方針に基づく下位規程・手順書(以下「ISMS文書」と総称する。)は、当社の以下の事業場及び施設に適用する。
- 本社(東京都港区)
- 大阪支社、名古屋支社、福岡支社
- 国内データセンター(東日本リージョン主系、西日本リージョン副系)
- 利用するパブリッククラウド環境(AWS、Microsoft Azure、Google Cloud)上に構築された当社管理下の論理的環境
- 在宅勤務、サテライトオフィス、出張先等の社外就労環境(リモートアクセス経由のアクセスを含む)

### 第5条(人的適用範囲)
本方針は、以下の全ての者に適用する。
1. 当社の取締役、監査役、執行役員その他の役員
2. 当社と雇用契約を締結する全ての従業員(正社員、契約社員、嘱託、パートタイマー、アルバイト)
3. 当社に派遣される派遣労働者
4. 当社の業務を受託する委託先、再委託先の役職員(現地常駐者、リモート従事者を問わない)
5. 当社施設に一時的に立ち入る訪問者(該当範囲のルールに限る)

### 第6条(対象情報資産)
本方針が保護対象とする情報資産は、媒体・形態を問わず以下を含む。
- 電子データ(構造化データ、非構造化データ、ログデータ、ソースコード、設計書等)
- 紙文書、マイクロフィルム、写真、図面等の物理媒体
- 情報システム(ハードウェア、ソフトウェア、ネットワーク機器、クラウドサービス、SaaS)
- 知的財産(特許、ノウハウ、営業秘密)及び役職員の知識・経験
- 第三者から預託された情報資産

### 第7条(認証範囲)
当社はISO/IEC 27001:2022認証(認証番号:IS XXXXXX、認証機関:●●審査登録機構)、ISO/IEC 27017:2015(クラウドサービス)、ISO/IEC 27701:2019(プライバシー情報マネジメント)を取得しており、認証範囲は適用宣言書(Statement of Applicability, SoA)に明示する。

## 3. 体制
<!-- section_id: sec_3 -->

### 第8条(ISMS推進体制)
当社のISMS推進体制は以下のとおり構成する。

| 役割 | 任命者 | 主たる責任 |
|------|--------|-----------|
| 情報セキュリティ統括責任者(CISO) | 代表取締役が役員より任命 | ISMS全体の最終責任、経営層への報告、重大インシデントの最終判断 |
| ISMS委員会委員長 | CISOが兼任 | 委員会運営、年次計画承認 |
| ISMS事務局長 | 情報システム部長 | ISMS文書管理、リスクアセスメント主導、内部監査計画 |
| 個人情報保護管理者(CPO) | 法務部長 | 個人情報保護法・GDPR遵守監督 |
| CSIRT責任者 | 情報システム部セキュリティ課長 | インシデント対応指揮、外部CSIRT連携(JPCERT/CC、NISC) |
| 部門情報セキュリティ責任者 | 各部部長 | 所管部署の情報資産管理、教育徹底、リスク低減策実施 |
| 情報資産管理責任者 | 各課課長 | 情報資産台帳維持、機密度区分付与、ライフサイクル管理 |
| 利用者(全従業員) | ─ | 規程遵守、インシデント報告、教育受講 |

### 第9条(ISMS委員会)
ISMS委員会は四半期に1回以上開催し、以下を審議する。
- 情報セキュリティリスクアセスメント結果のレビュー
- セキュリティインシデント発生状況及び是正処置の進捗
- 内部監査・外部審査の指摘事項対応状況
- 教育計画及び受講状況
- ISMS文書の制定・改定
- 投資計画(セキュリティ製品導入、外部診断費用等)

### 第10条(マネジメントレビュー)
代表取締役は、年1回(原則として5月)マネジメントレビューを実施し、ISMSの適切性、妥当性、有効性を評価し、改善方針を決定する。インプットには、内部監査結果、外部審査結果、KPI達成状況、利害関係者からの意見、リスク状況の変化を含む。

### 第11条(外部連携)
当社は以下の外部組織と連携し、最新の脅威情報を収集するとともに、有事の際の支援体制を確保する。
- 一般社団法人JPCERTコーディネーションセンター(JPCERT/CC)
- 独立行政法人情報処理推進機構(IPA)
- 日本シーサート協議会(NCA)
- 所轄警察署サイバー犯罪対策課
- 個人情報保護委員会
- 外部フォレンジック調査会社(年間契約)

## 4. 目標
<!-- section_id: sec_4 -->

### 第12条(年度情報セキュリティ目標)
当社は、当年度(2026年度)の情報セキュリティ目標として以下のKPIを設定し、ISMS委員会において四半期ごとに進捗を測定する。

| 指標カテゴリ | KPI項目 | 目標値 | 測定方法 |
|-------------|---------|--------|---------|
| インシデント | 重大インシデント(L1〜L2)発生件数 | 0件/年 | CSIRT管理台帳 |
| インシデント | 報告SLA(検知から1次報告まで)遵守率 | 100% | チケットタイムスタンプ |
| 教育 | 全社情報セキュリティ教育受講率 | 100%(年1回) | LMS受講記録 |
| 教育 | 標的型攻撃メール訓練の開封率 | 5%未満 | 訓練ツール集計 |
| 脆弱性管理 | Critical脆弱性のSLA(7日以内対応)遵守率 | 100% | 脆弱性管理台帳 |
| 脆弱性管理 | High脆弱性のSLA(30日以内対応)遵守率 | 95%以上 | 脆弱性管理台帳 |
| 監査 | 内部監査是正処置完了率(期日内) | 100% | 監査管理表 |
| 監査 | 外部審査における重大不適合 | 0件 | 審査報告書 |
| アクセス管理 | 退職者ID無効化SLA(72時間以内)遵守率 | 100% | ID管理ログ |
| 委託先管理 | 委託先セキュリティチェック実施率 | 100%(年1回) | 委託先評価記録 |
| BCP | 事業継続演習の実施回数 | 年2回以上 | BCP訓練記録 |

### 第13条(継続的改善)
当社は、PDCAサイクル(Plan-Do-Check-Act)に基づき、ISMSの有効性を継続的に改善する。前年度のKPI達成状況、新たな脅威動向、技術革新、法令改正、事業環境の変化を踏まえ、毎年度の目標を見直す。

### 第14条(違反時の措置)
本方針及び下位規程に違反した役職員に対しては、就業規則第●●条(懲戒)及び「懲戒規程」の定めに従い、譴責、減給、出勤停止、降格、諭旨退職又は懲戒解雇等の懲戒処分を行うことがある。また、当社が損害を被った場合は、別途損害賠償を請求することがある。委託先による違反については、契約に基づき契約解除、損害賠償請求、再発防止計画の提出等の措置を行う。
