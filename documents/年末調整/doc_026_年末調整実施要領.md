## 1. 対象者
<!-- section_id: sec_1 -->

株式会社サンプル(以下「当社」)における年末調整は、所得税法第190条に基づき、給与等の支払を受ける居住者に対して、毎月の源泉徴収税額と年間の正しい所得税額との過不足を精算する手続である。本要領は、対象者の範囲、特殊ケースの取扱いを明確にする。

### 1.1 対象者の原則
当年12月31日時点において当社に在籍する給与所得者(役員を含む)で、給与所得者の扶養控除等(異動)申告書を提出している者を年末調整の対象とする。具体的には以下に該当する者。
1. 1年を通じて勤務している者
2. 年の途中で就職し、年末まで勤務している者(中途入社者)
3. 年の途中で海外勤務などにより非居住者となり、再び居住者となって年末まで勤務している者
4. 年の途中で死亡退職した者(死亡時に年末調整を実施)
5. 著しい心身障害により退職し、本年中の再就職が困難な者
6. 12月支給給与の受給後に退職した者
7. パート従業員等で給与年収103万円以下の見込みで退職した者

### 1.2 対象外となる者
所得税法上、以下に該当する者は年末調整の対象外となり、自身で確定申告(翌年2月16日〜3月15日)を行う必要がある。
1. **給与年収が2,000万円超**(所得税法第190条但書)
2. 災害減免法により、その年の給与所得に対する源泉徴収の猶予・還付を受けた者
3. 2か所以上から給与の支払を受けており、他社で扶養控除等申告書を提出している者
4. 非居住者(年末時点で日本に住所を有しない者)
5. 日雇労働者(継続的給与でなく日々雇い入れられる者)
6. 副業の事業所得・不動産所得・譲渡所得等が20万円超ある者(これは年末調整は通常通り実施するが確定申告も必要)

### 1.3 中途入社者の取扱い
本年中に他社から転職し当社に入社した者は、前職分の給与・源泉徴収税額・社会保険料を合算して当社で年末調整を行う必要がある。前職源泉徴収票の提出は必須であり、未提出の場合は当社で年末調整できず、本人が確定申告を行うこととなる。
- 提出期限:11月15日(年末調整書類締切日)
- 紛失時:前職へ再発行依頼(税法上、退職後1ヶ月以内に交付義務あり)
- 前職が休業中で発行困難な場合:本人が税務署に「源泉徴収票不交付の届出書」を提出

### 1.4 育休・産休・休職者の取扱い
- **育休・産休中**:当年中に1日でも給与支給があれば対象。傷病手当金・出産手当金は所得税非課税のため算入しない
- **休職中(無給)**:給与支給がない場合も対象。社会保険料控除・生命保険料控除等は申告可能
- **長期休職復帰者**:復帰後初の給与計算月に源泉徴収を再開、年末調整は通常通り実施

### 1.5 役員の取扱い
取締役・執行役員も給与所得者として年末調整対象。ただし役員報酬年額2,000万円超は対象外(自身で確定申告)。

## 2. 必要書類
<!-- section_id: sec_2 -->

### 2.1 申告書(全員提出)
当社の年末調整では、以下3種類の申告書を全員に記入・提出を求める。SmartHR上で電子フォーム提出が可能。
1. **給与所得者の扶養控除等(異動)申告書**(扶養控除、障害者控除、寡婦・ひとり親控除、勤労学生控除)
2. **給与所得者の基礎控除申告書 兼 配偶者控除等申告書 兼 所得金額調整控除申告書**(略称:基配所申告書)
3. **給与所得者の保険料控除申告書**(該当者のみ証明書添付)

### 2.2 該当者のみ提出
- **給与所得者の住宅借入金等特別控除申告書**(住宅ローン控除2年目以降)
- **前職源泉徴収票**(本年中の中途入社者)

### 2.4 主要な控除額(2026年分)
所得税法・租税特別措置法に基づく主要控除額は以下のとおり。

| 控除項目 | 控除額 |

| 給与所得控除 | 給与収入162.5万円以下で55万円、最低保証額。最大195万円(収入850万円超) |
| 配偶者控除 | 38万円(70歳以上の老人控除対象配偶者は48万円) |
| 配偶者特別控除 | 配偶者の合計所得に応じ最大38万円 |
| 扶養控除(一般) | 38万円 |
| 扶養控除(特定:19歳以上23歳未満) | 63万円 |
| 扶養控除(老人:70歳以上) | 48万円(同居老親等58万円) |
| 障害者控除 | 27万円(特別障害者40万円、同居特別障害者75万円) |
| 寡婦控除 | 27万円 |
| ひとり親控除 | 35万円 |
| 勤労学生控除 | 27万円 |
| 所得金額調整控除 | 給与収入850万円超で23歳未満扶養親族等あり等の場合に適用 |

### 2.5 マイナンバーの取扱い
扶養控除等申告書には、本人および配偶者・扶養親族のマイナンバー記載が必要。SmartHR上で過年度に登録済の場合は自動連携。新規登録時は本人確認書類(マイナンバーカードの写し、または通知カード+運転免許証等)の添付が必須。マイナンバー情報は当社「特定個人情報取扱規程」に基づき厳格管理する。

## 3. 提出方法と期限
<!-- section_id: sec_3 -->

### 3.1 提出方法:SmartHR電子申告
当社は2023年度よりSmartHRの年末調整機能を全社導入し、原則として電子提出としている。

**(1) 電子申告の手順**
1. 11月1日:SmartHRより全社員に依頼メール配信
2. 社員はマイページにログイン
3. 質問形式(ウィザード)で各項目を回答(扶養家族の追加・削除、保険料控除等)
4. 控除証明書をスマートフォンで撮影しアップロード(または保険会社のマイナポータル連携で自動取得)
5. 内容確認・電子署名で提出
6. 人事部が内容点検、不備があれば差戻
7. 修正後再提出

**(2) マイナポータル連携**
2020年以降、生命保険・地震保険・iDeCo・住宅ローンの一部について、マイナポータル経由で控除証明書データをXML形式で自動取得できる。手入力・撮影アップロードが不要となり、誤入力リスクも低減する。当社では推奨利用としている。

### 3.2 紙提出の特例
以下に該当する場合のみ、紙申告書での提出を認める。
1. デジタル機器の操作が困難で、上長が認めた者
2. 海外駐在等でSmartHRアクセスに支障がある者
3. 産休・育休等で長期間ログインできない者

紙申請の場合、当社所定の申告書をダウンロード・印刷し、必要事項記入・押印の上、控除証明書原本を添付し、人事部宛て社内便または郵送(書留推奨)で提出する。

### 3.3 提出期限

| 期限 | 内容 |
|---|---|
| 11月1日 | SmartHR配信開始、申告受付開始 |
| 11月15日 | **第一次締切**(中途入社者の前職源泉徴収票含む) |
| 11月25日 | **最終締切**(差戻し再提出含む) |
| 12月5日 | 年税額計算 |
| 12月10日 | 監査・チェック |
| 12月25日 | 還付/追徴反映の給与振込 |
| 1月31日 | 給与支払報告書を市区町村提出 |
| 1月31日 | 法定調書を税務署提出 |

最終締切(11月25日)を過ぎた場合、原則として当社年末調整は実施できず、本人が確定申告(翌年2月16日〜3月15日)を行う必要がある。

### 3.4 申告内容の修正
12月給与計算前(12月10日まで)であれば、申告内容の修正は受け付ける。それ以降、翌年1月31日までに修正が必要となった場合は、再年末調整(法令上「年末調整のやり直し」)を実施できる。1月31日以降の修正は、本人が確定申告にて対応する。

## 4. 還付・追徴
<!-- section_id: sec_4 -->

### 4.1 精算の基本
年末調整の精算は、原則として12月25日支給給与にて反映する。年税額が源泉徴収済合計額を下回る場合は「還付」、上回る場合は「追徴」となる。

### 4.2 還付の方法
還付額は、12月支給給与に上乗せして給与振込口座へ振り込む。給与明細にも還付額を明示する。
- 還付額の典型例:生命保険料控除上限活用、扶養親族追加、住宅ローン控除適用などにより数万円〜十数万円の還付となるケースが多い
- 還付額が大きく(目安50万円超)、12月給与1回で全額還付しきれない場合は、翌1月給与に繰越して還付する

### 4.3 追徴の方法
追徴額は、12月支給給与から控除する。給与明細に追徴額を明示する。
- 追徴の典型例:扶養親族の所得超過(配偶者の収入が見込みを超えた等)、賞与査定により年収が想定を上回ったケース
- 追徴額が大きく(目安5万円超)、本人の生活に影響する場合は、人事部と事前協議の上、12月・1月・2月の3回分割控除を認める

### 4.4 源泉徴収票の交付
精算完了後、源泉徴収票を以下のタイミングで電子交付する(所得税法第226条)。
- 12月25日:給与明細とともにSmartHR上で電子交付
- 紙交付希望者:人事部に申請、12月最終営業日までに郵送

源泉徴収票は、住宅ローン審査・保育園入園手続・確定申告・社会保険手続等で頻繁に求められるため、本人で適切に保管すること。

### 4.5 確定申告との関係
以下のケースでは、年末調整完了後も確定申告が必要または有利となる。
1. **医療費控除**:本人および生計を一にする親族の医療費が年間10万円超(または所得の5%超)
2. **寄附金控除**:ふるさと納税(6自治体以上または医療費控除等と併用)、認定NPO・公益法人等への寄付
3. **住宅ローン控除1年目**:初年度は確定申告必要(2年目以降は年末調整)
4. **雑損控除**:災害・盗難等による損失
5. **副業所得20万円超**:給与以外の所得が20万円を超える場合
6. **特定支出控除**:給与所得者の特定支出(研修費・資格取得費等)が一定額超

### 4.6 住民税の取扱い
年末調整結果に基づき、当社が翌年1月31日までに各市区町村へ「給与支払報告書」を提出。市区町村は5月までに住民税額を決定し、6月支給給与から特別徴収(天引)を開始する。住民税の年税額は5月下旬に配布される「住民税決定通知書」で本人通知する。

### 4.7 過誤発覚時の対応
年末調整完了後に申告漏れ・誤りが発覚した場合、対応は以下のとおり。
- **翌年1月31日まで**:当社で再年末調整可能(人事部に連絡)
- **2月1日以降**:本人で確定申告(または更正の請求)を行う
- **追加で還付が見込まれる場合**:確定申告で還付請求(5年遡及可能)
- **追加で納税が必要な場合**:速やかに修正申告(放置で延滞税・加算税)
