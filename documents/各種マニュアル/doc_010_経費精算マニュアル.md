## 1. 精算対象
<!-- section_id: sec_1 -->

株式会社サンプルにおける経費精算は、業務遂行上必要かつ社会通念上相当な支出を対象とします。本章では、科目別の精算可否・上限・特記事項を整理します。

### 1.1 科目別精算可否一覧

| 科目コード | 科目名 | 内容 | 上限・基準 | 事前申請 |
| :--- | :--- | :--- | :--- | :--- |
| 511 | 旅費交通費(国内) | 電車・バス・タクシー・自家用車燃料 | タクシーは22時以降または公共交通機関不可時のみ | 不要 |
| 512 | 旅費交通費(出張) | 新幹線・航空・宿泊 | 出張規程に準拠 | 必要 |
| 521 | 会議費 | 社内/社外会議の飲食、会議室外部レンタル | 1人あたり3,000円 | 不要 |
| 522 | 接待交際費 | 取引先との会食・贈答 | 1人あたり10,000円(超過は事前承認) | 10,000円超は必要 |
| 531 | 消耗品費 | 文具、トナー、雑貨 | 1件30,000円未満 | 30,000円以上は必要 |
| 532 | 書籍代 | 業務関連書籍、有料記事 | 月10,000円目安 | 不要 |
| 541 | 研修受講料 | 外部セミナー、資格受験料 | 都度上長判断 | 必要 |
| 551 | 通信費 | 在宅勤務手当外の業務通話 | 実費 | 不要 |
| 591 | 諸会費 | 業界団体年会費 | 部門予算内 | 必要 |

### 1.2 精算対象外(自己負担)

1. 通勤定期券区間内の電車賃(別途通勤手当で支給済)。
2. 私的飲食・贈答(家族行事、個人的会合)。
3. 駐車違反金、交通反則金等の罰金。
4. 業務外での個人的タクシー利用。
5. 私物の購入(イヤホン、私用バッグ等)。

> **注意:** 30,000円(税込)を超える支出は、原則として支出前に「事前申請書(Concur 申請区分:PRE-EXP)」を起票し、上長承認を得てから発注・購入してください。事後申請は支払拒否の対象となります。

> **FAQ:** Q. クライアント先までの直行・直帰時の交通費はどう精算しますか?  A. 自宅最寄駅〜訪問先の往復実費から、通常通勤区間の運賃を差し引いた差額を精算してください。Concur の「直行直帰」テンプレに自動計算機能があります。

## 2. 必要書類
<!-- section_id: sec_2 -->

精算には領収書または適格請求書(インボイス)が必須です。電子帳簿保存法(電帳法)スキャナ保存制度に対応するため、紙領収書は原則7営業日以内にスキャンしConcurへアップロードしてください。

### 2.1 領収書の要件チェックリスト

| No. | 項目 | 必須/任意 |
| :--- | :--- | :--- |
| 1 | 発行年月日 | 必須 |
| 2 | 発行者(店名・会社名)と住所 | 必須 |
| 3 | 宛名「株式会社サンプル」 | 30,000円以上で必須 |
| 4 | 金額(税込・税抜・消費税の区別) | 必須 |
| 5 | 但し書き(商品・サービス内容) | 必須(「お品代」NG) |
| 6 | 適格請求書発行事業者の登録番号(T+13桁) | 仕入税額控除を受ける場合は必須 |
| 7 | 軽減税率(8%)対象品目の明示 | 該当時必須 |

> **NG例:** 但し書きが「お品代として」のみ → 経理差戻し対象。再発行を依頼するか、購入明細(レシート)を併せて添付してください。
>
> **OK例:** 「会議用ホワイトボードマーカー(黒・赤・青 各2本)」のように具体的に記載されている。

### 2.2 接待交際費の追加書類

接待交際費(科目522)は、税務上の証憑要件を満たすため次の情報を「接待実施報告書(Concur 添付:ENT-RPT-01)」に必ず記載してください。

1. 日時・場所(店名)
2. 参加者氏名・所属(社内・社外を区別、社外は会社名・役職も)
3. 参加人数(社内×名、社外×名)
4. 目的・効果(新規取引開拓、契約交渉、関係維持等)
5. 1人あたり金額(自動計算)

### 2.3 電帳法・インボイス対応

| 制度 | 当社対応 | 社員アクション |
| :--- | :--- | :--- |
| 電子取引データ保存義務 | Concur が要件を満たすシステムとして指定 | メール領収書はPDF保存し原本をConcurへアップ |
| スキャナ保存制度 | タイムスタンプ自動付与 | スキャン後の紙領収書は経理部へ月次提出 |
| 適格請求書(インボイス) | 仕入税額控除に必要 | 登録番号なし領収書は「免税事業者」フラグを立てる |

> **注意:** 紙領収書を破棄する前に、Concurへのアップロード完了通知メールを必ず確認してください。タイムスタンプ未付与のまま原本破棄すると経理差戻しとなり、最悪の場合自己負担となります。

## 3. システム入力
<!-- section_id: sec_3 -->

経費精算は SAP Concur Expense(以下「Concur」)で行います。SSO認証(Google Workspace)で sample.concursolutions.com にログインしてください。

### 3.1 新規申請の基本手順

1. Concur トップ画面の **[+ 新規] > [経費レポート作成]** をクリック。
2. レポート名(例:「2026/05 営業部 田中 経費」)、レポート用途(国内出張・通常経費等)、開始日・終了日を入力し **[次へ]**。
3. **[経費を追加]** で個別明細を入力。科目・日付・支払先・金額・通貨・支払方法を順に入力。
4. 領収書画像を **[領収書を添付]** からアップロード(JPG/PNG/PDF、1ファイル5MBまで)。
5. すべての明細追加後、**[送信]** をクリック → 承認者(直属上長)を選択 → **[送信完了]**。

### 3.2 入力項目の標準値

| 項目名 | 必須/任意 | 記入例 |
| :--- | :--- | :--- |
| 取引日 | 必須 | 2026/05/15 |
| 経費タイプ | 必須 | 511 旅費交通費(国内) |
| 取引先 | 必須 | JR東日本 |
| 支払方法 | 必須 | 自己立替/法人カード/Suica |
| 金額(税込) | 必須 | 1,520 |
| 通貨 | 必須 | JPY |
| 摘要(目的) | 必須 | A社訪問(本社→渋谷往復) |
| 案件コード | プロジェクト紐付け時必須 | PRJ-2026-042 |
| 出張ID | 出張経費時必須 | TRP-2026-018 |

### 3.3 Suica/PASMO の取込

1. Concur モバイルアプリ(iOS/Android)を起動し、ホーム > **[交通系ICカード読込]** をタップ。
2. iPhone はFelica、Android はNFCでカードをかざす(物理カードのみ。モバイルSuicaは「楽天モバイル/Apple Pay連携」を選択)。
3. 直近20件の利用履歴が表示されるので、業務利用分のみチェック → **[経費に追加]** をタップ。
4. 自動的に「511 旅費交通費」として明細化される。摘要欄を必ず手動で補記すること(例:「B社訪問」)。

> **NG例:** 通勤区間と業務移動を区別せず全件取込 → 通勤手当との二重支給となり、経理差戻し+遡及調整対象。
>
> **OK例:** 取込後に通勤区間と重なる行を削除し、業務移動のみ残す。

### 3.4 エラー時の対処

| エラーメッセージ | 原因 | 対処 |
| :--- | :--- | :--- |
| 「領収書が読み取れません」 | 画像の解像度不足 | 300dpi以上で再スキャン |
| 「承認者が設定されていません」 | 組織マスタ未更新 | 人事部に組織反映を依頼(jinji@sample.co.jp) |
| 「予算超過」 | 部門予算残不足 | 部門予算管理者の事前承認を取得 |
| 「インボイス番号形式不正」 | T+13桁になっていない | 取引先に再発行依頼、または免税事業者フラグを立てる |

## 4. 承認フロー
<!-- section_id: sec_4 -->

経費精算は申請者→1次承認→2次承認(該当時)→経理部最終確認→振込の順に進みます。

### 4.1 役職別承認権限表

| 金額(税込) | 1次承認 | 2次承認 | 3次承認 |
| :--- | :--- | :--- | :--- |
| 1万円未満 | 直属上長(課長) | - | - |
| 1万円〜10万円未満 | 課長 | 部長 | - |
| 10万円〜50万円未満 | 課長 | 部長 | 本部長 |
| 50万円〜100万円未満 | 課長 | 部長 | 本部長+経理部長 |
| 100万円以上 | 課長 | 部長 | 本部長+CFO |

### 4.2 標準処理日数

1. 1次承認:申請後3営業日以内
2. 2次承認:1次承認後2営業日以内
3. 経理部最終確認:全承認完了後3営業日以内
4. 振込:毎月15日締め、翌月25日(銀行休業日の場合は前営業日)に給与口座へ振込

### 4.3 差戻し時の対応

承認者または経理部から差戻しがあった場合、Concur から自動でメール・Slack通知が届きます。

1. Concur にログインし **[マイレポート] > [差戻し]** タブを開く。
2. コメント欄で差戻し理由を確認。
3. 該当明細を修正後、再度 **[送信]**。
4. 差戻し件数が月3件を超えると人事部にアラートが飛びます(評価対象ではないが運用改善対象)。

### 4.4 代理承認

承認者が出張・休暇等で5営業日以上不在となる場合、Concur の **[代理人設定]** から事前に代理承認者を指名します。代理承認権限は同等職位以上に限定されます。

> **注意:** 「とりあえず承認」は禁止です。承認者は領収書原本・摘要・案件コードを必ず確認してください。承認者責任で不適切支出を見落とした場合、承認者にも責任が及びます。

## 5. 注意事項
<!-- section_id: sec_5 -->

本章では、よくあるトラブルと回避策、税務・コンプライアンス上の重要事項を整理します。

### 5.1 申請期限

| 種別 | 期限 |
| :--- | :--- |
| 通常経費 | 支出日から30日以内 |
| 出張精算 | 帰任日から10営業日以内 |
| 海外出張精算 | 帰任日から15営業日以内 |
| 年度跨ぎ案件 | 3月20日までに当年度分を申請 |

> **注意:** 期限超過は原則自己負担となります。やむを得ない事情(長期病欠等)がある場合は、申請時に「期限超過理由書」を添付し経理部長承認を得てください。

### 5.2 タクシー利用基準

| 状況 | 利用可否 | 必要事項 |
| :--- | :--- | :--- |
| 22時以降の業務終了 | 可 | 摘要に終業時刻を記載 |
| 始発前の出張移動 | 可 | 摘要に列車時刻を記載 |
| 重量物・機密物の運搬 | 可 | 摘要に運搬物を記載 |
| 体調不良時 | 可 | 上長Slack事前連絡 |
| 単なる時間短縮目的 | 不可 | - |
| 同方向の同僚と相乗り推奨 | - | GO/Uberの相乗り機能を活用 |

### 5.3 立替限度・法人カード推奨

1人あたりの立替総額が月20万円を超えると見込まれる場合は、法人クレジットカード(三井住友VISA法人カード)の申請を強く推奨します。申請は経理部(keiri@sample.co.jp)宛に「法人カード発行申請書」を提出してください(発行まで2〜3週間)。

### 5.4 上限超過・特例承認

科目上限を超過する支出が必要な場合は、Concur の **[特例承認申請]** から起票し、本部長以上の事前承認を取得してください。承認なしの超過支出は半額自己負担となる場合があります。

> **NG例:** 取引先との会食で1人あたり15,000円使ったが、事前承認なし → 上限10,000円超過分5,000円は本人負担。
>
> **OK例:** 重要顧客との会食で1人15,000円が見込まれるため、事前にConcur特例承認を取得 → 全額会社負担。

### 5.5 よくあるFAQ

> **FAQ:** Q. 領収書を紛失しました。  A. 5,000円未満の公共交通機関(電車・バス)はConcurの「領収書なし申請(摘要に経路明記)」が可能です。それ以外は「支払証明書(店舗印付き)」を取得するか、「経費支払申立書」を起票して上長承認を得てください(月2件まで)。

> **FAQ:** Q. 海外出張で外貨建ての領収書はどう入力しますか?  A. 取引日時点の三菱UFJ銀行TTMレート(月初公表値)で円換算します。Concurは自動換算機能あり、原通貨と為替レートを記載すれば自動計算されます。
