## 1. 事前申請
<!-- section_id: sec_1 -->

株式会社サンプルにおける出張は、業務上必要な訪問・調査・研修・会議参加等を目的とし、出張規程(社内ポータル参照)および本マニュアルに従って実施します。すべての出張は **事前申請・上長承認** が必須です。

### 1.1 出張の定義

| 区分 | 距離・時間 | 該当例 |
| :--- | :--- | :--- |
| 国内出張(日帰り) | 片道50km超 または 4時間超 | 名古屋日帰り商談 |
| 国内出張(宿泊) | 自宅・勤務地を離れて宿泊 | 大阪1泊2日訪問 |
| 海外出張 | 国境を越える業務移動 | 米国出張1週間 |
| 出張外 | 50km以内かつ4時間以内 | 都内営業活動 |

### 1.2 申請リードタイム(最低)

| 区分 | 最低リードタイム | 推奨リードタイム |
| :--- | :--- | :--- |
| 国内出張(日帰り) | 3営業日前 | 1週間前 |
| 国内出張(宿泊) | 1週間前 | 2週間前 |
| 海外出張 | 2週間前 | 1ヶ月前 |
| 海外出張(ビザ要) | 1ヶ月前 | 2ヶ月前 |

> **注意:** リードタイムを満たさない申請は原則差戻となります。緊急時のみ、上長+本部長の事前合意を取得したうえで「緊急出張」として申請してください。

### 1.3 承認権限表

| 区分 | 概算費用 | 1次承認 | 2次承認 | 3次承認 |
| :--- | :--- | :--- | :--- | :--- |
| 国内日帰り | 3万円未満 | 課長 | - | - |
| 国内宿泊 | 3〜10万円 | 課長 | 部長 | - |
| 国内宿泊(長期) | 10万円超 | 課長 | 部長 | 本部長 |
| 海外(アジア) | 〜30万円 | 課長 | 部長 | 本部長 |
| 海外(欧米) | 30〜80万円 | 課長 | 部長 | 本部長+CFO |
| 海外(長期) | 80万円超 | 課長 | 部長 | 本部長+CFO+人事部 |

### 1.4 申請手順(Concur Travel)

1. SAP Concur(sample.concursolutions.com)にログイン。
2. **[出張] > [新規出張申請]** をクリック。
3. 入力項目:

| 項目 | 必須 | 例 |
| :--- | :--- | :--- |
| 出張ID | 自動採番 | TRP-2026-0185 |
| 出張区分 | 必須 | 国内宿泊 |
| 出発日・帰任日 | 必須 | 2026/06/01〜2026/06/03 |
| 出発地 | 必須 | 東京(本社) |
| 訪問先・宿泊地 | 必須 | 大阪、京都 |
| 訪問先企業・担当者 | 必須 | A社/田中様 |
| 目的・期待効果 | 必須 | 新規取引商談、契約締結見込み |
| 同行者 | 任意 | 山田、佐藤 |
| 概算費用 | 必須 | 交通6万+宿泊3万+日当0.6万=9.6万円 |
| 案件コード | プロジェクト紐付け時必須 | PRJ-2026-042 |

4. **[送信]** → 承認ルートに沿って自動回付。
5. 全承認完了後、Concur Travelで予約手配開始可能。

### 1.5 出張命令書

承認完了時にConcurから **出張命令書(PDF)** が自動発行されます。出張中は携行(電子可)し、災害・事故時の身分証として活用してください。

### 1.6 日当(出張手当)

| 区分 | 一般職 | 管理職 | 役員 |
| :--- | :--- | :--- | :--- |
| 国内日帰り | 2,000円 | 3,000円 | 4,000円 |
| 国内宿泊 | 3,000円/日 | 5,000円/日 | 7,000円/日 |
| 海外(アジア) | 5,000円/日 | 7,000円/日 | 10,000円/日 |
| 海外(欧米) | 7,000円/日 | 10,000円/日 | 15,000円/日 |

日当は食事・雑費・現地交通の合算とみなされ、別途領収書添付は不要です(税務上の益金算入なし)。

## 2. 交通・宿泊予約
<!-- section_id: sec_2 -->

交通・宿泊予約はBTM(Business Travel Management)契約先である **楽天ビジネス・出張手配センター(以下「BTM」)** 経由を原則とします。Concur Travelから一気通貫で予約可能です。

### 2.1 予約手段の優先順位

| 優先 | 方法 | 利用ケース |
| :--- | :--- | :--- |
| 1 | Concur Travel(BTM連携) | 標準。請求一括化・履歴管理可 |
| 2 | 楽天ビジネスダイレクト | Concur障害時、特殊宿泊 |
| 3 | 出張手配センター(電話) | 海外複雑経路、緊急時 |
| 4 | 個人立替手配 | 上記不可時のみ、要事前承認 |

### 2.2 交通機関グレード基準

| 距離・時間 | 推奨 | 一般職 上限 | 管理職 上限 | 役員 上限 |
| :--- | :--- | :--- | :--- | :--- |
| 新幹線 | のぞみ等指定席 | 普通車指定席 | 普通車指定席 | グリーン車 |
| 在来線(片道2時間超) | 特急指定席 | 普通車指定席 | 普通車指定席 | グリーン車 |
| 国内航空(2時間以内) | エコノミー | エコノミー | エコノミー | エコノミー |
| 海外航空(6時間以内) | エコノミー | エコノミー | エコノミー | プレミアムエコノミー |
| 海外航空(6〜10時間) | エコノミー/プレエコ | エコノミー | プレエコ | ビジネス |
| 海外航空(10時間超) | プレエコ/ビジネス | プレエコ | ビジネス | ビジネス/ファースト |

> **注意:** 上限を超えるグレードを希望する場合(例:健康上の理由でビジネス必要)は、医師診断書または特例承認(本部長)を取得してください。

### 2.3 宿泊基準

| 地域 | 一般職 上限(1泊・税込) | 管理職 上限 | 役員 上限 |
| :--- | :--- | :--- | :--- |
| 東京23区・大阪市 | 15,000円 | 20,000円 | 30,000円 |
| 国内主要都市 | 12,000円 | 17,000円 | 25,000円 |
| 国内地方 | 10,000円 | 15,000円 | 22,000円 |
| 海外(アジア) | 20,000円 | 25,000円 | 40,000円 |
| 海外(欧米主要都市) | 30,000円 | 40,000円 | 60,000円 |

繁忙期(年末年始、GW、お盆、国際イベント開催時)で上限超過が見込まれる場合、BTMから代替提案がない限り上限超過分を会社負担します(理由をConcurに記載)。

### 2.4 予約手順(Concur Travel)

1. Concur > **[出張] > [予約]** をクリック。
2. **[フライト]/[ホテル]/[新幹線・JR]/[レンタカー]** タブから検索。
3. 出発地・目的地・日時・人数等を入力。
4. 検索結果から選択 → **[予約]**。
5. 出張IDに自動紐付け、予約手配料も自動精算。

### 2.5 マイル・ポイント

| ポイント種別 | 個人帰属 | 注意事項 |
| :--- | :--- | :--- |
| 航空会社マイル | 可 | 業務利用が条件、私的利用部分は申告 |
| ホテルポイント | 可 | 同上 |
| 楽天ポイント等経由ポイント | 不可(会社帰属) | BTM契約に基づく |
| クレジットカードポイント | 法人カード使用時は会社帰属 | 個人カード立替分は個人帰属 |

### 2.6 自己手配時の注意

やむを得ず個人で手配する場合は、必ず事前にConcurで **「自己手配」フラグ** を立てて承認を得てください。事後申請は半額しか精算されない場合があります。

> **NG例:** Concur申請せず自分でホテルを予約、帰任後に「ついでに精算したい」と申告 → 規程上、半額または不可。
>
> **OK例:** Concurで「自己手配」と申請し、上長承認後に予約 → 全額精算可。

## 3. 海外出張準備
<!-- section_id: sec_3 -->

海外出張は安全管理・法令遵守の観点で国内とは異なる準備が必要です。本章のチェックリストを必ず実施してください。

### 3.1 準備チェックリスト(出発前2週間〜)

| No. | 項目 | 担当 | 期限 |
| :--- | :--- | :--- | :--- |
| 1 | パスポート残存期間6ヶ月以上 | 本人 | 2ヶ月前 |
| 2 | ビザ取得(必要国) | 本人(総務部支援) | 1ヶ月前 |
| 3 | 海外旅行傷害保険付保 | 総務部(会社契約) | 1週間前 |
| 4 | GeIN(Global e-Itinerary Notification)登録 | 本人 | 1週間前 |
| 5 | 渡航国リスクレベル確認 | 本人 | 1週間前 |
| 6 | 必要予防接種 | 本人(産業医相談) | 1〜2ヶ月前 |
| 7 | 国際電話可能なスマホ手配 | IT部 | 1週間前 |
| 8 | 現地通貨両替・トラベルプリペイドカード | 本人 | 数日前 |
| 9 | 名刺英語版印刷 | 総務部発注 | 2週間前 |
| 10 | 危機対応キット受領 | 総務部 | 出発前 |
| 11 | 緊急連絡先一覧 | 本人作成・上長共有 | 出発前 |

### 3.2 ビザ

| 国 | 短期商用ビザ | 取得期間目安 |
| :--- | :--- | :--- |
| 米国 | ESTA(オンライン)or B1ビザ | ESTA:即日、B1:1ヶ月 |
| 中国 | M(商務)ビザ | 1〜2週間 |
| インド | e-Business Visa | 1週間 |
| EU諸国 | シェンゲン圏短期査免(90日) | - |
| 英国 | 短期商用査免(90日) | - |
| 韓国 | 査免 | - |
| シンガポール | 査免 | - |

ビザ申請は **総務部・グローバル業務課(global-affairs@sample.co.jp)** に相談。代行業者(JTBグローバル)と連携。

### 3.3 海外旅行傷害保険

会社契約の包括保険(AIG法人契約)に自動加入されます。補償内容は以下:

| 補償項目 | 補償額 |
| :--- | :--- |
| 傷害死亡・後遺障害 | 5,000万円 |
| 治療費用 | 無制限 |
| 救援者費用 | 1,000万円 |
| 賠償責任 | 1億円 |
| 携行品損害 | 30万円 |

付保証明書(英文)はConcurから出張IDで発行可能。出国時に必ず携行(または電子保存)してください。

### 3.4 GeIN(渡航情報登録)

会社の安全管理のため、すべての海外出張者は **GeIN(社内システム、Notion DB)** に渡航情報を登録します。

| 項目 | 内容 |
| :--- | :--- |
| 渡航期間 | 出国日〜帰国日 |
| 渡航国・都市 | 全立寄国 |
| フライト便名 | 往復・乗継含む |
| 宿泊先(電話番号含む) | ホテル名・住所・電話 |
| 現地連絡先(本人) | 現地携帯番号 |
| 現地連絡先(取引先) | 緊急時のホスト連絡先 |
| 緊急時の家族連絡先 | 国内連絡先 |

### 3.5 リスクレベル確認

外務省「海外安全ホームページ(anzen.mofa.go.jp)」で渡航先のリスクレベルを確認します。

| レベル | 名称 | 出張可否 |
| :--- | :--- | :--- |
| レベル0 | 安全 | 通常承認 |
| レベル1 | 十分注意 | 通常承認、注意事項共有 |
| レベル2 | 不要不急の渡航中止 | 本部長承認必要 |
| レベル3 | 渡航中止勧告 | 役員承認+対策本部設置 |
| レベル4 | 退避勧告 | 渡航禁止 |

### 3.6 危機対応キット

総務部から **危機対応キット** を出発前に受領します。

1. 緊急連絡カード(24時間サポートデスクの電話番号)
2. 海外SOSサービス利用案内(AIG・International SOS)
3. 簡易応急セット(常備薬・絆創膏等)
4. 災害発生時の安否確認手順
5. 在外公館の連絡先リスト
6. 現地語(英・中・西等)の緊急フレーズ集

### 3.7 出張中の安否確認

1. 出張中、毎朝1回 **GeINチェックイン(Slack #travel-status)** を投稿。
2. 大規模災害・テロ・感染症発生時、人事部から **国際SMS+メール** で安否確認。72時間以内に応答必須。
3. 応答できない場合、家族連絡先・現地ホストへ会社が連絡。

> **注意:** スマホ盗難・紛失時は、SOSデスクへ電話 + 帰任後IT部へ報告(MDM経由でリモートワイプ)。

## 4. 帰任後手続き
<!-- section_id: sec_4 -->

出張は帰任後の振り返り・成果共有が完了して初めて価値が生まれます。期限内に手続きを完了してください。

### 4.1 期限一覧

| 手続き | 期限 |
| :--- | :--- |
| 帰任報告(Slack #prj-XXX) | 帰任翌営業日 |
| 出張報告書提出 | 帰任後5営業日以内 |
| 経費精算(Concur) | 帰任後10営業日以内(海外は15営業日) |
| 名刺整理・CRM登録 | 帰任後5営業日以内 |
| 成果共有(部内勉強会等) | 帰任後1ヶ月以内 |

### 4.2 出張報告書

Concur > **[出張] > [出張報告書作成]** から起票します。

| 項目 | 内容 |
| :--- | :--- |
| 出張ID | 自動連携(TRP-XXXX) |
| 出張期間・訪問先 | 自動連携 |
| 訪問先と面談相手 | 役職・氏名 |
| 議事概要 | 訪問先ごとに簡潔に記載 |
| 主な成果 | 商談進捗、契約見込、情報入手等 |
| 今後のアクション | 担当者・期限 |
| 添付資料 | 議事録、現地写真、配布資料 |

### 4.3 経費精算

経費精算は doc_010(経費精算マニュアル)に準拠。海外分は以下に注意:

1. 領収書の **円換算** は **三菱UFJ銀行TTMレート(月初公表値)** を使用。
2. 為替手数料・カード手数料は別科目「為替差損益」で精算可。
3. クレジットカード明細(法人カード)はConcurに自動取込。
4. 外貨現金両替の差額は領収書(両替時の交換レート明記)を添付。

### 4.4 名刺整理

1. 受領した名刺は **Sansan** にスキャン(モバイルアプリ or 本社1F受付スキャナ)。
2. Salesforce連携で取引先データに自動登録。
3. 個人連絡先(プライベート関係)は **個人スマホへの取込禁止**(情報持ち出し)。

### 4.5 成果共有

| 規模 | 共有方法 |
| :--- | :--- |
| 軽微 | Slack #部門名 で報告投稿 |
| 中 | 週次部門定例でプレゼン(15分) |
| 大(海外・大型商談) | 月次本部報告会(30分) |
| 戦略級 | 経営会議報告 |

報告資料は **Box > Sales > Travel Reports** に保存し、社内ナレッジ化します。

### 4.6 振り返り

帰任後1ヶ月以内に上長と15分の振り返り1on1を実施し、以下を確認します。

1. 出張目的の達成度
2. 想定外の発見・課題
3. 次回出張への改善点(行程・宿泊・現地支援等)
4. フォローアップアクションの進捗

> **NG例:** 帰任後に報告書を出さず、経費精算だけ提出 → 上長承認で差戻、組織知が蓄積されない。
>
> **OK例:** 帰任翌日に Slack で速報、5営業日以内に正式報告書、1ヶ月以内に成果プレゼン。

> **FAQ:** Q. 出張に有給休暇をくっつけて旅行を延長してもよいですか?  A. 可能です。Concurで「業務外延長」期間を明示し、その間の宿泊・食事・現地交通は完全自己負担となります。航空券差額があれば自己負担。出張保険は業務期間のみ適用。
