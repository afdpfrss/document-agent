## 1. ミッション
<!-- section_id: sec_1 -->

### 1-1. ミッション・ステートメント

> **「テクノロジーの力で、ビジネスと社会の未来を共創する。」**
>
> ― We co-create the future of business and society, powered by technology.

このミッションは、株式会社サンプル(以下「当社」)が存在する社会的意義を最も簡潔に言語化したものです。1985年の創業以来、当社の存在意義は「お客様と社会の課題に対し、テクノロジーという手段を通じて、共に答えを生み出す存在であり続けること」であると考えてきました。2020年のミッション再策定にあたり、社員1,200名のワークショップと、お客様・株主・取引先200名へのインタビューを経て、現在のステートメントに集約しました。

### 1-2. 各キーワードの含意

| キーワード | 含意 |
| --- | --- |
| **テクノロジーの力** | 技術はそれ自体が目的ではなく、人と社会を解放するための「力」である。最新技術への追従と同時に、技術の限界・倫理を理解し、責任ある実装を行う。 |
| **ビジネスと社会** | 当社の主たる顧客は企業であるが、その先には必ず人と社会がいる。短期的なビジネス成果と、長期的な社会的価値の両立を経営の前提とする。 |
| **未来** | 過去・現在の延長ではなく、未だ存在しない可能性を切り拓く態度を意味する。10年先、30年先を見据えた判断を是とする。 |
| **共創** | 「提供する/される」関係ではなく、お客様・パートナー・地域・社員と共に新しい価値をつくり出す関係。一方向の納品ではなく、相互変容を伴う協働。 |

### 1-3. 創業者の言葉

> 「ソフトウェア会社の本当の財産は、設備でも特許でもなく、お客様の業務を、お客様以上に深く理解しようとする態度にある。お客様の困りごとに、自分の家族のことのように身を乗り出せる人間が、何人いるか。それが会社の未来を決める。」
> ― 創業者 田中 信一(1985年・創業時の社員10名への訓示より)

このメッセージは、当社の文化的DNAとして40年以上受け継がれており、「顧客起点で考える」というValue #1「Customer First」の原典となっています。

### 1-4. ミッションの実装

ミッションは額縁に飾るための言葉ではなく、日々の意思決定の基準として用いられます。経営会議では、新規事業・大型投資・M&A・撤退判断のすべてにおいて「このミッションに照らしてどうか」が必ず議論されます。社員一人ひとりの目標設定(MBO)においても、職務目標と並んで「ミッション貢献」を1項目記載することを2024年から義務化しています。

## 2. ビジョン
<!-- section_id: sec_2 -->

### 2-1. 2030年長期ビジョン

> **「Be the Most Trusted Co-Creation Partner in Asia. ― アジアで最も信頼される共創パートナーへ。」**

2030年を到達目標とする長期ビジョンとして、上記を掲げています。「最大」「最強」ではなく「最も信頼される(Most Trusted)」を選んだのは、量的規模ではなく、お客様・パートナー・社会・社員から寄せられる「信頼」の総量こそが、長期的な企業価値の源泉であるという経営判断によります。地理的範囲を「アジア」としたのは、日本国内のみならず、ASEAN・南アジアまでを当社の事業領域として明確に位置付けるためです。

### 2-2. 数値目標(2030年)

| 指標 | 2025年実績 | 2030年目標 |
| --- | --- | --- |
| 連結売上高 | 1,648億円 | 3,500億円 |
| 海外売上比率 | 19.3% | 35.0% |
| 営業利益率 | 13.0% | 18.0% |
| ROE | 14.8% | 20.0% |
| 顧客NPS | +28 | +50 |
| 従業員エンゲージメント | 72pt | 80pt |
| GHG排出量(Scope 1+2) | △42%(2019比) | △60%(2019比) |

### 2-3. 中期経営計画「CSV2027」との接続

長期ビジョン2030の中間マイルストーンとして、中期経営計画「CSV2027(Co-create Sample Value 2027)」(2025〜2027年度)を策定しています。「CSV」は Creating Shared Value の頭文字を兼ね、経済的価値と社会的価値の同時創出を経営の中核に据える宣言でもあります。CSV2027の3つの戦略軸は次の通りです。

1. **プロダクト主導の成長加速**:SaaS ARR比率を50%以上に引き上げ、安定した収益構造を確立
2. **グローバル事業の再加速**:アジア4極(日・星・越・印)体制の完成、海外売上比率30%突破
3. **人と組織の進化**:D&I・健康経営・人的資本投資の継続強化、エンゲージメント80pt

CSV2027の進捗は、四半期ごとに統合報告書・投資家説明会・社内タウンホールで開示・共有しています。

### 2-4. ビジョンに込めた覚悟

ビジョンの達成は、現在の事業の延長線上では実現できません。2030年までに新規事業・グローバルが既存事業を上回るスピードで成長することを前提に、既存事業のキャッシュを未来に再投資する経営方針を取締役会で確認しています。短期業績の最適化に偏らず、長期ビジョン実現のための投資判断を、株主・社員に対して説明責任を果たしながら継続します。

## 3. バリュー(5つの価値観)
<!-- section_id: sec_3 -->

ミッションとビジョンを日々の行動に落とし込むための「判断基準」「行動原理」として、当社では5つのバリューを定めています。採用基準・人事評価・表彰制度の中心に置き、組織文化として根付かせています。

### 3-1. Value #1: Customer First(顧客起点)

**定義**:すべての判断の起点をお客様に置く。社内事情・短期利益・自分の都合より、お客様の成功と長期的価値を優先する。

| 体現する行動例 | 反する行動例 |
| --- | --- |
| お客様の業務現場まで足を運び、一次情報を取りに行く | 提案資料を社内会議だけで作り上げる |
| 「お客様にとってベストか?」と最終確認してから提案する | 自社プロダクトの売りやすさで提案する |
| 不利な事実・リスクを先に開示する | 都合の悪い情報を隠す/後出しする |
| お客様の成果が出るまで伴走する | 納品で関係を終わらせる |

### 3-2. Value #2: Ownership(主体性)

**定義**:自分の役割の枠を超え、組織全体の成果と顧客への約束に対して当事者として責任を持つ。

| 体現する行動例 | 反する行動例 |
| --- | --- |
| 「自分の担当ではない」課題にも一歩踏み出す | 「それは別部署の仕事」で止まる |
| 問題を発見したら原因究明と再発防止まで関与する | 報告だけで終わる |
| 上司の判断を待たずに、自らの権限で前進させる | 指示待ち・許可待ちで時間を費やす |
| 失敗を自分のものとして引き受け、学びにする | 言い訳・他責で終わる |

### 3-3. Value #3: Speed & Quality(俊敏性と品質)

**定義**:速さと品質はトレードオフではない。両立できる前提で仕組みと判断を設計する。

| 体現する行動例 | 反する行動例 |
| --- | --- |
| 80点で出して20点を改善する | 100点を目指して期限を逃す |
| 早い段階で関係者にドラフトを共有する | 完成まで誰にも見せない |
| プロセスをドキュメント化・自動化する | 属人化したまま放置する |
| 品質基準を明確に定義し、計測する | 「なんとなく良い」で判断する |

### 3-4. Value #4: Diversity & Respect(多様性と尊重)

**定義**:多様な属性・経験・意見の違いを組織の力に変える。異なる意見に敬意を持って耳を傾け、誰もが安心して発言・挑戦できる場をつくる。

| 体現する行動例 | 反する行動例 |
| --- | --- |
| 会議で発言の少ない人に意見を求める | 声の大きい人だけで議論を進める |
| 反対意見を歓迎し、建設的に深掘りする | 反対意見を「やる気がない」と切り捨てる |
| 自分と異なる背景の人にも積極的に学びに行く | 自分の常識を絶対視する |
| ハラスメント・差別的言動を見逃さず指摘する | 同調圧力・空気で流す |

### 3-5. Value #5: Integrity(誠実)

**定義**:誰も見ていない場面でも、法令・社内ルール・倫理に忠実に行動する。短期的利益のために、長期的信頼を犠牲にしない。

| 体現する行動例 | 反する行動例 |
| --- | --- |
| 不正を発見したら声を上げる(Speak Up) | 見て見ぬふりをする |
| 機密情報・個人情報を厳格に取り扱う | SNS・酒席で軽率に話す |
| 約束した期限・品質を守り、守れない場合は早期に伝える | 黙って遅延させる |
| 利益相反の可能性を自ら開示する | 隠したまま取引する |

## 4. 社員行動指針
<!-- section_id: sec_4 -->

### 4-1. サンプル・クレド10カ条

5つのバリューを日々の業務行動に翻訳した「サンプル・クレド」を、全社員のIDカード裏面に印刷し、入社時・期初・人事評価面談時に確認しています。

1. お客様の現場に答えがある。まず行き、聴き、見よ。
2. 「自分ごと」で考える。「誰かがやる」は、誰もやらない。
3. スピードは品質を上げる。完璧主義より、早い学習。
4. ドラフトを早く出せ。フィードバックを歓迎せよ。
5. 違いを面白がれ。同質性は組織を退化させる。
6. 反対意見に感謝せよ。空気は読まずに、本音を語れ。
7. 約束を守る。守れないときは、最も早く伝える。
8. 不正・差別・ハラスメントを見て見ぬふりはしない(Speak Up)。
9. 後輩の成長を、自分の評価より優先する瞬間を持て。
10. 仕事の意味を、お客様と社会から逆算して語れ。

### 4-2. 人事評価との連動

人事評価は「業績評価」(MBO達成度)と「行動評価」(バリュー体現度)の2軸で構成され、いずれも50:50のウェイトです(管理職は40:60で行動評価のウェイトが高くなります)。行動評価は5つのバリューそれぞれを5段階で評価し、上司による評価・本人による自己評価・360度フィードバック(同僚3名以上、部下2名以上)を組み合わせて総合判定します。バリュー体現が著しく低い場合は、業績が高くても昇格・昇給を見送るルールを2023年から導入しています。

### 4-3. 表彰制度「Sample Value Award」

四半期ごとに「Sample Value Award」を開催し、各バリューを最も体現した個人・チームを表彰しています。

| 表彰区分 | 受賞対象 | 副賞 |
| --- | --- | --- |
| Customer First Award | 顧客から最高の評価を得た個人・チーム | 賞金30万円、社長楯 |
| Ownership Award | 全社課題に主体的に取り組んだ個人 | 賞金20万円、海外研修権 |
| Speed & Quality Award | 業務プロセス改善・自動化に貢献した個人・チーム | 賞金20万円、技術カンファレンス参加権 |
| Diversity & Respect Award | インクルーシブな文化醸成に貢献した個人・チーム | 賞金20万円、社外研修権 |
| Integrity Award | 困難な状況で誠実な判断を貫いた個人 | 賞金30万円、CEO Dinner招待 |
| Grand Prix | 5つのバリュー全般で最も卓越した個人 | 賞金100万円、特別休暇1週間、社内全体表彰 |

年間総表彰人数は約120名(全社員の約2.5%)。受賞ストーリーは社内報・ポータル動画・経営層タウンホールで全社共有され、文化伝承の重要な機会となっています。

### 4-4. 1on1とバリュー対話

全社員に対し、上司との1on1を「最低週1回・30分以上」実施することを義務化しています。1on1では業務進捗だけでなく、「最近、どのバリューを意識した場面があったか」「バリューと現実が衝突した瞬間はあったか」を月1回必ず対話することを推奨しています。これにより、バリューが「飾り」ではなく「日々の判断軸」として機能するよう仕組み化しています。

### 4-5. バリューを守るための制度

- **Speak Up Sample**:不正・ハラスメント・倫理違反を匿名で通報できる窓口(社内・社外弁護士の二系統)
- **コンプライアンス研修**:全社員必修・年1回・eラーニング90分
- **トーン・アット・ザ・トップ**:社長・役員が四半期ごとにタウンホールでバリュー実践事例とディレンマを率直に共有
- **入社時オンボーディング**:新卒・中途問わず、入社後1か月のうち2日間を「ミッション・ビジョン・バリュー」プログラムに充当
