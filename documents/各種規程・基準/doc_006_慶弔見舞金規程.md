## 第1章 慶事
<!-- section_id: sec_1 -->

**第1条(目的)**
本規程は、株式会社サンプル(以下「当社」という。)の役員及び従業員の慶事、弔事、災害等に際し、当社が支給する祝金、弔慰金、見舞金等(以下総称して「慶弔見舞金」という。)の種類、金額、支給条件及び申請手続を定めることにより、福利厚生の充実と従業員相互の連帯感の醸成を図ることを目的とする。

**第2条(適用範囲)**
1. 本規程は、当社の役員、正社員、契約社員、嘱託社員、再雇用社員(以下「従業員等」という。)に適用する。
2. パートタイマー、アルバイト等の短時間労働者については、勤続6ヶ月以上の者に限り、本規程に定める金額の50%を支給する。
3. 業務委託契約者、派遣社員、出向受入者には適用しない。

**第3条(慶事祝金)**
従業員等が慶事に該当する事由により申請した場合、次の祝金を支給する。

| 事由 | 支給額 | 支給対象 |
|---|---|---|
| 本人の結婚 | 50,000円 | 婚姻届提出時。初婚・再婚を問わない(同一従業員2回目以降は30,000円)。 |
| 子の結婚 | 20,000円 | 本人の実子・養子(配偶者の連れ子を含む)の結婚。 |
| 本人の出産(配偶者の出産を含む) | 第1子 30,000円、第2子 50,000円、第3子以降1人につき 100,000円 | 双子等の複数出産は人数分支給。 |
| 銀婚式(結婚25周年) | 30,000円 | 入社後に迎える銀婚式に限る。 |
| 金婚式(結婚50周年) | 50,000円 | 入社後に迎える金婚式に限る。 |
| 子の入学(小学校) | 10,000円 | 第一子から第三子まで。 |

**第4条(勤続表彰)**
1. 当社は、長期にわたる勤続と功労に報いるため、毎年の創立記念日に勤続表彰を行う。
2. 勤続表彰の対象者及び表彰内容は次のとおりとする。

| 勤続年数 | 表彰金 | 副賞 |
|---|---|---|
| 勤続5年 | 30,000円 | 表彰状 |
| 勤続10年 | 100,000円 | 表彰状 + リフレッシュ休暇5日 |
| 勤続15年 | 150,000円 | 表彰状 |
| 勤続20年 | 200,000円 | 表彰状 + リフレッシュ休暇7日 |
| 勤続25年 | 250,000円 | 表彰状 |
| 勤続30年 | 300,000円 | 表彰状 + リフレッシュ休暇10日 + 記念品(時計等) |
| 勤続40年 | 500,000円 | 表彰状 + 記念品 |

3. 算定基準日は当該年度の4月1日とし、当日在籍する者を対象とする。
4. リフレッシュ休暇の取扱いは「休暇規程」(doc_003)第23条による。

**第5条(住宅取得祝金)**
従業員が居住用住宅を新築又は購入した場合、登記簿謄本の提出により30,000円を支給する。同一従業員につき1回限り。

**第6条(資格取得祝金)**
業務に関連する公的資格・社内認定資格を新規取得した場合、その難易度区分に応じて次の祝金を支給する(資格取得受験料・テキスト代等の補助は別途「自己啓発支援規程」による)。

| 難易度区分 | 祝金 | 例 |
|---|---|---|
| 特級 | 200,000円 | 公認会計士、税理士、弁護士、社会保険労務士、技術士、一級建築士、中小企業診断士 |
| 1級 | 50,000円 | 簿記1級、応用情報技術者、宅地建物取引士、第一種衛生管理者 |
| 2級 | 20,000円 | 簿記2級、基本情報技術者 |
| 語学 | TOEIC 800以上:30,000円、900以上:50,000円 | 受験料の補助とは別。 |

## 第2章 弔事
<!-- section_id: sec_2 -->

**第7条(弔慰金)**
従業員等本人又はその親族が死亡した場合、次の弔慰金を支給する。

| 対象 | 業務上死亡 | 業務外死亡 |
|---|---|---|
| 本人 | 5,000,000円 | 1,000,000円 |
| 配偶者(事実婚含む) | — | 100,000円 |
| 父母(義父母含む)・子 | — | 50,000円 |
| 祖父母・兄弟姉妹 | — | 30,000円 |
| 配偶者の父母・配偶者の祖父母・配偶者の兄弟姉妹 | — | 20,000円 |
| 孫 | — | 20,000円 |
| 同居の家族(上記以外) | — | 10,000円 |

1. 本人の業務上死亡については、別途「労働災害補償規程」に基づく遺族補償と併給する。
2. 弔慰金は香典とは別に支給する。香典については別表第2「香典・供花料規程」による。

**第8条(供花料)**
親族の葬儀に際し、次のとおり供花又は供花料を会社名義で手配する。

| 対象 | 供花料 |
|---|---|
| 本人 | 50,000円(花輪1基) |
| 配偶者・父母・子 | 20,000円(供花1基) |
| 祖父母・兄弟姉妹 | 15,000円(供花1基又は線香等) |
| 配偶者の父母 | 15,000円(供花1基) |
| その他親族 | 10,000円(線香等) |

家族葬・密葬その他遺族の意向により供花を辞退される場合は、供花料相当額を弔慰金に上乗せして支給する。

**第9条(忌引日数)**
忌引による特別休暇(有給)の日数は「休暇規程」(doc_003)第21条に準拠し、概要は次のとおりとする(忌引日数の起算日は死亡日又は本人が知った日のうち遅い方)。

| 対象 | 忌引日数 | 喪主加算 | 遠隔地加算 |
|---|---|---|---|
| 配偶者 | 7日 | +2日 | 片道300km以上で+1日 |
| 父母・子 | 5日 | +2日 | +1日 |
| 祖父母・兄弟姉妹・配偶者の父母 | 3日 | +2日 | +1日 |
| 孫・配偶者の兄弟姉妹・配偶者の祖父母 | 1日 | +1日 | +1日 |

**第10条(社葬の取扱い)**
1. 役員又は会社功労著しい従業員の死亡時、取締役会の決議により社葬を執行することができる。
2. 社葬にかかる費用は会社負担とし、別途「社葬取扱要領」に従う。

## 第3章 災害見舞金
<!-- section_id: sec_3 -->

**第11条(災害見舞金)**
従業員等本人又は同居家族の居住する住宅若しくは家財が、地震、津波、洪水、土砂崩れ、火災、爆発、暴風雨その他の天災・人災により被災した場合、次の災害見舞金を支給する。

| 罹災程度 | 見舞金額 |
|---|---|
| 全焼・全壊・全流失 | 500,000円 |
| 半焼・半壊 | 250,000円 |
| 一部損壊(損壊度20%以上) | 100,000円 |
| 床上浸水 | 150,000円 |
| 床下浸水 | 50,000円 |
| 家財の重大な損壊(損害額50万円以上) | 50,000円 |

1. 罹災程度の判定は、り災証明書(市区町村発行)又は被害状況写真、保険会社の調査報告書等により行う。
2. 複数の被害が同時に発生した場合は、最も高額の区分に該当する見舞金を支給する。

**第12条(傷病見舞金)**
1. 従業員等が業務外の傷病により連続15日以上欠勤又は休職した場合、次の傷病見舞金を支給する。
   - 入院を伴うもの:30,000円
   - 入院を伴わないが連続15日以上の欠勤・休職:20,000円
2. 同一傷病については1年に1回を限度とする。再発の場合は前回支給から1年経過後に再度支給可能。
3. 業務上災害による傷病については、別途「労働災害補償規程」による。

**第13条(犯罪被害見舞金)**
従業員等が、住居侵入、強盗、傷害その他の犯罪被害に遭った場合、その状況に応じ30,000円から200,000円の範囲で見舞金を支給することができる(人事担当役員の決裁による)。

**第14条(交通事故見舞金)**
従業員等が通勤途上又は私生活において重大な交通事故(自身が被害者である場合に限る)に遭い、3日以上の入院を要した場合、30,000円の見舞金を支給する。

**第15条(申請手続)**
1. 慶弔見舞金の支給を受けようとする者は、事由発生後速やかに、所定の「慶弔見舞金申請書」に必要な証憑(婚姻届受理証明書、母子健康手帳の写し、死亡診断書又は会葬礼状の写し、り災証明書、入院証明書等)を添付の上、所属長を経由して人事部に提出する。
2. 申請期限は事由発生から3ヶ月以内とする。やむを得ない事情がある場合は6ヶ月以内まで延長を認める。
3. 死亡退職等本人による申請が困難な場合は、人事部が遺族・関係者と連絡を取り、職権により支給手続を進める。

**第16条(支給時期)**
慶弔見舞金は、申請書受理後、原則として翌月の給与支給日と併せて本人(死亡の場合は受給権者)の指定口座へ振り込む。緊急性が高い場合(災害見舞金等)は別途現金支給又は前倒し振込を行う。

**第17条(他制度との関係)**
1. 健康保険、雇用保険、労災保険その他の公的給付、当社が加入する団体生命保険・GLTD等の保険給付は、本規程に基づく慶弔見舞金とは別に取り扱う。
2. 当社労使福祉会(共済会)からの給付がある場合は、当該給付と本規程による支給は重複して受給できる。

**第18条(税法上の取扱い)**
慶弔見舞金については、社会通念上相当な範囲内のものは原則として福利厚生費(非課税)として処理する。ただし、結婚祝金等の高額部分、業務上死亡弔慰金等で給与所得・退職所得・相続税の対象となる場合は、関係法令に従い適正に処理する。
