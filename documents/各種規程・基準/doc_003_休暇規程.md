## 第1章 年次有給休暇
<!-- section_id: sec_1 -->

**第1条(目的及び根拠)**
本規程は、株式会社サンプル(以下「当社」という。)の就業規則(doc_001)第22条以下の規定に基づき、従業員の各種休暇の付与基準、取得手続、給与の取扱い等を定め、もって労働基準法、育児・介護休業法、男女雇用機会均等法その他関係法令の遵守を図ることを目的とする。

**第2条(年次有給休暇の付与日数)**
1. 当社は、労働基準法第39条に基づき、雇入れの日から起算して6ヶ月間継続勤務し、かつ全労働日の8割以上出勤した従業員に対し、10日の年次有給休暇を付与する。
2. その後は1年間継続勤務し全労働日の8割以上出勤するごとに、次の日数を付与する。

| 勤続年数 | 付与日数 |
|---|---|
| 6ヶ月 | 10日 |
| 1年6ヶ月 | 11日 |
| 2年6ヶ月 | 12日 |
| 3年6ヶ月 | 14日 |
| 4年6ヶ月 | 16日 |
| 5年6ヶ月 | 18日 |
| 6年6ヶ月以上 | 20日 |

3. パートタイマー等の短時間勤務者については、所定労働日数に応じて比例付与する(労働基準法施行規則第24条の3)。
4. 入社時に当社独自の優遇措置として、4月1日入社の新入社員には入社日から5日の特別年次有給休暇を前倒し付与する。

**第3条(時間単位年休)**
1. 労使協定に基づき、年次有給休暇のうち年5日(40時間)を上限として、時間単位で取得することができる。
2. 1時間未満の単位での取得はできない。1日の所定労働時間に満たない端数(本規程上は1日=8時間として計算)については切り上げる。

**第4条(計画的付与)**
1. 当社は、労使協定により、年次有給休暇のうち5日を超える日数について計画的付与を行うことがある。
2. 主な計画的付与日として、夏季休業の前後、年末年始休業に連続する日、ゴールデンウィーク期間等を設定する。
3. 計画的付与日として指定された日は、原則として他の日への振替はできない。

**第5条(時季変更権)**
従業員が指定した時季に有給休暇を取得することが事業の正常な運営を妨げる場合(同一部署内で同時取得者多数、繁忙期の重要業務担当者、顧客対応等)、当社は他の時季への変更を求めることがある(労働基準法第39条第5項)。

**第6条(年5日の確実な取得)**
労働基準法第39条第7項に基づき、当社は年10日以上の年次有給休暇が付与される従業員に対し、付与日から1年以内に5日について、時季を指定して取得させる。本人の希望及び計画的付与により既に5日以上を取得している場合は、時季指定の対象外とする。

**第7条(時効と買上げ)**
1. 年次有給休暇は付与日から2年間で時効により消滅する(労働基準法第115条)。
2. 当社は原則として未消化の年次有給休暇を金銭で買い上げない。ただし、退職時に残存する未消化日数のうち、最大20日を限度として、本人の希望により1日当たり「基本給+役職手当+職能給」を1ヶ月所定労働日数で除した額で買い上げることができる(退職時特例)。

**第8条(取得時の給与)**
年次有給休暇を取得した日については、所定労働時間労働した場合に支払われる通常の賃金を支給する。

## 第2章 育児休暇
<!-- section_id: sec_2 -->

**第9条(育児休業の対象者)**
1歳に満たない子を養育する従業員(性別を問わない。)は、申し出により育児休業を取得することができる。有期雇用社員については、子が1歳6ヶ月に達する日までに労働契約が満了することが明らかでないことを要件とする(育児・介護休業法第5条)。

**第10条(育児休業の期間)**
1. 育児休業は、原則として子が1歳に達する日(誕生日の前日)までの連続した期間とする。
2. 保育所への入所申込みを行ったが入所できない等の事情がある場合は、子が1歳6ヶ月に達する日まで、さらに2歳に達する日まで延長することができる。
3. 育児休業は、原則として子1人につき2回まで分割取得することができる(2022年10月施行の改正法対応)。

**第11条(出生時育児休業)**
1. 子の出生後8週間以内に4週間(28日)まで、出生時育児休業(産後パパ育休)を取得することができる。
2. 出生時育児休業は、原則として2回まで分割して取得することができる。
3. 労使協定に基づき、出生時育児休業中は休業期間中の所定労働日・所定労働時間の半分を上限として、就業することができる(本人の同意がある場合に限る)。

**第12条(配偶者の出産休暇)**
配偶者が出産する従業員は、出産予定日の前後合計7日の範囲内で、2日の有給特別休暇(配偶者出産休暇)を取得することができる。

**第13条(子の看護休暇)**
小学校就学前の子を養育する従業員は、子1人につき年5日(2人以上の場合は年10日)を限度として、子の負傷・疾病の世話又は予防接種・健康診断のため、子の看護休暇を取得することができる(時間単位取得可。育児・介護休業法第16条の2)。

**第14条(育児短時間勤務)**
3歳に満たない子を養育する従業員は、1日6時間の短時間勤務制度を選択することができる。小学校就学までの子を養育する従業員については、所定外労働(残業)の免除を請求できる。

**第15条(育児休業期間中の給与等)**
1. 育児休業期間中は無給とする。
2. 雇用保険から育児休業給付金(休業開始から180日まで賃金の67%、それ以降50%)が支給される。両親ともに育児休業を取得する場合は、別途「出生後休業支援給付金」等の制度が適用される場合がある。
3. 健康保険料・厚生年金保険料は、育児休業等期間中の保険料免除制度により本人・会社負担分とも免除される。

## 第3章 介護休暇
<!-- section_id: sec_3 -->

**第16条(介護休業の対象家族)**
要介護状態(介護保険法上の要介護2以上又はこれに準ずる常時介護を必要とする状態)にある次の家族を介護する従業員は、介護休業を取得することができる。
- 配偶者(事実婚を含む)
- 父母(養父母を含む)
- 子
- 配偶者の父母
- 祖父母、兄弟姉妹、孫

**第17条(介護休業の期間)**
1. 対象家族1人につき通算93日まで介護休業を取得することができる。
2. 介護休業は3回まで分割して取得することができる。

**第18条(介護休暇)**
要介護状態にある対象家族の介護・通院付添等のため、対象家族1人につき年5日(2人以上の場合は年10日)の介護休暇を、時間単位又は半日単位で取得することができる。

**第19条(介護短時間勤務等)**
要介護状態の家族を介護する従業員は、申出により次の措置のうち1つ以上を選択して講じる。利用開始日から3年以上の期間で、少なくとも2回以上利用できる。
1. 短時間勤務制度(1日6時間)
2. フレックスタイム制
3. 始業・終業時刻の繰上げ・繰下げ
4. 介護サービス費用の補助

**第20条(介護休業期間中の給与等)**
1. 介護休業期間中は無給とする。
2. 雇用保険から介護休業給付金(賃金の67%)が支給される。
3. 社会保険料については免除制度の対象外であり、本人負担分は休業期間中も納付を要する(会社が立替えた上で復職後給与から控除する方法も可)。

## 第4章 特別休暇(慶弔)
<!-- section_id: sec_4 -->

**第21条(特別休暇の種類及び日数)**
従業員が次の各号に該当する事由により休暇を申請した場合、有給の特別休暇を付与する。

| 事由 | 日数 | 起算 |
|---|---|---|
| 本人の結婚 | 5日 | 結婚日の前後3ヶ月以内に取得 |
| 子の結婚 | 2日 | 挙式日を含む期間 |
| 配偶者の出産 | 2日 | 出産日の前後7日以内 |
| 配偶者の死亡 | 7日 | 死亡日から |
| 父母・子の死亡 | 5日 | 死亡日から |
| 兄弟姉妹・祖父母・配偶者の父母の死亡 | 3日 | 死亡日から |
| 配偶者の兄弟姉妹・配偶者の祖父母・孫の死亡 | 1日 | 死亡日から |
| 喪主を務める場合の加算 | 上記に2日加算 | |
| 葬儀のため遠隔地へ移動を伴う場合 | 上記に1日加算 | 片道300km以上 |
| 父母の法事(三回忌まで) | 1日 | 当日 |
| 転居(自己都合転勤・住居災害時等) | 1日 | 転居日 |

**第22条(裁判員・公職等)**
裁判員、検察審査員、選挙の投票、その他公の職務に従事する場合は、必要な時間につき有給の特別休暇を付与する。

**第23条(リフレッシュ休暇)**
勤続10年、20年、30年に達した従業員に対し、勤続表彰の一環として有給のリフレッシュ休暇(勤続10年:5日、20年:7日、30年:10日)を付与する。1年以内に取得すること。

**第24条(ボランティア休暇)**
災害復興支援、地域社会貢献活動等のボランティア活動に従事する場合、年5日を限度に無給のボランティア休暇を付与する。会社が指定する重大災害復興ボランティアについては有給とする。

**第25条(ドナー休暇)**
骨髄、末梢血幹細胞、献血の提供のため必要な日数について、有給特別休暇を付与する。

## 第5章 生理休暇・母性健康管理
<!-- section_id: sec_5 -->

**第26条(生理休暇)**
生理日の就業が著しく困難な女性従業員が休暇を請求した場合、これを与える(労働基準法第68条)。月2日までは有給とし、3日目以降は無給とする。

**第27条(妊婦健診のための時間)**
妊娠中の女性従業員に対し、母子保健法に定める妊婦健康診査を受けるために必要な時間を確保する(男女雇用機会均等法第12条)。具体的には次の回数を上限として有給扱いとする。
- 妊娠23週まで:4週間に1回
- 妊娠24週から35週まで:2週間に1回
- 妊娠36週以降出産まで:1週間に1回
- 産後1年以内:医師の指示に基づく回数

**第28条(通勤緩和及び休憩措置)**
医師等の指導があった場合、妊娠中の女性従業員に対し、時差出勤、通勤手段の変更、勤務時間中の休憩時間の延長・追加、業務軽減等の措置を講じる(男女雇用機会均等法第13条)。

**第29条(産前産後休業)**
1. 産前6週間(多胎妊娠の場合は14週間)、産後8週間の休業を付与する(労働基準法第65条)。
2. 産前休業は本人の請求により取得、産後休業は強制(産後6週間)及び請求(7〜8週目)による。
3. 休業中は無給とするが、健康保険から出産手当金(標準報酬日額の3分の2)が支給される。

**第30条(育児時間)**
生後1年未満の子を育てる女性従業員から請求があったときは、1日2回それぞれ少なくとも30分の育児時間を有給で付与する(労働基準法第67条)。

## 第6章 取得手続き
<!-- section_id: sec_6 -->

**第31条(年次有給休暇の申請)**
1. 年次有給休暇を取得しようとする者は、原則として取得希望日の3営業日前までに、勤怠管理システムを通じて上長に申請しなければならない。
2. 連続5日以上の取得を希望する場合は、原則として2週間前までに申請する。
3. 傷病・忌引等で事前申請が困難な場合は、当日始業時刻までに電話又はビジネスチャットで連絡し、事後速やかに正式申請を行う。

**第32条(育児・介護休業の申請)**
1. 育児休業、介護休業を取得しようとする者は、開始予定日の1ヶ月前(出生時育児休業は2週間前)までに、所定の「育児休業等申出書」を人事部に提出する。
2. 添付書類:住民票記載事項証明書、母子健康手帳の写し、要介護認定通知書の写し等。

**第33条(特別休暇の申請)**
1. 特別休暇は、事由発生後速やかに、勤怠管理システム及び所定の「特別休暇申請書」により申請する。
2. 弔事については、後日、会葬礼状又は死亡診断書の写し等の証憑を提出する。
3. 慶事(結婚・出産)については、戸籍記載事項証明書、母子健康手帳の写し等の証憑を提出する。

**第34条(繁忙期の調整)**
1. 各部門の繁忙期(決算期、年度末、製品リリース直前、棚卸期間等)については、所属長が事前に「休暇取得制限期間」を設定し、当該期間中は5日以上の連続休暇取得を原則として認めない。
2. 制限期間の設定は3ヶ月以上前に対象者へ周知する。

**第35条(取得後の業務引継ぎ)**
休暇取得者は、休暇前に業務の引継ぎ、後任者の指名、緊急連絡先の明示等を行うとともに、休暇中に業務上の連絡を要する場合に備えて、所属長に対応方針を確認すること。
