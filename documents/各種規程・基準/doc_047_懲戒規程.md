## 1. 懲戒の種類
<!-- section_id: sec_1 -->

**第1条(目的)**
本規程は、株式会社サンプル(以下「当社」という。)の就業規則(doc_001)第46条その他関連条項に基づき、当社の役員及び従業員の服務規律違反その他の非違行為に対する懲戒処分の種類、事由、手続及び不服申立について定めることにより、職場秩序の維持と公正な処分運用を確保することを目的とする。

**第2条(適用範囲)**
本規程は、当社の正社員、契約社員、嘱託社員、再雇用社員、パートタイマー及びアルバイトに適用する。役員に対する処分(解任、報酬減額等)は会社法及び別途定める「役員懲戒に関する内規」による。

**第3条(懲戒処分の基本原則)**
1. 懲戒処分は、罪刑法定主義、二重処罰禁止、相当性、平等取扱い、適正手続の各原則に従い、公正かつ慎重に運用する。
2. 一つの行為に対して二重に懲戒処分を行うことはしない。
3. 同種の事案については、過去の処分事例との均衡を考慮する。
4. 行為と処分の相当性を維持し、軽微な行為に対する過重な処分を行わない。

**第4条(懲戒処分の種類)**
懲戒処分は、軽い順に次のとおりとする。

1. **訓戒(けんかい)**:口頭又は書面による厳重注意。処分歴として人事記録に残るが、給与・賞与・昇格への直接の影響は与えない。ただし、評価上の参考とする場合がある。

2. **譴責(けんせき)**:始末書を提出させ、将来を戒める。処分歴として人事記録に残し、当該年度の人事評価及び賞与査定に影響することがある。

3. **減給**:譴責に加え、給与を減額する。1回の額は平均賃金の1日分の半額を超えず、総額は1ヶ月の賃金総額の10分の1を超えない(労働基準法第91条)。

4. **出勤停止**:譴責に加え、7労働日以内の出勤を停止する。出勤停止期間中は無給とし、勤続年数からは控除しない。ただし、退職金算定上の勤続年数及び年次有給休暇付与上の出勤率には算入しない。

5. **降格**:譴責に加え、職務等級又は役職を引き下げる。これに伴い給与、職能給、役職手当が減額となる。降格後の処遇は「給与規程」(doc_002)による。

6. **諭旨解雇(諭旨退職)**:懲戒解雇に相当する事由があるが、本人の反省及び情状を考慮し、退職を勧告するもの。本人がこれに応じない場合は、懲戒解雇とする。退職金は「退職金規程」(doc_004)第4条に基づき原則として50%減額する。

7. **懲戒解雇**:即時解雇する重大な処分。退職金は原則として支給しない(全額不支給)。労働基準法第20条に基づく解雇予告を行わず、解雇予告手当も支給しないことについては、所轄労働基準監督署長の認定を受けて取り扱う。

**第5条(付加措置)**
懲戒処分に併せて、次の付加措置を講じることがある。
1. 自宅待機命令(調査期間中、賃金は原則として支払う)
2. 配置転換、業務命令の変更
3. 役職解任、各種手当の支給停止
4. 損害賠償請求(故意又は重大な過失により会社に損害を与えた場合)
5. 民事・刑事告訴・告発(横領、背任、強制わいせつ、傷害等の場合)

## 2. 懲戒事由
<!-- section_id: sec_2 -->

**第6条(訓戒・譴責に該当する事由)**
従業員が次の各号のいずれかに該当する場合は、原則として訓戒又は譴責とする。事案の悪質性、繰り返しの有無等により、より重い処分に該当することがある。
1. 正当な理由なく遅刻、早退、私用外出、欠勤を繰り返したとき(月3回以上)
2. 勤務時間中に私用に従事し、業務の遂行を怠ったとき
3. 業務上の連絡・報告を怠り、業務に支障を生じさせたとき
4. 上長の正当な業務命令に従わなかったとき(軽微なもの)
5. 職場の風紀又は秩序を乱す軽微な行為をしたとき
6. 会社の備品・消耗品を私的に使用したとき
7. SNS等への投稿により当社の信用を軽微に毀損したとき
8. その他本規程及び就業規則の軽微な違反

**第7条(減給・出勤停止に該当する事由)**
従業員が次の各号のいずれかに該当する場合は、原則として減給又は出勤停止とする。
9. 正当な理由のない欠勤、遅刻が常習的(月7回以上)であるとき
10. 勤怠管理の不正(打刻ごまかし、他人の代理打刻、虚偽申告)を行ったとき
11. 経費の不正請求、領収書の改ざんを行ったとき(軽微なもの)
12. 業務上の指示・命令に意図的に従わなかったとき
13. 業務上のミスを隠蔽、虚偽報告したとき
14. 軽微なハラスメント行為を行ったとき(初回・本人反省あり)
15. 取引先からの社会通念上相当な範囲を超える金品授受を行ったとき
16. 会社所有の機密書類・データを許可なく持ち出したとき(私的悪用なし)
17. 飲酒運転、交通違反等により会社の信用を毀損したとき(重大なものを除く)
18. 副業・兼業を無届けで行ったとき
19. 安全衛生に関する規律違反を繰り返したとき
20. SNS等への投稿により当社の信用を相応に毀損したとき
21. 同僚、部下、後輩に対する繰り返しのいじめ・嫌がらせを行ったとき

**第8条(降格に該当する事由)**
従業員が次の各号のいずれかに該当する場合は、原則として降格(必要に応じて減給・出勤停止と併科)とする。
22. 管理職として自部署のハラスメント、不正、安全管理違反等を看過・放置したとき
23. 管理職として部下に対するパワーハラスメントを行ったとき
24. 管理職として虚偽報告を行い、上位者の判断を誤らせたとき
25. 著しい職務怠慢により、所管業務に重大な支障を生じさせたとき

**第9条(諭旨解雇・懲戒解雇に該当する事由)**
従業員が次の各号のいずれかに該当する場合は、原則として諭旨解雇又は懲戒解雇とする。情状により出勤停止・降格にとどめることがある。
26. 重要な経歴(学歴、職歴、資格、犯罪歴)を詐称し採用されたことが判明したとき
27. 正当な理由なく連続14日以上無断欠勤し、出勤の督促に応じないとき
28. 業務上の横領、背任、贈収賄を行ったとき
29. 会社の金品、商品、資産を窃取・着服したとき
30. 取引先・顧客からの私的な金品授受、リベート取得を行ったとき
31. 業務上、暴行、傷害、脅迫、強要等の刑事犯罪に該当する行為を行ったとき
32. 強制わいせつ、強姦に類するセクシュアルハラスメントを行ったとき
33. 重度のパワーハラスメントにより、被害者の休職・退職・重大な疾患・自殺等を招いたとき
34. 当社の機密情報、顧客個人情報を漏洩し、又は不正利用したとき
35. 競業避止義務に違反し、競業他社への情報提供又は競業事業を行ったとき
36. 反社会的勢力(暴力団、特殊詐欺グループ等)と関係を持ち、又は当社業務に介入させたとき
37. 在職中に当社又は取引先に対し、刑事事件で有罪判決(執行猶予含む)を受けたとき
38. 飲酒運転、麻薬・覚醒剤の使用等、社会的責任の重大な違反を行ったとき
39. 同種の懲戒処分を繰り返し受け、改善の見込みがないとき
40. 内部通報制度を悪用し、虚偽の通報により他者を陥れたとき
41. インサイダー取引、当社株式に関する不正取引を行ったとき
42. 当社IT資産への不正アクセス、データ破壊、業務妨害を行ったとき
43. 業務上の事故、不正等を隠蔽し、結果として重大な被害・損害を拡大させたとき

**第10条(懲戒事由対応表)**
標準的な懲戒事由と処分の対応関係は次表のとおり。具体的処分は、悪質性、被害規模、再犯性、本人の反省、業務影響度等を総合勘案して決定する。

| 行為類型 | 標準処分(初回) | 累犯時 |
|---|---|---|
| 軽微な遅刻・私用外出 | 訓戒 | 譴責→減給 |
| 勤怠不正(打刻ごまかし) | 譴責又は減給 | 出勤停止→降格 |
| 軽微なハラスメント | 譴責 | 減給→出勤停止 |
| 重度ハラスメント | 出勤停止→降格 | 諭旨解雇 |
| 軽微な経費不正(数万円) | 減給+全額弁済 | 諭旨解雇 |
| 業務上横領(数十万円以上) | 懲戒解雇 + 告訴 | — |
| 機密情報の意図的漏洩 | 懲戒解雇 + 損害賠償請求 | — |
| 競業避止違反 | 諭旨解雇又は懲戒解雇 | — |
| 反社関係 | 懲戒解雇 | — |
| 14日以上の無断欠勤 | 懲戒解雇 | — |
| 飲酒運転(業務中) | 懲戒解雇 | — |
| 飲酒運転(私生活) | 出勤停止→降格 | 諭旨解雇 |

## 3. 処分手続き
<!-- section_id: sec_3 -->

**第11条(調査の開始)**
1. 当社は、従業員に懲戒事由に該当する疑いがある事実を認識したときは、人事部が速やかに事実調査を開始する。
2. 調査責任者は人事部長とし、案件の重要度・複雑性に応じて、法務部、内部監査室、外部弁護士、社外専門家(公認会計士、デジタルフォレンジック専門家等)の協力を得る。

**第12条(調査方法)**
1. 調査は、関係者ヒアリング、文書・電子データ・メール・チャット記録の精査、業務システムのログ分析、現場確認等の方法により行う。
2. デジタルフォレンジック調査は、本人の事前同意又は会社の業務命令権に基づき実施する。
3. 調査期間中、本人に対し自宅待機命令(原則として有給)を発することができる。

**第13条(自宅待機命令)**
1. 調査の対象となった者については、調査の公正性確保、証拠保全、被害拡大防止等の観点から、必要な期間、自宅待機を命じることがある。
2. 自宅待機期間中の賃金は原則として全額支給する(平均賃金の60%以上を支払うものとし、当社の都合による就業不能として通常賃金を支払う運用とする)。
3. 自宅待機期間は原則3ヶ月以内とし、調査の進展に応じて短縮・延長する。

**第14条(本人弁明の機会の付与)**
1. 当社は、懲戒処分を行う前に、対象者に対し弁明の機会を付与しなければならない。
2. 弁明の機会の付与は、想定される懲戒事由(対象事実)、想定される処分の概要を書面で通知し、原則として通知後7日以上の準備期間を確保した上で、口頭弁明会(必要に応じてオンライン)又は書面弁明により行う。
3. 弁明会には、本人が希望する場合、社内の同伴者(同僚、上長、組合代表者)又は弁護士を同席させることができる。

**第15条(懲戒委員会)**
1. 出勤停止以上の懲戒処分及び管理職に対する全ての懲戒処分については、懲戒委員会の審議を経て決定する。
2. 懲戒委員会の構成は次のとおりとする。
   - 委員長:人事担当役員
   - 委員:人事部長、法務部長、対象者の所属本部長(対象者と利害関係のない場合)、外部弁護士、社外有識者(必要に応じて)
   - オブザーバー:サンプル労働組合代表(対象者が組合員の場合、本人の同意を得て)
3. 懲戒委員会は、調査結果、本人弁明、関係証拠を総合的に審議し、処分案を取りまとめ、代表取締役に答申する。
4. 諭旨解雇・懲戒解雇については、代表取締役の決裁を必要とする。

**第16条(処分の通知)**
1. 処分は「懲戒処分通知書」により本人に書面交付する。通知書には、懲戒事由、認定事実、適用条文、処分内容、処分の効力発生日、不服申立方法を記載する。
2. 本人が受領を拒否した場合は、配達証明付き郵便により本人住所に送付する。

**第17条(社内公示)**
1. 処分内容は、再発防止の観点から、個人を特定する情報(氏名、所属、職位等)を匿名化した上で、社内ポータル等で事案概要を公示することがある。
2. 取締役、執行役員等経営層に対する処分については、株主、投資家、取引先等に対する説明責任を考慮し、当社の情報開示ポリシーに基づき適切に公表する。

**第18条(時効)**
懲戒事由となる行為があった日から起算して3年を経過したものについては、原則として懲戒処分を行わない。ただし、横領、背任、機密漏洩、性犯罪等の重大事案については、この限りでない(当社が事実を知った日から1年以内であれば処分可能)。

**第19条(自首・自己申告の取扱い)**
懲戒事由に該当する行為を行った者が、調査開始前に自ら申告した場合は、その情状を考慮し、処分の軽減を検討する。

## 4. 不服申立
<!-- section_id: sec_4 -->

**第20条(不服申立権)**
懲戒処分を受けた者は、処分通知書を受領した日から2週間以内に、所定の「懲戒処分不服申立書」を人事部に提出することにより、再審査を申し立てることができる。

**第21条(再審査委員会の設置)**
1. 不服申立を受けたときは、当社は再審査委員会を設置し、原処分の妥当性を再審査する。
2. 再審査委員会の構成は次のとおりとし、原処分の決定に関与した者は委員に含めない。
   - 委員長:代表取締役以外の取締役(独立社外取締役が望ましい)
   - 委員:法務部長、外部弁護士(原処分時の外部弁護士とは別の者)、社外有識者(労働法専門家、社会保険労務士等)
   - オブザーバー:サンプル労働組合代表(対象者の同意がある場合)

**第22条(再審査の手続)**
1. 再審査委員会は、原処分の調査記録、弁明記録、処分理由を再検証するとともに、必要に応じて本人及び関係者から追加のヒアリングを実施する。
2. 本人は、再審査において再度弁明する機会を有する。
3. 再審査期間は、不服申立受理から原則として60日以内とする。複雑な事案については、本人に通知の上で延長することがある。

**第23条(再審査の決定)**
1. 再審査委員会は、審理結果を踏まえ、次のいずれかの結論を答申する。
   - 原処分の維持
   - 原処分の変更(処分の軽減又は変更)
   - 原処分の取消し
2. 代表取締役は、答申を踏まえ、最終決定を行い、本人に書面で通知する。
3. 原処分が取り消された場合、当社は本人に対し、自宅待機期間中の賃金以外の損失(処分により減額された賃金、賞与、降格に伴う差額等)を遡って支払う。

**第24条(再審査決定後の救済手続)**
本人は、再審査決定の結果に不服がある場合、労働組合との団体交渉、労働局・労働委員会への申立、民事訴訟等の法的手続をとることができる。当社は、これらの手続をとったことを理由に、本人にいかなる不利益取扱いも行わない。

**第25条(関係者の保護)**
1. 不服申立及び再審査手続に関与した本人、証人、調査協力者のプライバシーは厳格に保護する。
2. 関係者に対する報復、不利益取扱いは厳禁とし、これに違反した者は別途懲戒処分の対象とする。
