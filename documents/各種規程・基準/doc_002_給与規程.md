## 第1章 給与の構成
<!-- section_id: sec_1 -->

**第1条(目的)**
本規程は、株式会社サンプル(以下「当社」という。)の就業規則(doc_001)第47条その他関連条項に基づき、当社に在籍する従業員の給与に関する事項を定めることを目的とする。

**第2条(適用範囲)**
本規程は当社に在籍する正社員に適用する。契約社員、嘱託社員、パートタイマー、再雇用社員については個別の労働契約又は別途定める規程に従い、本規程の一部を準用する。

**第3条(給与の構成)**
給与は、所定内給与と所定外給与から構成され、その内訳は次のとおりとする。

| 区分 | 項目 | 内容 |
|---|---|---|
| 所定内給与(月例) | 基本給 | 職務等級、職務遂行能力、人事評価に基づき決定する固定給。 |
| | 役職手当 | 役職に応じて支給。 |
| | 職能給 | 等級ごとに設定される能力給。 |
| | 調整給 | 中途採用、職務変更等の事情により暫定的に支給する給与。 |
| | 諸手当 | 通勤手当、住宅手当、家族手当、資格手当、単身赴任手当、在宅勤務手当等。 |
| 所定外給与 | 時間外勤務手当 | 法定時間外労働、法内残業に対する手当。 |
| | 深夜勤務手当 | 22:00〜翌5:00の労働に対する割増。 |
| | 休日勤務手当 | 法定休日及び所定休日労働に対する手当。 |
| 賞与 | 夏季賞与・冬季賞与 | 年2回支給。第5章参照。 |

**第4条(基本給)**
1. 基本給は、職務等級制度に基づき決定する。等級は1等級(新人)から10等級(執行役員クラス)までとし、各等級の月額レンジは別表第1のとおりとする。
2. 同一等級内における昇給は、年1回の人事評価結果に基づき決定する(第6章)。

**第5条(役職手当)**
役職に就く者には、次の月額役職手当を支給する。

| 役職 | 月額 |
|---|---|
| 本部長 | 100,000円 |
| 部長 | 70,000円 |
| 次長 | 55,000円 |
| 課長 | 40,000円 |
| 係長・主任 | 25,000円 |
| グループリーダー | 15,000円 |

役職を解かれた月の翌月から支給を停止する。役職手当受給者は、原則として時間外勤務手当の支給対象外とする(管理監督者の取扱いは労働基準法第41条第2号により別途判断)。

**第6条(職能給)**
等級別の職能給は別表第2による。

**第7条(調整給)**
中途採用者、職務転換者、休職復帰者等で、本規程による給与額が前職又は従前の給与水準を著しく下回る場合、人事部長の決裁により最大3年間、調整給を支給することがある。

## 第2章 計算期間と支払日
<!-- section_id: sec_2 -->

**第8条(計算期間)**
給与計算期間は、毎月1日から当月末日までとする。

**第9条(支払日)**
1. 給与は計算期間の当月25日に支払う。
2. 支払日が土曜日、日曜日、国民の祝日に該当する場合は、その直前の銀行営業日に繰り上げて支払う。

**第10条(支払方法)**
1. 給与は労働基準法第24条及び労使協定に基づき、本人名義の指定銀行口座に振り込む方法により支払う。
2. 退職時の未払賃金は、本人の請求がある場合に限り、退職日から7日以内に支払う(労働基準法第23条)。

**第11条(端数処理)**
1. 1ヶ月の時間外労働時間は1分単位で集計し、1ヶ月分の合計について30分未満の端数を切り捨て、30分以上の端数を1時間に切り上げることができる(労使協定による)。
2. 計算結果の1円未満の端数は50銭未満切捨て、50銭以上切上げとする。

**第12条(控除)**
労使協定及び法令に基づき、給与より次の項目を控除する。
1. 源泉所得税
2. 住民税(特別徴収)
3. 健康保険料、介護保険料、厚生年金保険料、雇用保険料の本人負担分
4. サンプル労働組合 組合費(組合員のみ)
5. 社内預金、財形貯蓄、団体保険料、社員持株会拠出金(本人申込分)
6. その他労使協定により定めた項目

**第13条(欠勤・遅刻・早退控除)**
欠勤、遅刻、早退、私用外出等不就労時間がある場合、次の算式により給与から控除する。
- 月給控除額 = (基本給 + 役職手当 + 職能給) ÷ 当月の所定労働日数 × 不就労日数
- 時間控除額 = (基本給 + 役職手当 + 職能給) ÷ (当月の所定労働日数 × 8時間) × 不就労時間

**第14条(非常時払)**
従業員又はその扶養家族の出産、疾病、災害、結婚、死亡、やむを得ない事由による1週間以上の帰郷の場合、既往の労働に対する賃金については、支払期日前であっても請求により支払う(労働基準法第25条)。

## 第3章 諸手当
<!-- section_id: sec_3 -->

**第15条(通勤手当)**
1. 通勤に要する実費を、最も経済的かつ合理的な経路・方法により計算した額を月額150,000円を上限として支給する。
2. 6ヶ月定期券相当額を原則として6ヶ月ごとに前払いする。
3. 自動車・自転車通勤は、公共交通機関の利用が困難な場合に限り、片道距離に応じて次のとおり支給する(所得税法施行令第20条の2に準拠)。

| 片道距離 | 月額 |
|---|---|
| 2km未満 | 支給なし(原則徒歩) |
| 2km以上10km未満 | 4,200円 |
| 10km以上15km未満 | 7,100円 |
| 15km以上25km未満 | 12,900円 |
| 25km以上35km未満 | 18,700円 |
| 35km以上45km未満 | 24,400円 |
| 45km以上55km未満 | 28,000円 |
| 55km以上 | 31,600円 |

4. 在宅勤務日数が月の所定労働日数の半数を超える場合は、定期券方式に代えて出社日数に応じた実費支給(往復実費 × 出社日数)に切り替えることができる。

**第16条(住宅手当)**
1. 自宅から事業場までの片道通勤時間が90分以内に居住する従業員に対し、住宅手当を次のとおり支給する。

| 区分 | 月額 |
|---|---|
| 世帯主(賃貸) | 30,000円 |
| 世帯主(持家) | 20,000円 |
| 非世帯主(賃貸) | 15,000円 |
| 単身者(賃貸) | 18,000円 |

2. 借上社宅制度の対象者は、住宅手当の支給対象外とする。

**第17条(家族手当)**
扶養親族(所得税法上の控除対象配偶者及び扶養親族)を有する従業員に対し、次の家族手当を支給する。

| 区分 | 月額 |
|---|---|
| 配偶者 | 13,000円 |
| 子(満22歳に達する年度末まで)1人目 | 8,000円 |
| 子 2人目以降(1人につき) | 5,000円 |
| 同居の父母(60歳以上、所得要件あり) | 5,000円 |

ただし、共働き世帯では夫婦のいずれか一方のみに支給する。

**第18条(資格手当)**
業務に直接関連する公的資格・社内認定資格の保有者に対し、次の月額手当を支給する(支給は当該資格が必要な職務に就いている期間に限る)。

| 資格 | 月額 |
|---|---|
| 公認会計士・税理士・弁護士・社会保険労務士 | 30,000円 |
| 中小企業診断士・1級建築士・技術士 | 20,000円 |
| 簿記1級・宅地建物取引士・第一種衛生管理者 | 10,000円 |
| TOEIC 800点以上(2年更新) | 8,000円 |
| 情報処理 高度試験 | 8,000円 |
| 簿記2級・基本情報技術者 | 3,000円 |

**第19条(単身赴任手当)**
転勤に伴い家族と別居して単身赴任となる者に対し、月額60,000円を支給する。加えて、月1回の帰宅旅費(普通乗車券相当額)を実費支給する。

**第20条(在宅勤務手当)**
「在宅勤務規程」(doc_008)に基づき在宅勤務を行う者に対し、通信費・光熱費等の補填として月額5,000円を支給する。在宅勤務日数が月5日に満たない場合は支給しない。

## 第4章 時間外勤務手当
<!-- section_id: sec_4 -->

**第21条(算定基礎額)**
1. 時間外勤務手当、深夜勤務手当及び休日勤務手当の計算基礎額は、基本給、役職手当(管理監督者の場合を除く)及び職能給の合計額を、所定労働時間で除して得た時間単価とする。
2. 労働基準法施行規則第21条に定める除外賃金(家族手当、通勤手当、住宅手当、別居手当、子女教育手当、臨時に支払われた賃金、1ヶ月を超える期間ごとに支払われる賃金)は算定基礎に含めない。
3. 月平均所定労働時間は、年間所定労働日数 × 1日所定労働時間 ÷ 12 により算出する(目安として概ね160時間〜170時間)。

**第22条(割増率)**

| 区分 | 割増率 |
|---|---|
| 法定時間外労働(週40時間又は1日8時間超) | 25% |
| 法定時間外労働で月60時間超の部分 | 50% |
| 深夜労働(22:00〜翌5:00) | 25% |
| 法定休日労働(週1日の法定休日) | 35% |
| 法定休日かつ深夜 | 60%(35% + 25%) |
| 法定時間外かつ深夜 | 50%(25% + 25%) |

法内残業(所定労働時間を超え法定労働時間の範囲内)については、割増は付さず時間単価100%相当額を支給する。

**第23条(固定残業代制度)**
1. 特定の職種(営業職、企画職、専門職)について、月30時間相当分の時間外労働を見込んだ固定残業代(みなし残業手当)を月例給与に含めて支給することがある。
2. 固定残業代に相当する時間を超えて時間外労働を行った場合、超過分について別途時間外勤務手当を支給する。
3. 固定残業代の対象となる時間数及び金額は、個別の労働契約書及び給与通知書に明記する。

**第24条(管理監督者の取扱い)**
労働基準法第41条第2号に定める管理監督者(原則として課長級以上のうち、職務権限・処遇・労働時間管理の自由度等から管理監督者と判断される者)については、時間外勤務手当及び休日勤務手当を支給しない。ただし、深夜勤務手当は支給する。

**第25条(代休と割増)**
所定休日の労働について事後に代休を付与した場合でも、当該労働が法定時間外労働となる部分については割増賃金(25%相当)を支給する。

## 第5章 賞与
<!-- section_id: sec_5 -->

**第26条(賞与の支給)**
1. 賞与は、会社の業績、本人の人事評価、勤怠状況、貢献度等を総合勘案し、原則として年2回支給する。
2. 支給日及び算定期間は次のとおりとする。

| 区分 | 算定期間 | 支給日 |
|---|---|---|
| 夏季賞与 | 前年11月1日から当年4月30日まで | 当年7月10日 |
| 冬季賞与 | 当年5月1日から当年10月31日まで | 当年12月10日 |

3. 支給日が休日に該当する場合は、その直前の銀行営業日に繰り上げて支払う。

**第27条(支給対象)**
1. 賞与の支給対象は、支給日に在籍する従業員のうち、算定期間中に在籍した者とする。
2. 算定期間中の在籍月数に応じて按分支給する。1ヶ月未満の在籍は支給対象外とする。
3. 退職予定者(支給日前に退職届を提出済みの者)については、原則として支給するが、業績連動部分等について個別判断する場合がある。

**第28条(算定方法)**
1. 標準支給額は、基本給に算定月数(夏季・冬季それぞれ標準2.5ヶ月、年合計5.0ヶ月を目安)を乗じた額とする。
2. 個別支給額は、標準支給額に、会社業績係数(0.7〜1.3)と個人評価係数(0.5〜1.5)を乗じて算定する。
3. 算定期間中の出勤率が80%未満の場合は、出勤率に応じて支給額を減額することがある。

**第29条(査定方法)**
個人評価は、別途定める「人事評価規程」に基づき、目標達成度(MBO)、行動評価(コンピテンシー)、360度フィードバック等の結果を総合的に判定する。

## 第6章 昇給
<!-- section_id: sec_6 -->

**第30条(定期昇給)**
1. 定期昇給は原則として年1回、4月1日付で実施する。
2. 直近1年間(前年1月から12月まで)の人事評価結果に基づき、各等級の昇給テーブル(別表第3)により昇給額を決定する。

**第31条(ベースアップ)**
経済情勢、物価変動、業界水準、会社業績等を勘案し、サンプル労働組合との春季労使交渉の結果として、毎年度ベースアップを実施することがある。

**第32条(査定昇給)**
人事評価結果が継続的に高い(直近2年連続SS評価又はS評価)従業員、又は職務等級の昇格があった従業員については、定期昇給とは別に査定昇給を行う。

**第33条(降給)**
1. 人事評価結果が継続的に著しく不良(直近2年連続D評価以下)である場合、又は懲戒処分(降格)を受けた場合は、職務等級の降格に伴う降給を行うことがある。
2. 降給に際しては、本人への事前説明、改善機会の付与等の手続を経た上で実施する。

**第34条(中途採用者の取扱い)**
中途採用者の昇給は、入社月により次のとおり調整する。
- 1月から6月入社:翌年4月の定期昇給より満額対象
- 7月から12月入社:翌年4月の定期昇給は在籍期間に応じて按分し、翌々年4月より満額対象

**附則**
1. 本規程は2018年4月1日より施行する。
2. 第1回改定:2020年4月1日(働き方改革関連法対応、月60時間超の割増率を中小企業適用に対応し50%に統一)
3. 第2回改定:2023年4月1日(在宅勤務手当を新設、固定残業代制度を明文化)
4. 第3回改定:2024年12月1日(資格手当の見直し、TOEIC基準引上げ)
5. 第4回改定:2026年5月19日(全面改定。別表方式の導入、職能給テーブル整備。)本改定版はサンプル労働組合との合意により2026年6月1日より施行する。
6. 別表第1(等級別基本給レンジ)、別表第2(職能給テーブル)、別表第3(評価別昇給テーブル)は人事部にて管理し、社内ポータルで閲覧可能とする。
