## 第1章 適用範囲と支給要件
<!-- section_id: sec_1 -->

**第1条(目的)**
本規程は、株式会社サンプル(以下「当社」という。)の就業規則(doc_001)第47条に基づき、従業員の退職に際して支給する退職金(退職一時金及び確定拠出年金事業主掛金)について必要な事項を定め、もって従業員の長期にわたる勤続と功労に報い、退職後の生活の安定を図ることを目的とする。

**第2条(適用範囲)**
1. 本規程は、当社の正社員のうち、勤続3年(36ヶ月)以上の者に対し適用する。
2. 契約社員、嘱託社員、パートタイマー、再雇用社員、出向受入者、業務委託契約者には本規程を適用しない(ただし、再雇用社員のうち定年退職前の正社員期間に係る退職金は別途算定する。)。
3. 試用期間中に退職又は解雇となった者には適用しない。

**第3条(支給要件)**
従業員が次の各号のいずれかに該当する場合、本規程に基づき退職金を支給する。
1. 自己都合による退職
2. 定年退職
3. 会社都合による退職(整理解雇、事業所閉鎖等)
4. 業務上又は業務外の傷病による退職
5. 死亡退職
6. 普通解雇(本人の責めに帰すべき重大な事由によらないもの)

**第4条(不支給及び減額事由)**
1. 次の各号のいずれかに該当する場合、退職金の全部又は一部を支給しないことがある。
   - 懲戒解雇となったとき:原則として全額不支給とする。
   - 諭旨解雇となったとき:原則として50%を減額する。
   - 在職中の行為により会社に重大な損害を与えたとき:損害額の範囲で減額・相殺する。
   - 退職後に在職中の重大な背信行為(横領、機密漏洩、競業避止違反等)が判明したとき:返還を請求する。
   - 退職後の競業避止義務に違反したとき:競業避止特約に基づき返還を請求する。
2. 不支給又は減額の決定は、退職金審査委員会(人事担当役員、人事部長、法務部長、社外監査役を委員とする。)の審議を経て、代表取締役の決裁により行う。

**第5条(勤続年数の計算)**
1. 勤続年数は、入社年月日から退職日までの期間とし、1ヶ月未満の端数は切り上げて1ヶ月とする。
2. 休職期間(私傷病休職、自己都合休職)は勤続年数から控除する。ただし、育児休業期間、介護休業期間、業務上災害による休職期間、会社都合による出向期間は控除しない。
3. 退職後再入社した場合、原則として前職期間は通算しない(会社が特に認めた場合を除く)。

## 第2章 算定方法
<!-- section_id: sec_2 -->

**第6条(退職金の構成)**
退職金は、次の2つの部分から構成される。
1. ポイント制退職金(基本部分):在籍中に毎年付与されるポイントの累積に基づく金額。
2. 確定拠出年金事業主掛金(第4章):月額基本給の3%相当を毎月拠出。

**第7条(ポイント制退職金の算定式)**
ポイント制退職金額 = 累積退職金ポイント × ポイント単価(1ポイント=10,000円) × 退職事由係数

**第8条(累積退職金ポイント)**
1. 退職金ポイントは、職務等級ポイントと功労ポイントの合計を、毎年4月1日(基準日)に付与する。
2. 職務等級ポイントは、別表第1のとおり、その年度の4月1日時点の等級により定める。

| 等級 | 年間付与ポイント |
|---|---|
| 1等級 | 8 |
| 2等級 | 10 |
| 3等級 | 13 |
| 4等級 | 17 |
| 5等級 | 22 |
| 6等級 | 28 |
| 7等級 | 35 |
| 8等級 | 43 |
| 9等級 | 52 |
| 10等級 | 62 |

3. 功労ポイントは、人事評価結果(SS、S、A、B、C、D)に応じて、別表第2のとおり加算又は減算する。

| 評価 | 加減ポイント |
|---|---|
| SS | +5 |
| S | +3 |
| A | +1 |
| B | 0 |
| C | △2 |
| D | △4 |

**第9条(退職事由係数)**
退職事由に応じ、次の係数を乗じる。

| 退職事由 | 係数 |
|---|---|
| 会社都合(整理解雇、事業閉鎖等) | 1.50 |
| 死亡(業務上) | 1.50 |
| 死亡(業務外) | 1.30 |
| 定年退職 | 1.30 |
| 業務上の傷病による退職 | 1.30 |
| 業務外の傷病による退職(勤続10年以上) | 1.10 |
| 自己都合退職(勤続10年以上) | 1.00 |
| 自己都合退職(勤続3年以上10年未満) | 0.70 |
| 諭旨解雇 | 0.50 |
| 懲戒解雇 | 0.00 |

**第10条(参考:勤続年数別目安額)**
新卒入社・平均評価・自己都合退職を仮定した場合の概ねの目安は別表第3のとおり(あくまで目安であり、実際の支給額は等級・評価・退職事由により大きく異なる)。

| 勤続年数 | 累積ポイント目安 | 退職金目安(自己都合) |
|---|---|---|
| 3年 | 30 | 約21万円 |
| 5年 | 60 | 約42万円(係数0.70) |
| 10年 | 180 | 約180万円(係数1.00) |
| 20年 | 500 | 約500万円 |
| 30年 | 950 | 約950万円 |
| 38年(定年) | 1,400 | 約1,820万円(係数1.30) |

**第11条(在職中の死亡)**
業務上又は業務外の死亡退職の場合、別途遺族に対し弔慰金(慶弔見舞金規程doc_006参照)を支給する。退職金は法定相続人に対し、相続分に応じて支給する。

## 第3章 支給時期
<!-- section_id: sec_3 -->

**第12条(支給時期)**
1. 退職一時金は、退職日の属する月の翌々月末日までに(退職後概ね60日以内)、本人(死亡退職の場合は法定相続人)の指定する銀行口座に振り込む方法により支給する。
2. 退職金審査委員会の審議を要する事案、退職金算定に必要な書類の提出遅延等がある場合は、当社からの通知により支給時期を延期することがある。

**第13条(死亡退職の場合の受給者)**
1. 死亡退職の場合の受給権者の順位は、労働基準法施行規則第42条から第45条に準じ、次のとおりとする。
   - 第1順位:配偶者(事実婚を含む)
   - 第2順位:子(死亡時に生計を維持されていた者を優先)
   - 第3順位:父母
   - 第4順位:孫
   - 第5順位:祖父母
   - 第6順位:兄弟姉妹
2. 同順位の者が複数ある場合は均等に支給する。

**第14条(税法上の取扱い)**
1. 退職金は所得税法上の退職所得に該当し、源泉徴収を行う。
2. 退職者から「退職所得の受給に関する申告書」の提出を受けた場合は、退職所得控除を適用した上で分離課税方式により源泉徴収する。提出がない場合は退職金額の20.42%を源泉徴収する。
3. 住民税についても退職所得分離課税により別途特別徴収する。

**第15条(差押え等の取扱い)**
退職金が差押え、譲渡、相続等の対象となる場合、当該裁判所命令書、相続関係証明書等の確認を経て支給する。

## 第4章 確定拠出年金
<!-- section_id: sec_4 -->

**第16条(企業型確定拠出年金制度の採用)**
当社は、確定拠出年金法に基づき、企業型確定拠出年金制度(以下「企業型DC」という。)を採用する。詳細な規約は別途「企業型確定拠出年金規約」に定める。

**第17条(加入対象者)**
1. 企業型DCの加入対象者は、当社の正社員(60歳未満)とする。
2. 入社月の翌月1日付で自動加入とする(本人の意思による不加入は認めない)。

**第18条(事業主掛金)**
1. 事業主掛金は、各月の基本給の3%(円未満切上げ)を毎月拠出する。
2. 拠出限度額(月55,000円)を超える部分は、当該月の給与に上乗せして支給する。
3. 事業主掛金は、退職金規程上の退職金の一部を構成するものとして取り扱う。

**第19条(マッチング拠出)**
1. 加入者は、本人の選択により事業主掛金と同額を上限として、自身の給与から追加拠出(マッチング拠出)を行うことができる。
2. マッチング拠出額は給与から天引きされ、所得控除(小規模企業共済等掛金控除)の対象となる。

**第20条(運用商品)**
1. 加入者は、当社が選定した運営管理機関の提供する運用商品ラインアップ(預金、保険、投資信託等)の中から自ら運用商品を選択する。
2. 加入者が運用指図を行わない場合は、指定運用方法(バランス型投資信託)により運用する(確定拠出年金法第25条の2)。
3. 当社は、年1回以上、投資教育(継続投資教育)を実施する。

**第21条(資産の移換)**
1. 加入者が退職した場合は、原則として6ヶ月以内に他企業の企業型DC又は個人型確定拠出年金(iDeCo)に資産を移換する手続を行う必要がある。
2. 期限内に移換手続を行わなかった場合は、国民年金基金連合会へ自動移換され、運用が行われない状態となるとともに、手数料が発生する。

**第22条(給付)**
1. 老齢給付金は、原則として60歳以降に受給可能(加入期間10年以上が必要)。
2. 障害給付金、死亡一時金、脱退一時金については確定拠出年金法及び規約に定めるところによる。

**第23条(退職金規程との関係)**
1. 本章に定める確定拠出年金事業主掛金は、第7条に定めるポイント制退職金とは別建てで支給される。
2. 懲戒解雇等により退職金が不支給又は減額となった場合でも、過去に拠出済みの確定拠出年金資産は加入者個人の権利として確保される(没収はできない。確定拠出年金法第32条)。
