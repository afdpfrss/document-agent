## 1. 制度概要
<!-- section_id: sec_1 -->

平素より業務にご精励いただき誠にありがとうございます。
当社は、2024年度に実施した社員意識調査(回答率94%、有効回答数1,512名)において、福利厚生制度に対する満足度が「やや満足以上」45.2%、「やや不満以下」31.6%と、改善余地が大きい結果が示されました。とりわけ、自由記述では「画一的な制度ではライフステージや家族構成に合わない」「保養所や社員旅行は利用機会が限られる」「育児・介護・自己啓発・健康増進など、個別ニーズに応じた選択肢が欲しい」といった声を多数いただきました。

これら社員の皆様の声、ならびに「人的資本経営の実装」「ダイバーシティ&インクルージョンの推進」「健康経営優良法人(ホワイト500)継続認定」「ISO30414(人的資本情報開示)対応」等の経営課題を踏まえ、2024年12月の経営会議および2025年1月の労使協議会(過半数代表者:西川一郎氏、各事業所選出社員代表計11名)における慎重な議論を経て、従来型の画一的な福利厚生制度を抜本的に見直し、社員一人ひとりがライフスタイル・ライフステージに応じて柔軟に利用メニューを選択できる「カフェテリアプラン」方式の福利厚生制度を、2025年4月1日付で導入することと致しました。

本制度の運営は、福利厚生アウトソーシング業界最大手の「株式会社ベネフィット・ワン」と業務提携し、同社が提供する「ベネフィット・ステーション」を基盤利用とすると同時に、当社専用のカフェテリアプラン管理ポータル「Cafeteria@Sample」をカスタマイズ構築致しました。これにより、ベネフィット・ステーション約140万件のサービスメニューと、当社独自の補助項目との両方を、シームレスにご利用いただけます。

### 1-1. 基本構成
1. ポイント付与:全社員(派遣社員除く)に対し、毎年度4月1日付で「カフェテリアポイント」を一律60,000ポイント付与(1ポイント=1円)。役職・勤続年数による加算は設けず、すべての社員に公平に同額を付与する。
2. 育児・介護中の社員向け加算:小学校就学前の子(配偶者の連れ子・養子を含む)を養育する社員、ならびに要介護認定2以上の家族を介護する社員に対しては、別途20,000ポイントを加算付与する(申告制、毎年度4月15日締切)。
3. 有効期限:付与日から1年間(翌年3月31日まで)とし、未使用分は失効する(翌年度繰越不可)。
4. ベネフィット・ステーション併用:ポイント利用とは別に、ベネフィット・ステーションが提供する会員特典(レストラン優待、宿泊優待、レジャー優待、健康相談等)は、全社員が無制限に利用可能。

### 1-2. 設計思想
1. 公平性:勤続年数・役職・性別・婚姻状況・家族構成等にかかわらず、全社員へ一律ポイントを付与し、付加価値を一人ひとりに均等に届ける。
2. 選択の自由:約60カテゴリ・約2,000項目のメニューから、自身のニーズに応じて自由に選択可能。
3. 透明性:ポイント付与・利用・残高はCafeteria@Sample上でリアルタイムに可視化。
4. 持続可能性:制度導入に伴う総人件費影響は、廃止する旧制度(保養所、社員食堂補助の一部、社員旅行補助、慶弔以外の祝品等)の予算を再配分することで、原則として総額中立を維持。

## 2. 利用メニュー
<!-- section_id: sec_2 -->

利用可能なメニューは、以下の6カテゴリ(各カテゴリ内に複数の項目)に体系化致しました。主な利用例を以下に示します。詳細な対象範囲、補助率、上限額、添付書類要件、対象外項目については、Cafeteria@Sample内「メニュー一覧2025年度版」をご確認ください。

### 2-1. 育児カテゴリ
- 認可・認可外保育園、企業主導型保育園、学童保育、病児保育、産前産後ケアハウス等の利用料補助(月額上限15,000ポイント)
- ベビーシッター利用料補助(内閣府ベビーシッター割引券との併用可、1回あたり3,000ポイント)
- 育児用品購入(おむつ、粉ミルク、ベビーカー、チャイルドシート等。1点5,000ポイント以上のみ対象)
- 妊活・不妊治療費補助(年間上限20,000ポイント、領収書添付)
- 子ども関連教育(英会話、プログラミング、習い事の月謝補助、月額上限10,000ポイント)

### 2-2. 介護カテゴリ
- 訪問介護・通所介護・短期入所(ショートステイ)等の介護保険サービス自己負担分補助
- 紙おむつ等の介護用品購入補助(月額上限8,000ポイント)
- 介護休業中の家族訪問交通費補助(往復1回につき10,000ポイント上限)
- 介護研修受講(ケアマネージャ・介護福祉士・実務者研修等)費補助

### 2-3. 健康カテゴリ
- スポーツジム・フィットネスクラブ会費(月額上限8,000ポイント、コナミスポーツ・ティップネス・ライザップ等)
- 人間ドック追加オプション(脳ドック、PET-CT、女性検診、内視鏡検査等。法定健診の補完)
- 禁煙外来費用補助
- マッサージ・整体・鍼灸(国家資格保有院に限る)費補助
- メンタルヘルス相談(ベネフィット・ステーション提携の外部カウンセリング、24時間対応)

### 2-4. 自己啓発カテゴリ
- 資格取得費用補助(受験料、対策講座、合格祝金)。対象資格は人事本部「業務関連資格一覧2025」に掲載のもの(中小企業診断士、情報処理技術者試験各種、TOEIC、簿記、社会保険労務士、各種ベンダー資格等)
- 通信教育・eラーニング受講料(Udemy Business、グロービス学び放題、Schoo等の法人プラン併用)
- ビジネス書籍購入(年間上限20,000ポイント、業務関連書籍のみ)
- 語学学校・オンライン英会話レッスン費補助

### 2-5. 住宅カテゴリ
- 引越し費用補助(会社都合異動以外の自己都合転居、年1回上限30,000ポイント)
- 住宅ローン繰上返済支援(年1回上限30,000ポイント)
- 賃貸住宅更新料補助
- リフォーム・住宅設備修繕費補助
- 防災用品購入(非常食、簡易トイレ、防災リュック等)

### 2-6. 余暇カテゴリ
- 旅行費用補助(国内・海外宿泊、ベネフィット・ステーション提携施設での割引と併用可、年間上限25,000ポイント)
- レジャー施設チケット(テーマパーク、水族館、美術館、コンサート等)
- 文化・スポーツ観戦
- 自己リフレッシュ目的のスパ・温泉施設利用

## 3. 申請方法
<!-- section_id: sec_3 -->

### 3-1. 利用ポータル
1. 専用ポータル「Cafeteria@Sample」(URL:https://cafeteria.sample.co.jp)へ、社員番号およびシングルサインオン(Microsoft Entra ID連携)でログインしてください。スマートフォン用iOS/Androidアプリも提供致します。
2. トップページにて、自身のポイント残高、利用履歴、有効期限、申請中案件のステータスをご確認いただけます。

### 3-2. 申請区分
申請は、利用形態に応じて以下の2区分とします。

(1) 事前申請型
セミナー受講、ジム入会、人間ドック追加検査、旅行手配など、利用前に補助の有無を確認する必要があるもの。利用予定日の14営業日前までにCafeteria@Sampleで申請し、人事本部労務厚生部の承認を得てから利用を開始してください。

(2) 事後精算型
書籍購入、ベビーシッター利用、引越し費用、医療費等、既に発生した費用に対する精算。費用発生から60日以内に申請してください。

### 3-3. 必要書類
1. 領収書(原本またはPDF・JPEG等の画像データ)。宛名は原則として社員本人とし、但し書きに対象項目を明記してください。
2. サービス内容証明書(資格取得の合格証明書、修了証、保育園の在園証明書、医師の診断書等)
3. 申請理由・利用目的の記述(50字以上)

### 3-4. 申請から振込みまでの流れ
1. 申請者がCafeteria@Sample上で必要事項を入力・書類アップロード
2. 人事本部労務厚生部が原則7営業日以内に審査
3. 承認後、毎月末締めで当社指定給与振込口座へ翌月25日に振込み(または、給与明細上に補助項目として加算表示)
4. 不承認の場合は、理由を付してCafeteria@Sample上で通知

### 3-5. 申請期限
1. 各年度のポイント利用申請は、当該年度3月31日までに提出されたものを有効とします。4月以降の到着分は新年度ポイントから引落します。
2. 退職・転籍時は、退職日までに申請されたものを精算対象とし、退職日翌日以降は無効となります(残ポイントは失効)。

### 3-6. Q&A・サポート窓口
1. ベネフィット・ステーションメニューに関する一般的なお問合せ:ベネフィット・ステーション コンタクトセンター 0120-XXX-XXX(平日9時〜18時)
2. Cafeteria@Sample操作・申請手続:人事本部 労務厚生部 福利厚生担当 fukuri@sample.co.jp(内線2360)
3. 制度全般・苦情・改善要望:Cafeteria@Sample内「意見箱」フォーム(匿名投稿可)

## 4. 移行期の対応
<!-- section_id: sec_4 -->

### 4-1. 旧制度の取扱い
従来の福利厚生制度のうち、本制度導入により以下のとおり取扱いを変更致します。

| 旧制度 | 取扱い | 適用 |
|---|---|---|
| 全社員旅行(隔年実施) | 廃止(2024年度実施分をもって終了) | 2025年4月以降は実施しない |
| 社員食堂利用補助(1食300円補助) | 1食100円補助に縮減して継続 | 健康経営の観点で完全廃止せず |
| 軽井沢・伊豆の自社保養所2施設 | 2025年9月末閉鎖 | 4月〜9月は従来通り利用可、その後施設売却 |
| 提携保養所(全国12施設) | 廃止 | ベネフィット・ステーション宿泊メニューへ統合 |
| 永年勤続表彰(10年・20年・30年) | 継続(金額拡充あり) | 表彰式は年1回実施 |
| 慶弔見舞金(結婚・出産・弔慰・傷病) | 継続(就業規則別表に基づく) | 変更なし |
| 健康診断(法定健診) | 継続(法定義務) | 変更なし |
| 社会保険・確定拠出年金 | 継続 | 変更なし |
| インフルエンザ予防接種(全額補助) | 継続 | 変更なし |
| 育児・介護休業給付金上乗せ(就業規則第60条) | 継続 | 変更なし |

### 4-2. 並行運用期間
1. 2025年4月1日〜2025年9月30日の6ヶ月間を「並行運用期間」と位置付け、新カフェテリアプランと旧制度の一部(社員食堂補助、自社保養所利用権)が並走します。
2. 並行運用期間中、社員の皆様は、旧制度の利用と新制度のポイント利用を併用可能ですが、同一支出に対する重複利用は不可とします。例えば、自社保養所の宿泊費を新制度の余暇カテゴリで申請することはできません。
3. 並行運用期間終了時(2025年9月30日)に未消化となる旧保養所利用権(各社員年間2泊分の宿泊権)は、本制度施行に伴う移行措置として、自動的にカフェテリアポイント8,000ポイント分へ換算・付与致します。

### 4-3. ポイント付与スケジュール(2025年度・移行特例)
1. 2025年4月1日:基本ポイント60,000ポイント付与
2. 2025年4月15日:育児・介護加算20,000ポイント付与(該当者のみ、4月10日までの申告者)
3. 2025年10月1日:旧保養所利用権換算ポイント8,000ポイント付与(全社員)
4. 2026年3月31日:本年度ポイント有効期限到来、未使用分は失効

### 4-4. 説明会・周知計画
1. 全社員向け制度説明会(オンライン・録画あり):2025年3月17日(月)13時〜14時、3月19日(水)17時〜18時、3月24日(月)10時〜11時(計3回、Microsoft Teams Liveイベント)
2. 部署単位の個別Q&Aセッション(対面・オンラインのいずれも対応):3月25日〜31日、各部門総務担当窓口経由で予約受付
3. 個別相談(育児・介護・特別事情等の個別性が高いもの):随時、人事本部労務厚生部窓口で受付(Microsoft Bookings予約)
4. 社内ポータル特設サイト「Cafeteria@Sample 制度ナビ」を3月10日開設し、FAQ、操作動画、メニュー一覧、よくある誤りを掲載
